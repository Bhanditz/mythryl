
The {\tt ogl\_client\_for\_server\_in\_main\_process} package implements base-level access to the Ogl widget 
toolkit.

The {\tt ogl\_client\_for\_server\_in\_main\_process} package implements the \ahrefloc{api:Ogl\_Client}{Ogl\_Client} api.

The {\tt ogl\_client\_for\_server\_in\_main\_process} package source code is in \ahrefloc{src/bnd/ogl/src/ogl-client-for-server-in-main-process.pkg}{src/bnd/ogl/src/ogl-client-for-server-in-main-process.pkg}; 
it merely invokes the {\tt ogl\_client\_g} generic package from 
\ahrefloc{src/bnd/ogl/src/ogl-client-g.pkg}{src/bnd/ogl/src/ogl-client-g.pkg}; with the {\tt ogl\_client\_driver\_for\_server\_in\_main\_process} package from 
\ahrefloc{src/bnd/ogl/src/ogl-client-driver-for-server-in-main-process.pkg}{src/bnd/ogl/src/ogl-client-driver-for-server-in-main-process.pkg} as argument. 

See also:  \ahrefloc{pkg:ogl\_client\_for\_server\_in\_c\_subprocess}{ogl\_client\_for\_server\_in\_c\_subprocess}.
See also:  \ahrefloc{pkg:easy\_ogl}{easy\_ogl}.
