
The standard library {\tt Ogl\_Client} api defines access to base-level Ogl functionality.

The {\tt Ogl\_Client} api is implemented by the \ahrefloc{pkg:ogl\_client\_for\_library\_in\_main\_process}{ogl\_client\_for\_library\_in\_main\_process}
the \ahrefloc{pkg:ogl\_client\_for\_library\_in\_c\_subprocess}{ogl\_client\_for\_library\_in\_c\_subprocess}
and packages.

The {\tt Ogl\_Client} api source code is in \ahrefloc{src/bnd/ogl/src/ogl-client.api}{src/bnd/ogl/src/ogl-client.api}.

See also:  \ahrefloc{api:Ogl\_Client\_Driver}{Ogl\_Client\_Driver}.
