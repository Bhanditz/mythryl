
The {\tt ogl\_client\_for\_server\_in\_c\_subprocess} package implements base-level access to the Ogl (OpenGL) graphics
toolkit.

The {\tt ogl\_client\_for\_server\_in\_c\_subprocess} package implements the \ahrefloc{api:Ogl\_Client}{Ogl\_Client} api.
