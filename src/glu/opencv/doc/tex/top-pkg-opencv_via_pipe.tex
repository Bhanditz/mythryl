
The {\tt opencv\_client\_for\_library\_in\_c\_subprocess} package implements base-level access to the Opencv (OpenCV) computer
vision toolkit.

The {\tt opencv\_client\_for\_library\_in\_c\_subprocess} package implements the \ahrefloc{api:Opencv\_Client}{Opencv\_Client} api.

The {\tt opencv\_client\_for\_library\_in\_c\_subprocess} package source code is in \ahrefloc{src/glu/opencv/src/opencv-client-for-library-in-c-subprocess.pkg}{src/glu/opencv/src/opencv-client-for-library-in-c-subprocess.pkg}.
