
The {\tt qqq\_client\_for\_library\_in\_main\_process} package implements base-level access to the Qqq widget 
toolkit.

The {\tt qqq\_client\_for\_library\_in\_main\_process} package implements the \ahrefloc{api:Qqq\_Client}{Qqq\_Client} api.

The {\tt qqq\_client\_for\_library\_in\_main\_process} package source code is in \ahrefloc{src/glu/qqq/src/qqq-client-for-library-in-main-process.pkg}{src/glu/qqq/src/qqq-client-for-library-in-main-process.pkg}; 
it merely invokes the {\tt qqq\_client\_g} generic package from 
\ahrefloc{src/glu/qqq/src/qqq-client-g.pkg}{src/glu/qqq/src/qqq-client-g.pkg}; with the {\tt qqq\_client\_driver\_for\_library\_in\_main\_process} package from 
\ahrefloc{src/glu/qqq/src/qqq-client-driver-for-library-in-main-process.pkg}{src/glu/qqq/src/qqq-client-driver-for-library-in-main-process.pkg} as argument. 

See also:  \ahrefloc{pkg:qqq\_client\_for\_library\_in\_c\_subprocess}{qqq\_client\_for\_library\_in\_c\_subprocess}.
