
The standard library {\tt Ogl\_Client\_Driver} api defines access to base-level Ogl 
driver functionality.

The {\tt Ogl\_Client\_Driver} api is implemented by the \ahrefloc{pkg:ogl\_client\_driver\_for\_library\_in\_c\_subprocess}{ogl\_client\_driver\_for\_library\_in\_c\_subprocess}
and \ahrefloc{pkg:ogl\_client\_driver\_for\_library\_in\_main\_process}{ogl\_client\_driver\_for\_library\_in\_main\_process}
packages.

The {\tt Ogl\_Client\_Driver} api source code is in \ahrefloc{src/glu/ogl/src/ogl-client-driver.api}{src/glu/ogl/src/ogl-client-driver.api}.
