
The {\tt easy\_ogl} package implements a high-level interface to 
{\tt Ogl} with the emphasis on making easy things easy.

At present this is very much a work in progress.

The {\tt easy\_ogl} package implements the \ahrefloc{api:Easy\_Ogl}{Easy\_Ogl} API.

The {\tt easy\_ogl} package source code is in \ahrefloc{src/glu/ogl/src/easy-ogl.pkg}{src/glu/ogl/src/easy-ogl.pkg}.

See also:  \ahrefloc{pkg:ogl}{ogl}.
