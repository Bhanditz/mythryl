
The {\tt opengl\_client\_for\_library\_in\_main\_process} package implements base-level access to the Opengl widget 
toolkit.

The {\tt opengl\_client\_for\_library\_in\_main\_process} package implements the \ahrefloc{api:Opengl\_Client}{Opengl\_Client} api.

The {\tt opengl\_client\_for\_library\_in\_main\_process} package source code is in \ahrefloc{src/glu/opengl/src/opengl-client-for-library-in-main-process.pkg}{src/glu/opengl/src/opengl-client-for-library-in-main-process.pkg}; 
it merely invokes the {\tt opengl\_client\_g} generic package from 
\ahrefloc{src/glu/opengl/src/opengl-client-g.pkg}{src/glu/opengl/src/opengl-client-g.pkg}; with the {\tt opengl\_client\_driver\_for\_library\_in\_main\_process} package from 
\ahrefloc{src/glu/opengl/src/opengl-client-driver-for-library-in-main-process.pkg}{src/glu/opengl/src/opengl-client-driver-for-library-in-main-process.pkg} as argument. 

See also:  \ahrefloc{pkg:opengl\_client\_for\_library\_in\_c\_subprocess}{opengl\_client\_for\_library\_in\_c\_subprocess}.
See also:  \ahrefloc{pkg:easy\_opengl}{easy\_opengl}.
