
The standard library {\tt Opengl\_Client\_Driver} api defines access to base-level Opengl 
driver functionality.

The {\tt Opengl\_Client\_Driver} api is implemented by the \ahrefloc{pkg:opengl\_client\_driver\_for\_library\_in\_c\_subprocess}{opengl\_client\_driver\_for\_library\_in\_c\_subprocess}
and \ahrefloc{pkg:opengl\_client\_driver\_for\_library\_in\_main\_process}{opengl\_client\_driver\_for\_library\_in\_main\_process}
packages.

The {\tt Opengl\_Client\_Driver} api source code is in \ahrefloc{src/glu/opengl/src/opengl-client-driver.api}{src/glu/opengl/src/opengl-client-driver.api}.

See also:  \ahrefloc{api:Opengl\_Client}{Opengl\_Client}.
