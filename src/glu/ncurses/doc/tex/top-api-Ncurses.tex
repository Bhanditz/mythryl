
The standard library {\tt Ncurses\_Client} api defines access to base-level Ncurses functionality.

The {\tt Ncurses\_Client} api is implemented by the \ahrefloc{pkg:ncurses\_client\_for\_library\_in\_main\_process}{ncurses\_client\_for\_library\_in\_main\_process}
the \ahrefloc{pkg:ncurses\_client\_for\_library\_in\_c\_subprocess}{ncurses\_client\_for\_library\_in\_c\_subprocess}
and packages.

The {\tt Ncurses\_Client} api source code is in \ahrefloc{src/glu/ncurses/src/ncurses-client.api}{src/glu/ncurses/src/ncurses-client.api}.

See also:  \ahrefloc{api:Ncurses\_Client\_Driver}{Ncurses\_Client\_Driver}.
