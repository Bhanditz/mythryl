
The standard library {\tt Gtk\_Driver} api defines access to base-level Gtk 
driver functionality.

The {\tt Gtk\_Driver} api is implemented by the \ahrefloc{pkg:gtk\_via\_pipe}{gtk\_via\_pipe} package.

The {\tt Gtk\_Driver} api source code is in \ahrefloc{src/glu/gtk/src/gtk-client-driver.api}{src/glu/gtk/src/gtk-client-driver.api}.

See also:  \ahrefloc{api:Gtk}{Gtk}.
See also:  \ahrefloc{api:Easy\_Gtk}{Easy\_Gtk}.
