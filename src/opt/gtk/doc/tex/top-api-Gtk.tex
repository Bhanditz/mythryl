
The standard library {\tt Gtk\_Client} api defines access to base-level Gtk functionality.

The {\tt Gtk\_Client} api is implemented by the \ahrefloc{pkg:gtk}{gtk} package.

The {\tt Gtk\_Client} api source code is in \ahrefloc{src/opt/gtk/src/gtk-client.api}{src/opt/gtk/src/gtk-client.api}.

See also:  \ahrefloc{api:Gtk\_Client\_Driver}{Gtk\_Client\_Driver}.
See also:  \ahrefloc{api:Easy\_Gtk}{Easy\_Gtk}.
