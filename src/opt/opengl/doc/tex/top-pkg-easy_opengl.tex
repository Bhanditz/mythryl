
The {\tt easy\_opengl} package implements a high-level interface to 
{\tt Opengl} with the emphasis on making easy things easy.

At present this is very much a work in progress.

The {\tt easy\_opengl} package implements the \ahrefloc{api:Easy\_Opengl}{Easy\_Opengl} API.

The {\tt easy\_opengl} package source code is in \ahrefloc{src/opt/opengl/src/easy-opengl.pkg}{src/opt/opengl/src/easy-opengl.pkg}.
