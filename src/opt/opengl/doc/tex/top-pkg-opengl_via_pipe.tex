
The {\tt opengl\_client\_for\_library\_in\_c\_subprocess} package implements base-level access to the Opengl (OpenGL) graphics 
toolkit.

The {\tt opengl\_client\_for\_library\_in\_c\_subprocess} package implements the \ahrefloc{api:Opengl\_Client}{Opengl\_Client} api.

The {\tt opengl\_client\_for\_library\_in\_c\_subprocess} package source code is in \ahrefloc{src/opt/opengl/src/opengl-client-for-library-in-c-subprocess.pkg}{src/opt/opengl/src/opengl-client-for-library-in-c-subprocess.pkg}.
