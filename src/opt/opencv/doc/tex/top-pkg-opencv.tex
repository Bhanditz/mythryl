
The {\tt opencv\_client\_for\_library\_in\_main\_process} package implements base-level access to the Opencv widget 
toolkit.

The {\tt opencv\_client\_for\_library\_in\_main\_process} package implements the \ahrefloc{api:Opencv\_Client}{Opencv\_Client} api.

The {\tt opencv\_client\_for\_library\_in\_main\_process} package source code is in \ahrefloc{src/opt/opencv/src/opencv-client-for-library-in-main-process.pkg}{src/opt/opencv/src/opencv-client-for-library-in-main-process.pkg}; 
it merely invokes the {\tt opencv\_client\_g} generic package from 
\ahrefloc{src/opt/opencv/src/opencv-client-g.pkg}{src/opt/opencv/src/opencv-client-g.pkg}; with the {\tt opencv\_client\_driver\_for\_library\_in\_main\_process} package from 
\ahrefloc{src/opt/opencv/src/opencv-client-driver-for-library-in-main-process.pkg}{src/opt/opencv/src/opencv-client-driver-for-library-in-main-process.pkg} as argument. 

See also:  \ahrefloc{pkg:opencv\_client\_for\_library\_in\_c\_subprocess}{opencv\_client\_for\_library\_in\_c\_subprocess}.
