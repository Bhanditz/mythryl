
The standard library {\tt Opencv\_Client\_Driver} api defines access to base-level Opencv 
driver functionality.

The {\tt Opencv\_Client\_Driver} api is implemented by the \ahrefloc{pkg:opencv\_client\_driver\_for\_library\_in\_c\_subprocess}{opencv\_client\_driver\_for\_library\_in\_c\_subprocess} 
and \ahrefloc{pkg:opencv\_client\_driver\_for\_library\_in\_main\_process}{opencv\_client\_driver\_for\_library\_in\_main\_process} 
packages.

The {\tt Opencv\_Client\_Driver} api source code is in \ahrefloc{src/opt/opencv/src/opencv-client-driver.api}{src/opt/opencv/src/opencv-client-driver.api}.

See also:  \ahrefloc{api:Opencv\_Client}{Opencv\_Client}.
