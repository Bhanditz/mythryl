
The standard library {\tt String} api defines access to immutable vectors of characters.

The {\tt String} api is implemented by the \ahrefloc{pkg:string}{string} package.

The {\tt String} api source code is in \ahrefloc{src/lib/std/src/string.api}{src/lib/std/src/string.api}.
