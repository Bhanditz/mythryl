13.4.140  set_sigalrm_frequency

The standard library set_sigalrm_frequency package implements access to host OS interval timer functionality. On Posix this is setitimer(), on MS-Windows this is win32StartTimer().

The set_sigalrm_frequency package implements the Set_Sigalrm_Frequency api.

The set_sigalrm_frequency package source code is in src/lib/std/src/nj/set-sigalrm-frequency.pkg.

The above information is manually maintained and may contain errors.
