
The standard library {\tt mailop} package implements mailop values and combinators --- basic Mythryl inter-thread communication functionality.

The {\tt mailop} package implements an extension of the \ahrefloc{api:Mailop}{Mailop} API.

The {\tt mailop} package source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/mailop.pkg}{src/lib/src/lib/thread-kit/src/core-thread-kit/mailop.pkg}.
