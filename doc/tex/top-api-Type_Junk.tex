
The compiler {\tt Type\_Junk} api defines access to functionality in 
support of compiler handling of type information.

The {\tt Type\_Junk} api is implemented by the \ahrefloc{pkg:type\_junk}{type\_junk} package.

The {\tt Type\_Junk} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/types/type-junk.api}{src/lib/compiler/front/typer-stuff/types/type-junk.api}.
