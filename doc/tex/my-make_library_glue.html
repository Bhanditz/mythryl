13.4.65  make_library_glue

The standard library make_library_glue package is support for autogeneration of much of the code needed for (e.g.) the Gtk and OpenGl C library Mythryl bindings.

The make_library_glue package implements the Make_Library_Glue api.

The make_library_glue package source code is in src/lib/make-library-glue/make-library-glue.pkg.

The above information is manually maintained and may contain errors.
