
The standard library {\tt Threadkit\_Synchronous\_Socket} api consists of that subset of the 
\ahrefloc{api:Threadkit\_Socket}{Threadkit\_Socket} api which includes no non-blocking operations.

The {\tt Threadkit\_Synchronous\_Socket} api is not implemented as such.

The {\tt Threadkit\_Synchronous\_Socket} api source code is in \ahrefloc{src/lib/std/src/socket/threadkit-synchronous-socket.api}{src/lib/std/src/socket/threadkit-synchronous-socket.api}.
