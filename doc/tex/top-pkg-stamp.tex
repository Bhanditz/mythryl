
The compiler {\tt stamp} package implements functionality for generating unique 
identifiers labelling various datastructure elements for the code being compiled.

The {\tt stamp} package implements the \ahrefloc{api:Stamp}{Stamp} package.

The {\tt stamp} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/basics/stamp.pkg}{src/lib/compiler/front/typer-stuff/basics/stamp.pkg}.
