
The {\tt ncurses\_client\_for\_library\_in\_main\_process} package implements base-level access to the Ncurses widget 
toolkit.

The {\tt ncurses\_client\_for\_library\_in\_main\_process} package implements the \ahrefloc{api:Ncurses\_Client}{Ncurses\_Client} api.

The {\tt ncurses\_client\_for\_library\_in\_main\_process} package source code is in \ahrefloc{src/opt/ncurses/src/ncurses-client-for-library-in-main-process.pkg}{src/opt/ncurses/src/ncurses-client-for-library-in-main-process.pkg}; 
it merely invokes the {\tt ncurses\_client\_g} generic package from 
\ahrefloc{src/opt/ncurses/src/ncurses-client-g.pkg}{src/opt/ncurses/src/ncurses-client-g.pkg}; with the {\tt ncurses\_client\_driver\_for\_library\_in\_main\_process} package from 
\ahrefloc{src/opt/ncurses/src/ncurses-client-driver-for-library-in-main-process.pkg}{src/opt/ncurses/src/ncurses-client-driver-for-library-in-main-process.pkg} as argument. 

See also:  \ahrefloc{pkg:ncurses\_client\_for\_library\_in\_c\_subprocess}{ncurses\_client\_for\_library\_in\_c\_subprocess}.
