13.2.10  data_file__premicrothread

The standard library data_file__premicrothread package implements file I/O for data ("binary") files.

The data_file__premicrothread package implements the Winix_Data_File_For_Os api.

On posix the data_file__premicrothread package is a convenience synonym for the winix_data_file_for_posix package.

On posix the data_file__premicrothread package source code is src/lib/std/src/posix/data-file–premicrothread.pkg. The above information is manually maintained and may contain errors.
