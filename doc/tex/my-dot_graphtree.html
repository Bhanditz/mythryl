13.4.17  dot_graphtree

The standard library dot_graphtree package implements in-memory representation of Graphviz .dot files.

The dot_graphtree package implements the Dot_Graphtree api.

The dot_graphtree package source code is in src/lib/std/dot/dot-graphtree.pkg.

The above information is manually maintained and may contain errors.
