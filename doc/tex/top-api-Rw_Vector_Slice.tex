
The standard library {\tt Rw\_Vector\_Slice} api defines access to subvectors within a 
\ahrefloc{api:Rw\_Vector}{Rw\_Vector} instance.  This allows referencing such subvectors in 
constant time and space without duplication of data.

The {\tt Rw\_Vector\_Slice} api is implemented by the \ahrefloc{pkg:rw\_vector\_slice}{rw\_vector\_slice} package.

The {\tt Rw\_Vector\_Slice} api source code is in \ahrefloc{src/lib/std/src/rw-vector-slice.api}{src/lib/std/src/rw-vector-slice.api}.


See also: \ahrefloc{api:Rw\_Vector}{Rw\_Vector}.
