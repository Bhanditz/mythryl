
The standard library {\tt List\_Sort} api defines access to sort algorithms 
for Lists.

The {\tt List\_Sort} api is implemented by the \ahrefloc{pkg:list\_mergesort}{list\_mergesort} package.

The {\tt List\_Sort} api source code is in \ahrefloc{src/lib/src/list-sort.api}{src/lib/src/list-sort.api}.
