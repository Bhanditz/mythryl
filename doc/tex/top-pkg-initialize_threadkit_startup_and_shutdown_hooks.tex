
The standard library {\tt initialize\_threadkit\_startup\_and\_shutdown\_hooks} package sets up the hooks for 
functions to be invoked at startup or shutdown of multithread programs.

The {\tt initialize\_threadkit\_startup\_and\_shutdown\_hooks} package source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/glue/initialize-threadkit-run-at.pkg}{src/lib/src/lib/thread-kit/src/glue/initialize-threadkit-run-at.pkg}.
