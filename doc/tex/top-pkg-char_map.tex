
The standard library {\tt char\_map} package implements simple vector-based mappings 
from type Char to any other type.

The {\tt char\_map} package implements the \ahrefloc{api:Char\_Map}{Char\_Map} API.

The {\tt char\_map} package source code is in \ahrefloc{src/lib/src/char-map.pkg}{src/lib/src/char-map.pkg}.
