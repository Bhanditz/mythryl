
The compiler {\tt compiled\_file} package implements Mythryl native object-code files, 
conventionally stored with {\tt .compiled} extensions.  These are analogous to unix 
{\tt .o} files, but customized to meet the specialized needs of the Mythryl 
compiler.

The {\tt compiled\_file} package implements the \ahrefloc{api:Compiled\_File}{Compiled\_File} API.

The {\tt compiled\_file} package source code is in \ahrefloc{src/lib/compiler/execution/compiledfile/compiledfile.pkg}{src/lib/compiler/execution/compiledfile/compiledfile.pkg}.
