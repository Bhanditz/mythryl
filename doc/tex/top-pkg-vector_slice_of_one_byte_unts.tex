
The standard library {\tt unt8\_vector\_slice} package implements slicing over 
\ahrefloc{pkg:vector\_of\_one\_byte\_unts}{vector\_of\_one\_byte\_unts} instances.  This allows constant-time, 
constant-space references to subvectors within a \ahrefloc{pkg:vector\_of\_one\_byte\_unts}{vector\_of\_one\_byte\_unts}.

The {\tt vector\_slice\_of\_one\_byte\_unts} package implements the \ahrefloc{api:Typelocked\_Vector\_Slice}{Typelocked\_Vector\_Slice} api.

The {\tt vector\_slice\_of\_one\_byte\_unts} package source code is in \ahrefloc{src/lib/std/src/vector-slice-of-one-byte-unts.pkg}{src/lib/std/src/vector-slice-of-one-byte-unts.pkg}.

See also:   \ahrefloc{pkg:vector\_of\_one\_byte\_unts}{vector\_of\_one\_byte\_unts}.
