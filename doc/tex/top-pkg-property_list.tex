
The compiler {\tt property\_list} package implements an arbitrary datastructure annotation 
facility patterned after Stephen Week's annotation library for Mlton.

The {\tt property\_list} package implements the \ahrefloc{api:Property\_List}{Property\_List} API.

The {\tt property\_list} package source code is in \ahrefloc{src/lib/src/property-list.pkg}{src/lib/src/property-list.pkg}.
