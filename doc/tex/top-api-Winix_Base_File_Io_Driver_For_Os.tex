
The standard library {\tt Winix\_Base\_File\_Io\_Driver\_For\_Os} api defines part of the I/O infrastructure for multithreaded Mythryl programs.

The {\tt Winix\_Base\_File\_Io\_Driver\_For\_Os} api is implemented by 
the {\tt winix\_base\_file\_io\_driver\_for\_posix\_g} generic from 
\ahrefloc{src/lib/std/src/io/winix-base-file-io-driver-for-posix-g.pkg}{src/lib/std/src/io/winix-base-file-io-driver-for-posix-g.pkg}.

The {\tt Winix\_Base\_File\_Io\_Driver\_For\_Os} api source code is in \ahrefloc{src/lib/std/src/io/winix-base-file-io-driver-for-os.api}{src/lib/std/src/io/winix-base-file-io-driver-for-os.api}.
