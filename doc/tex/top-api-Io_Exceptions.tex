
The standard library {\tt Io\_Exceptions} api defines access to exceptions generated during basic file input/output functionality.

The {\tt Io\_Exceptions} api is implemented by the \ahrefloc{pkg:io\_exceptions}{io\_exceptions} package.

The {\tt Io\_Exceptions} api source code is in \ahrefloc{src/lib/std/src/io/io-exceptions.api}{src/lib/std/src/io/io-exceptions.api}.
