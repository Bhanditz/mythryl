
The standard library {\tt Rw\_Vector\_Sort} api defines access to in-place sort algorithms for 
mutable vectors.

The {\tt Rw\_Vector\_Sort} api is implemented by the \ahrefloc{pkg:rw\_vector\_quicksort}{rw\_vector\_quicksort} package.

(Will somone please contribute a matching heapsort implementation?)

The {\tt Rw\_Vector\_Sort} api source code is in \ahrefloc{src/lib/src/rw-vector-sort.api}{src/lib/src/rw-vector-sort.api}.
