
The standard library {\tt io\_exceptions} package implements exceptions raised during basic file input/output functionality.

The {\tt io\_exceptions} package implements the \ahrefloc{api:Io\_Exceptions}{Io\_Exceptions} API.

The {\tt io\_exceptions} package source code is in \ahrefloc{src/lib/std/src/io/io-exceptions.pkg}{src/lib/std/src/io/io-exceptions.pkg}.
