
The compiler {\tt source\_map} package implements 
functionality mapping syntax tree elements 
to their originating source file regions, to allow accurate compiler diagnostic messages.

The {\tt source\_map} package implements the \ahrefloc{api:Source\_Map}{Source\_Map} API.

The {\tt source\_map} package source code is in \ahrefloc{src/lib/compiler/front/basics/source/line-number-db.pkg}{src/lib/compiler/front/basics/source/line-number-db.pkg}.
