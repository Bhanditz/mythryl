
The standard library {\tt Parser\_Combinator} api defines access to parser 
combinators over readers.

The {\tt Parser\_Combinator} api is implemented by the \ahrefloc{pkg:parser\_combinator}{parser\_combinator} package.

The {\tt Parser\_Combinator} api source code is in \ahrefloc{src/lib/src/parser-combinator.api}{src/lib/src/parser-combinator.api}.
