
The standard library {\tt template\_hostthread} package provides a clone-and-mutate template 
for defining new dedicated server hostthreads.

The {\tt template\_hostthread} package implements the \ahrefloc{api:Template\_Hostthread}{Template\_Hostthread} api.

The {\tt template\_hostthread} package source code is in \ahrefloc{src/lib/std/src/hostthread/template-hostthread.pkg}{src/lib/std/src/hostthread/template-hostthread.pkg}.
