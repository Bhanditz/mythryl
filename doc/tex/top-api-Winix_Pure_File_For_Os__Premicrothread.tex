
The standard library {\tt Winix\_Pure\_File\_For\_Os\_\_Premicrothread} api defines access to ``pure'' (side-effect free) input. 
(No attempt is made to implement pure output.)

Implementations of {\tt Winix\_Pure\_File\_For\_Os\_\_Premicrothread} are produced by 
{\tt winix\_data\_file\_for\_os\_g\_\_premicrothread} and {\tt winix\_text\_file\_for\_os\_g\_\_premicrothread} from respectively 
\ahrefloc{src/lib/std/src/io/winix-data-file-for-os-g--premicrothread.pkg}{src/lib/std/src/io/winix-data-file-for-os-g--premicrothread.pkg} and 
\ahrefloc{src/lib/std/src/io/winix-text-file-for-os-g--premicrothread.pkg}{src/lib/std/src/io/winix-text-file-for-os-g--premicrothread.pkg}.

Predefined packages generated by those two include the standard library \ahrefloc{pkg:winix\_text\_file\_for\_posix}{winix\_text\_file\_for\_posix} and 
\ahrefloc{pkg:winix\_data\_file\_for\_posix}{winix\_data\_file\_for\_posix} packages, more commonly referenced by the 
synonymous packages \ahrefloc{pkg:file}{file} and \ahrefloc{pkg:data\_file}{data\_file}.

The {\tt Winix\_Pure\_File\_For\_Os\_\_Premicrothread} api source code is in \ahrefloc{src/lib/std/src/io/winix-pure-file-for-os--premicrothread.api}{src/lib/std/src/io/winix-pure-file-for-os--premicrothread.api}.
