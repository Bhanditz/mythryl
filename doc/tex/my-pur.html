13.2.41  pur

The standard library pur ("pure I/O") package implements “pure” (side-effect free) input. (No attempt is made to implement pure output.)

The pur package implements the Winix_Pure_File_For_Os API.

The pur package exists as sub-packages of the winix_data_file_for_posix and winix_text_file_for_posix packages generated by src/lib/std/src/io/winix-data-file-for-os-g–premicrothread.pkg and src/lib/std/src/io/winix-text-file-for-os-g–premicrothread.pkg.

See also: Winix_File_For_Os. The above information is manually maintained and may contain errors.
