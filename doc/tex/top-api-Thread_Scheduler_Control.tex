
The standard library {\tt Thread\_Scheduler\_Control} api defines 
control of the Mythryl thread scheduler, in particular starting and stopping it.

The {\tt Thread\_Scheduler\_Control} api is implemented by the \ahrefloc{pkg:thread\_scheduler\_control}{thread\_scheduler\_control} package.

The {\tt Thread\_Scheduler\_Control} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/glue/thread-scheduler-control.api}{src/lib/src/lib/thread-kit/src/glue/thread-scheduler-control.api}.

See also: \ahrefloc{api:Threadkit\_Startup\_And\_Shutdown\_Hooks}{Threadkit\_Startup\_And\_Shutdown\_Hooks}.
