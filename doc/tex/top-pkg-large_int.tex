
The standard library {\tt large\_int} package implements 
indefinite-precision integer arithmetic.

The {\tt large\_int} package implements the \ahrefloc{api:Int}{Int} API.

The {\tt large\_int} package source code is in \ahrefloc{src/lib/std/large-int.pkg}{src/lib/std/large-int.pkg}.
