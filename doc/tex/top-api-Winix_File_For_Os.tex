
The standard library {\tt Threadkit\_Winix\_File\_For\_Os} api defines access to basic low-level 
functionality to read and write disk files and so forth.

The {\tt Threadkit\_Winix\_File\_For\_Os} api is included by the \ahrefloc{api:Threadkit\_Winix\_Data\_File\_For\_Os} api.

The {\tt Threadkit\_Winix\_File\_For\_Os} api source code is in \ahrefloc{src/lib/std/src/io/threadkit-winix-file-for-os.api}{src/lib/std/src/io/threadkit-winix-file-for-os.api}.
