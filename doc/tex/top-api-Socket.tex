
The standard library {\tt Socket} api defines access to {\sc TCP/IP} network socket functionality for multithreaded Mythryl programs.

The {\tt Socket} api is implemented by the \ahrefloc{pkg:socket}{socket} package.

The {\tt Socket} api source code is in \ahrefloc{src/lib/std/src/socket/socket.api}{src/lib/std/src/socket/socket.api}.
