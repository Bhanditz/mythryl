13.4.143  socket_junk

The standard library socket_junk package implements additional convenience functions on sockets, beyond those implemented in socket__premicrothread.

The socket_junk package implements the Socket_Junk API.

The socket_junk package source code is in src/lib/internet/socket-junk.pkg.
