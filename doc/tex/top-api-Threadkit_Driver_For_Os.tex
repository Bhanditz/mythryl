
The standard library {\tt Threadkit\_Driver\_For\_Os} api defines 
the interface to an OS specific module that glues the various 
OS-specific scheduling operations together (i.e., timeouts, I/O, signals, etc).

The {\tt Threadkit\_Driver\_For\_Os} api is implemented by the \ahrefloc{pkg:threadkit\_driver\_for\_posix}{threadkit\_driver\_for\_posix} package.

The {\tt Threadkit\_Driver\_For\_Os} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/posix/threadkit-driver-for-os.api}{src/lib/src/lib/thread-kit/src/posix/threadkit-driver-for-os.api}.
