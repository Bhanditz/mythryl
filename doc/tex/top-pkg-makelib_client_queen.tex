
The compiler {\tt makelib\_client\_queen} package is part of the internal 
support for distributed program compilation using multiple processes 
potentially on multiple hosts.  This facility is a nonfunctional 
work in progress.

The {\tt makelib\_client\_queen} package implements the \ahrefloc{src/app/makelib/concurrency/makelib-client-queen.pkg}{Makelib\_Client\_Queen} api.

The {\tt makelib\_client\_queen} package source code is in \ahrefloc{src/app/makelib/concurrency/makelib-client-queen.pkg}{src/app/makelib/concurrency/makelib-client-queen.pkg}.

See also: \ahrefloc{pkg:makelib}{makelib} package.

See also: \ahrefloc{src/app/makelib/main/makelib-server.pkg}{src/app/makelib/main/makelib-server.pkg}.
