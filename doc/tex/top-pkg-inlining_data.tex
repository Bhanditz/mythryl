
The standard library {\tt inlining\_data} package implements the 
compiler framework for passing around inlining data during typechecking.

The {\tt inlining\_datal} package has no explicit API.

The {\tt inlining\_data} package source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/basics/inlining-data.pkg}{src/lib/compiler/front/typer-stuff/basics/inlining-data.pkg}.

See also:  \ahrefloc{pkg:inlining\_mapstack}{inlining\_mapstack}.
