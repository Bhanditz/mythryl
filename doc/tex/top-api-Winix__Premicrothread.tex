
The standard library {\tt Winix\_\_Premicrothread} api defines access to cross-platform {\sc OS} functionality 
usable on both Windows and Posix (thus ``Winix'') platforms.  The {\tt Winix\_\_Premicrothread} api is thus an 
alternative to the \ahrefloc{api:Posixlib}{Posixlib} api offering increased portability at the 
cost of reduced functionality.

The {\tt Winix\_\_Premicrothread} api is implemented by the \ahrefloc{pkg:winix\_\_premicrothread}{winix\_\_premicrothread} package.

The {\tt Winix\_\_Premicrothread} api source code is in \ahrefloc{src/lib/std/src/winix/winix--premicrothread.api}{src/lib/std/src/winix/winix--premicrothread.api}.

See also: \ahrefloc{api:Posixlib}{Posixlib}.
