

The standard library {\tt Typelocked\_Vector} api defines access to vectors 
specialized to hold a particular element type such as {\tt Char}.

The {\tt Typelocked\_Vector} api is implemented by the 
\ahrefloc{pkg:vector\_of\_chars}{vector\_of\_chars}, 
\ahrefloc{pkg:vector\_of\_one\_byte\_unts}{vector\_of\_one\_byte\_unts} and 
\ahrefloc{pkg:vector\_of\_eight\_byte\_floats}{vector\_of\_eight\_byte\_floats} packages.

The {\tt Typelocked\_Vector} api source code is in \ahrefloc{src/lib/std/src/typelocked-vector.api}{src/lib/std/src/typelocked-vector.api}.

See also: \ahrefloc{api:Typelocked\_Rw\_Vector}{Typelocked\_Rw\_Vector}.

See also: \ahrefloc{api:Typelocked\_Vector\_Slice}{Typelocked\_Vector\_Slice}.

See also: \ahrefloc{section:tut:full-monte:typelocked-vectors}{Typelocked Vectors tutorial}.
