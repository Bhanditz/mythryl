
The standard library {\tt Paired\_Lists} api defines access to 
analogues of the list functions {\tt zip}, {\tt fold\_left} etc which operate in parallel 
on pairs of lists instead of on single lists.

The {\tt Paired\_Lists} api is implemented by the \ahrefloc{pkg:paired\_lists}{paired\_lists} package.

The {\tt Paired\_Lists} api source code is in \ahrefloc{src/lib/std/src/paired-lists.api}{src/lib/std/src/paired-lists.api}.
