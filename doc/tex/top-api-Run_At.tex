
The standard library {\tt Run\_At} api defines hooks for 
functions to be invoked at startup or shutdown of multithread programs.

The {\tt Run\_At} api is implemented by the \ahrefloc{pkg:run\_at}{run\_at} package.

The {\tt Run\_At} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/run-at.api}{src/lib/src/lib/thread-kit/src/core-thread-kit/run-at.api}.

See also: \ahrefloc{api:Run\_At\_\_Premicrothread}{Run\_At\_\_Premicrothread}.
See also: \ahrefloc{api:Thread\_Scheduler\_Control}{Thread\_Scheduler\_Control}.
