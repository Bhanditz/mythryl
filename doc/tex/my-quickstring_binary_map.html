13.4.108  quickstring_binary_map

The standard library quickstring_binary_map package implements quickstring__premicrothread-keyed maps.

The quickstring_binary_map package implements the Map API via binary_map_g from src/lib/src/binary-map-g.pkg.

The quickstring_binary_map package source code is in src/lib/src/quickstring-binary-map.pkg.

See also: quickstring_binary_set.
