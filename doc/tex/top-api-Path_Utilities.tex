
The standard library {\tt Path\_Utilities} api defines access to simple cross-platorm file-finding functionality.

The {\tt Path\_Utilities} api is implemented by the \ahrefloc{pkg:path\_utilities}{path\_utilities} package.

The {\tt Path\_Utilities} api source code is in \ahrefloc{src/lib/src/path-utilities.api}{src/lib/src/path-utilities.api}.
