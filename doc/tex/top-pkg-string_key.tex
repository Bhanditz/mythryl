
The standard library {\tt string\_key} package is an argument for {\tt red\_black\_map\_g} and 
{\tt red\_black\_set\_g}, used to define the \ahrefloc{pkg:string\_map}{string\_map} and 
\ahrefloc{pkg:string\_set}{string\_set} packages.

The {\tt string\_key} package source code is in \ahrefloc{src/lib/src/string-key.pkg}{src/lib/src/string-key.pkg}.

See also:  \ahrefloc{pkg:string\_map}{string\_map}.

See also:  \ahrefloc{pkg:string\_set}{string\_set}.
