13.5.9  global_controls

The compiler global_controls package implements various controls modulating the behavior of the Mythryl compiler.

The global_controls package implements the Global_Controls API.

The global_controls package source code is in src/lib/compiler/toplevel/main/global-controls.pkg.

The above information is manually maintained and may contain errors.
