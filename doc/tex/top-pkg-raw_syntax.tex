
The compiler {\tt raw\_syntax} package implements the Mythryl compiler data structure 
used to represent raw syntax trees of code under compilation.

A good point from which to start exploring the Mythryl compiler parsing code is 
\ahrefloc{src/lib/compiler/front/parser/main/mythryl-parser.pkg}{src/lib/compiler/front/parser/main/mythryl-parser.pkg}.

The {\tt raw\_syntax} package implements the \ahrefloc{api:Raw\_Syntax}{Raw\_Syntax} API.

The {\tt raw\_syntax} package source code is in \ahrefloc{src/lib/compiler/front/parser/raw-syntax/raw-syntax.pkg}{src/lib/compiler/front/parser/raw-syntax/raw-syntax.pkg}.

See also:  \ahrefloc{pkg:deep\_syntax}{deep\_syntax} package.

See also:  \ahrefloc{pkg:anormcode\_form}{anormcode\_form} package.

The Mythryl grammar specification proper may be found in {\tt src/lib/compiler/front/parser/yacc/mythryl.grammar} 
and the matching lexical level specification in {\tt src/lib/compiler/front/parser/lex/mythryl.lex}.
