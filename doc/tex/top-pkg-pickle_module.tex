
The compiler {\tt pickle\_module} package implements ``pickling'' (bytestream serialization) 
of ``modules'', the in-memory datastructures resulting from compilation of a Mythryl source 
code file.

The {\tt pickle\_module} package implements the \ahrefloc{api:Pickle\_Module}{Pickle\_Module} API.

The {\tt pickle\_module} package source code is in \ahrefloc{src/lib/compiler/front/semantic/pickle/pickler-junk.pkg}{src/lib/compiler/front/semantic/pickle/pickler-junk.pkg}.

See also: \ahrefloc{pkg:unpickle\_module}{unpickle\_module}.
