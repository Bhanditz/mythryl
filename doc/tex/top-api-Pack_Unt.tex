
The standard library {\tt Pack\_Unt} api defines access to functionality for 
serializing and unserializing unsigned integer values, which is to say, 
packing and unpacking them from bytestreams.

The {\tt Pack\_Unt} api is implemented by the 
\ahrefloc{pkg:pack\_little\_endian\_unt16}{pack\_little\_endian\_unt16}, 
\ahrefloc{pkg:pack\_big\_endian\_unt16}{pack\_big\_endian\_unt16}, 
\ahrefloc{pkg:pack\_little\_endian\_unt1}{pack\_little\_endian\_unt1} and 
\ahrefloc{pkg:pack\_big\_endian\_unt1}{pack\_big\_endian\_unt1} packages.

The {\tt Pack\_Unt} api source code is in \ahrefloc{src/lib/std/src/pack-unt.api}{src/lib/std/src/pack-unt.api}.
