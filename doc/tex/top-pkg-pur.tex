
The standard library {\tt pur} ({\tt "pure I/O"}) package implements 
``pure'' (side-effect free) input. 
(No attempt is made to implement pure output.)

The {\tt pur} package implements the \ahrefloc{api:Winix\_Pure\_File\_For\_Os}{Winix\_Pure\_File\_For\_Os} API.

The {\tt pur} package exists as sub-packages of the 
\ahrefloc{pkg:winix\_data\_file\_for\_posix}{winix\_data\_file\_for\_posix} and \ahrefloc{pkg:winix\_text\_file\_for\_posix}{winix\_text\_file\_for\_posix} packages 
generated by  
\ahrefloc{src/lib/std/src/io/winix-data-file-for-os-g--premicrothread.pkg}{src/lib/std/src/io/winix-data-file-for-os-g--premicrothread.pkg} and 
\ahrefloc{src/lib/std/src/io/winix-text-file-for-os-g--premicrothread.pkg}{src/lib/std/src/io/winix-text-file-for-os-g--premicrothread.pkg}.

See also: \ahrefloc{api:Winix\_File\_For\_Os}{Winix\_File\_For\_Os}.
