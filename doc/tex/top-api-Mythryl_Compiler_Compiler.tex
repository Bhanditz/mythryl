
The standard library {\tt Mythryl\_Compiler\_Compiler} api defines access to dedicated functionality 
for building the Mythryl compiler itself.

The {\tt Mythryl\_Compiler\_Compiler} api is implemented by the \ahrefloc{pkg:mythryl\_compiler\_compiler\_for\_this\_platform}{mythryl\_compiler\_compiler\_for\_this\_platform} package.

The {\tt Mythryl\_Compiler\_Compiler} api source code is in \ahrefloc{src/lib/core/internal/mythryl-compiler-compiler.api}{src/lib/core/internal/mythryl-compiler-compiler.api}.
