13.4.72  microthread_preemptive_scheduler

The standard library microthread_preemptive_scheduler package implements the scheduling queues and preemption mechanisms for microthreaded Mythryl programs.

The microthread_preemptive_scheduler package implements the Microthread_Preemptive_Scheduler api.

The microthread_preemptive_scheduler package source code is in src/lib/src/lib/thread-kit/src/core-thread-kit/microthread-preemptive-scheduler.pkg.
