
The compiler {\tt Module\_Junk} api defines access to functionality tracking modules 
in the compilation process.    (In this context, a ``module'' is essentially what you get from 
compiling one sourcefile.)

The {\tt Module\_Junk} api is implemented by the \ahrefloc{pkg:module\_junk}{module\_junk} package.

The {\tt Module\_Junk} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/modules/module-junk.api}{src/lib/compiler/front/typer-stuff/modules/module-junk.api}.

See also:  \ahrefloc{api:Type\_Package\_Language}{Type\_Package\_Language}.
