
The standard library {\tt rw\_vector\_of\_chars} package implements typelocked mutable vectors (one-dimensional arrays) 
specialized to hold char values.

The {\tt rw\_vector\_of\_chars} package implements the \ahrefloc{api:Typelocked\_Rw\_Vector}{Typelocked\_Rw\_Vector} API.

The {\tt rw\_vector\_of\_chars} package source code is in \ahrefloc{src/lib/std/src/rw-vector-of-chars.pkg}{src/lib/std/src/rw-vector-of-chars.pkg}, 
proxied in \ahrefloc{src/lib/std/rw-vector-of-chars.pkg}{src/lib/std/rw-vector-of-chars.pkg}.
