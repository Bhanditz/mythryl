
The compiler {\tt Varhome} api defines management of storage class information associated 
with a user variable during the compilation process.

The {\tt Varhome} api is implemented by the \ahrefloc{pkg:varhome}{varhome} package.

The {\tt Varhome} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/basics/varhome.api}{src/lib/compiler/front/typer-stuff/basics/varhome.api}.
