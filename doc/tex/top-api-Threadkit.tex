
The standard library {\tt Threadkit} api is the base of the multithreaded-programming api hierarchy.

The {\tt Threadkit} api is implemented by the \ahrefloc{pkg:threadkit}{threadkit} package.

The {\tt Threadkit} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/threadkit.api}{src/lib/src/lib/thread-kit/src/core-thread-kit/threadkit.api}.
