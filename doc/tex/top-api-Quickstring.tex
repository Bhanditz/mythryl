
The standard library {\tt Quickstring} api defines access to hashed strings supporting 
constant-time equality testing.

The {\tt Quickstring} api is implemented by the \ahrefloc{pkg:quickstring\_\_premicrothread}{quickstring\_\_premicrothread} package.

The {\tt Quickstring} api is also implemented by the \ahrefloc{pkg:quickstring}{quickstring} package.

The {\tt Quickstring} api source code is in \ahrefloc{src/lib/src/quickstring.api}{src/lib/src/quickstring.api}.
