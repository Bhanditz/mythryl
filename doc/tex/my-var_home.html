Varhome

The following information is manually maintained and may contain errors. The compiler varhome package manages storage class information associated with a user variable during the compilation process.

The varhome package implements the Varhome API.

The varhome package source code is in src/lib/compiler/front/typer-stuff/basics/varhome.pkg.
