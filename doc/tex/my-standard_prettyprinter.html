13.2.54  standard_prettyprinter

The standard library standard_prettyprinter package implements the standard prettyprinter functionality used for formatting intermediate code etc for human consumption.

The standard_prettyprinter package implements the Standard_Prettyprinter api.

The standard_prettyprinter package source code is in src/lib/prettyprint/big/src/standard-prettyprinter.pkg.

See also: base_prettyprinter.
