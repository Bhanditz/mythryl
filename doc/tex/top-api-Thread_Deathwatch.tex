
The standard library {\tt Thread\_Deathwatch} api defines functionality to detect and report unexpected thread terminations.

The {\tt Thread\_Deathwatch} api is implemented by the \ahrefloc{pkg:thread\_deathwatch}{thread\_deathwatch} package.

The {\tt Thread\_Deathwatch} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/lib/thread-deathwatch.api}{src/lib/src/lib/thread-kit/src/lib/thread-deathwatch.api}.
