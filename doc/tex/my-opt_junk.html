13.4.87  opt_junk

The standard library opt_junk package implements support for the ’opt’ facility for dis/including optional Mythryl C-library bindings e.g. those for OpenGl and Gtk.

The opt_junk package implements the Opt_Junk api.

The opt_junk package source code is in src/lib/make-library-glue/opt-junk.pkg.
