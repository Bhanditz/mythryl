
The standard library {\tt Io\_Now\_Possible\_Mailop} api defines an internal support facility used by multithreaded Mythryl programs.
It is not normally of direct interest to the application programmer.

The {\tt Io\_Now\_Possible\_Mailop} api is implemented by the \ahrefloc{pkg:io\_now\_possible\_mailop}{io\_now\_possible\_mailop} package.

The {\tt Io\_Now\_Possible\_Mailop} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/io-now-possible-mailop.api}{src/lib/src/lib/thread-kit/src/core-thread-kit/io-now-possible-mailop.api}.
