9.4  Native Mythryl libraries

Write solid, well-thought out, well documented Mythryl libraries. Again, don’t forget docs, examples and tutorials.

Current targets include:

  • Improve the general graph library: Add algorithms, visualization tools, and conversions to and from major other datastructures.
  • Write/port an OpenGL-based graph visualization library. GraphViz is probably the place to start.
  • A simple relational db library that plays nice with the Mythryl type system.
  • Port the Pacal symbolic algebra package to Mythryl. Six kilolines of clean code.
Comments and suggestions to: bugs@mythryl.org
