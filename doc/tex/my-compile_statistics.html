13.5.5  compile_statistics

The compiler compile_statistics package implements a facility for creating, maintaining, aggregating and reporting counts of various compile-related events.

The compile_statistics package implements the Compile_Statistics API.

The compile_statistics package source code is in src/lib/compiler/front/basics/stats/compile-statistics.pkg.

The above information is manually maintained and may contain errors.
