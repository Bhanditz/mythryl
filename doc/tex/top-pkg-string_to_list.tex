
The standard library {\tt string\_to\_list} package implements translations from 
Strings to Lists.

The {\tt string\_to\_list} package implements the \ahrefloc{api:String\_To\_List}{String\_To\_List} API.

The {\tt string\_to\_list} package source code is in \ahrefloc{src/lib/src/string-to-list.pkg}{src/lib/src/string-to-list.pkg}.

See also:  \ahrefloc{pkg:list\_to\_string}{list\_to\_string}.
