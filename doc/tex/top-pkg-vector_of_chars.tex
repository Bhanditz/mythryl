
The standard library {\tt char\_vector} package implements vectors specialized to hold 
{\tt Char} values.  Using this specialized typelocked vector implementation can 
yield a space savings of up to a factor of four on 32-bit implementations.

The {\tt char\_vector} package implements the \ahrefloc{api:Typelocked\_Vector}{Typelocked\_Vector} api.

The {\tt char\_vector} package source code is in \ahrefloc{src/lib/std/src/vector-of-chars.pkg}{src/lib/std/src/vector-of-chars.pkg}.

See also:   \ahrefloc{pkg:vector\_slice\_of\_chars}{vector\_slice\_of\_chars}.
