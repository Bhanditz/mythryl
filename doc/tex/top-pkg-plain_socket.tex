
The standard library {\tt plain\_socket} package implements vanilla {\sc TCP/IP} socket functionality for multithreaded Mythryl programs.

The {\tt plain\_socket} package implements the \ahrefloc{api:Threadkit\_Plain\_Socket}{Threadkit\_Plain\_Socket} API.

The Linux/unix {\tt plain\_socket} package source code is in \ahrefloc{src/lib/std/src/socket/plain-socket.pkg}{src/lib/std/src/socket/plain-socket.pkg}.

The {\sc MS-Windows} {\tt threadkit\_plain\_socket} package source code is in \ahrefloc{src/lib/std/src/socket/win32-plain-socket.pkg}{src/lib/std/src/socket/win32-plain-socket.pkg}.

See also: \ahrefloc{pkg:plain\_socket}{plain\_socket} package.
