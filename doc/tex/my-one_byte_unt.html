13.4.82  one_byte_unt

The standard library one_byte_unt package implements 8-bit unsigned integer arithmetic and related functionality.

The one_byte_unt package implements the Unt api.

The one_byte_unt package source code is in src/lib/std/src/one-byte-unt-guts.pkg, proxied in src/lib/std/one-byte-unt.pkg.
