
The compiler {\tt Unify\_Typoids} api defines access to {\tt unify} functionality 
for type inference.  Mythryl uses Prolog-style ``unification'' to perform type 
inference;  this api defines the needed {\tt unify\_typoids} operation. 

The {\tt Unify\_Typoids} api is implemented by the \ahrefloc{pkg:unify\_typoids}{unify\_typoids} package.

The {\tt Unify\_Typoids} api source code is in \ahrefloc{src/lib/compiler/front/typer/types/unify-typoids.pkg}{src/lib/compiler/front/typer/types/unify-typoids.pkg}.
