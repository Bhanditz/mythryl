
The standard library {\tt Commandline} api defines access to the commandline 
arguments used to invoke the program.

The {\tt Commandline} api is implemented by the \ahrefloc{pkg:commandline}{commandline} package.

The {\tt Commandline} api source code is in \ahrefloc{src/lib/std/commandline.api}{src/lib/std/commandline.api}.
