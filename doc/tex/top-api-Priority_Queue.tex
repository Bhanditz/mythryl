
The standard library {\tt Priority\_Queue} api defines access to priority queue implementations.

The {\tt Priority\_Queue} api is implemented by the \ahrefloc{pkg:heap\_priority\_queue}{heap\_priority\_queue} 
and \ahrefloc{pkg:leftist\_tree\_priority\_queue}{leftist\_tree\_priority\_queue} packages.

The {\tt Priority\_Queue} api source code is in \ahrefloc{src/lib/src/priority-queue.api}{src/lib/src/priority-queue.api}.

See also:  \ahrefloc{api:Priority}{Priority}.

See also:  \ahrefloc{api:Typelocked\_Priority\_Queue}{Typelocked\_Priority\_Queue}.
