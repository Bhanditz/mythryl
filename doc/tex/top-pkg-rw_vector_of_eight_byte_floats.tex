
The standard library {\tt rw\_float64\_vector} package implements typelocked mutable vectors (one-dimensional arrays) 
specialized to hold sixty-four bit floating point values.

The {\tt rw\_float64\_vector} package implements the \ahrefloc{api:Typelocked\_Rw\_Vector}{Typelocked\_Rw\_Vector} API.

The {\tt rw\_float64\_vector} package source code is in \ahrefloc{src/lib/std/src/rw-vector-of-eight-byte-floats.pkg}{src/lib/std/src/rw-vector-of-eight-byte-floats.pkg} 
and proxied in \ahrefloc{src/lib/std/rw-float-vector.pkg}{src/lib/std/rw-float-vector.pkg}.
