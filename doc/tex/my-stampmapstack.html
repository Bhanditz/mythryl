13.5.43  stampmapstack

The compiler stampmapstack package tracks information about modules being compiled. (In this context, a “module” is essentially what you get from compiling one sourcefile.)

The stampmapstack package implements the Stampmapstack API.

The stampmapstack package source code is in src/lib/compiler/front/typer-stuff/modules/stampmapstack.pkg.
