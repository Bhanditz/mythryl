
The standard library {\tt Float} api defines access to core floating point arithmetic functionality.

The {\tt Float} api is implemented by the \ahrefloc{pkg:eight\_byte\_float}{eight\_byte\_float},
a proxy for {\tt eight\_byte\_float\_guts} from \ahrefloc{src/lib/std/src/eight-byte-float-guts.pkg}{src/lib/std/src/eight-byte-float-guts.pkg}.

The {\tt Float} api source code is in \ahrefloc{src/lib/std/src/float.api}{src/lib/std/src/float.api}.
