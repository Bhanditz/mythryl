
The standard library {\tt socket} package implements access to {\sc TCP/IP} network socket functionality for multithreaded Mythryl programs.

The {\tt socket} package implements the \ahrefloc{api:Socket}{Socket} api.

The {\tt socket} package source code is in \ahrefloc{src/lib/std/src/socket/socket.pkg}{src/lib/std/src/socket/socket.pkg}.
