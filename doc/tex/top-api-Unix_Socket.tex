streams and datagrams via unix domain sockets.

The {\tt Unix\_Domain\_Socket\_\_Premicrothread} api is implemented by the \ahrefloc{pkg:unix\_domain\_socket\_\_premicrothread}{unix\_domain\_socket\_\_premicrothread} package.

The {\tt Unix\_Domain\_Socket\_\_Premicrothread} api source code is in \ahrefloc{src/lib/std/src/socket/unix-domain-socket--premicrothread.api}{src/lib/std/src/socket/unix-domain-socket--premicrothread.api}.
