
The compiler {\tt Make\_Compiler\_Backend} api defines access to top-level functionality 
for recompiling the compiler itself.  (Currently, this process involves too many 
special cases to be done using the standard \ahrefloc{pkg:makelib}{makelib} package.)

The {\tt Make\_Compiler\_Backend} api is implemented by the \ahrefloc{pkg:make\_compiler\_backend}{make\_compiler\_backend} package.

The {\tt Make\_Compiler\_Backend} api source code is in \ahrefloc{src/lib/core/internal/mythryl-compiler-compiler.api}{src/lib/core/internal/mythryl-compiler-compiler.api}.

See also:  \ahrefloc{api:Makelib}{Makelib} api.
