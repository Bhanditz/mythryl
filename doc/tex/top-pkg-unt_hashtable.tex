
The standard library {\tt unt\_hashtable} package implements hashtables specialized to unsigned integer keys.

The {\tt unt\_hashtable} package implements the \ahrefloc{api:Typelocked\_Hashtable}{Typelocked\_Hashtable} api.

The {\tt unt\_hashtable} package source code is in \ahrefloc{src/lib/src/unt-hashtable.pkg}{src/lib/src/unt-hashtable.pkg}.

See also:  \ahrefloc{pkg:int\_hashtable}{int\_hashtable}.
