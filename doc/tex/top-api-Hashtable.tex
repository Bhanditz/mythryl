
The standard library {\tt hashtable} api defines access to basic typeagnostic hashtables.

The {\tt hashtable} api is implemented by the \ahrefloc{pkg:hashtable}{hashtable} package.

The {\tt hashtable} api source code is in \ahrefloc{src/lib/src/hashtable.api}{src/lib/src/hashtable.api}.

See also:  \ahrefloc{api:Typelocked\_Hashtable}{Typelocked\_Hashtable}, defined in \ahrefloc{src/lib/src/typelocked-hashtable.api}{src/lib/src/typelocked-hashtable.api}, 
implemented by {\tt typelocked\_hashtable\_g} in \ahrefloc{src/lib/src/typelocked-hashtable-g.pkg}{src/lib/src/typelocked-hashtable-g.pkg}.

See also:  \ahrefloc{api:Hash\_Key}{Hash\_Key}.
