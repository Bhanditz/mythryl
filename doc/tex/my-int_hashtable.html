13.4.47  int_hashtable

The standard library int_hashtable package implements hashtables specialized to integer keys.

The int_hashtable package implements the Typelocked_Hashtable api.

The int_hashtable package source code is in src/lib/src/int-hashtable.pkg.

See also: unt_hashtable.
