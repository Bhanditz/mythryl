
The standard library {\tt substring} package implements functionality allowing 
parts ofh \ahrefloc{pkg:string}{string} instances to be referenced in 
constant time in space.

The {\tt substring} package implements the \ahrefloc{api:Substring}{Substring} api.

The {\tt substring} package source code is in \ahrefloc{src/lib/core/init/substring.pkg}{src/lib/core/init/substring.pkg}, 
proxied in \ahrefloc{src/lib/std/src/substring.pkg}{src/lib/std/src/substring.pkg} and 
\ahrefloc{src/lib/std/src/substring.pkg}{src/lib/std/src/substring.pkg}.
