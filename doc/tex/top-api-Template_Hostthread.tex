
The standard library {\tt Template\_Hostthread} api provides a clone-and-mutate template 
for defining new dedicated server hostthreads.

The {\tt Template\_Hostthread} api is implemented by the \ahrefloc{pkg:template\_hostthread}{template\_hostthread} package.

The {\tt Template\_Hostthread} api source code is in \ahrefloc{src/lib/std/src/hostthread/template-hostthread.api}{src/lib/std/src/hostthread/template-hostthread.api}.
