
The standard library {\tt Line\_Number\_Db} api defines functionality for tracking 
regions in source-code files, primarily for issuing error messages.

The {\tt Line\_Number\_Db} api is implemented by the \ahrefloc{pkg:line\_number\_db}{line\_number\_db} package.

The {\tt Line\_Number\_Db} api source code is in \ahrefloc{src/lib/compiler/front/basics/source/line-number-db.api}{src/lib/compiler/front/basics/source/line-number-db.api}.
