
The standard library {\tt two\_word\_unt} package implements 64-bit unsigned integers and related 
functionality on 32-bit architectures, 126-bit unsigned integers on 64-bit architectures.

The {\tt two\_word\_unt} package implements the \ahrefloc{api:Unt}{Unt} api.

The {\tt two\_word\_unt} package source code is in \ahrefloc{src/lib/std/src/two-word-unt.pkg}{src/lib/std/src/two-word-unt.pkg}.
