
The standard library {\tt List\_Fns} api defines additional convenience functions on lists beyond those included in the \ahrefloc{api:List}{List} api.

The {\tt List\_Fns} api is implemented by the \ahrefloc{pkg:list\_fns}{list\_fns} package.

The {\tt List\_Fns} api source code is in \ahrefloc{src/lib/src/list-fns.api}{src/lib/src/list-fns.api}.

See also: \ahrefloc{api:List}{List}.
