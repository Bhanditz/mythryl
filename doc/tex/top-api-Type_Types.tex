
The compiler {\tt Type\_Types} api exports various types used within 
the compiler for representing types in the program being compiled.

The {\tt Type\_Types} api is implemented by the \ahrefloc{pkg:type\_types}{type\_types} package.

The {\tt Type\_Types} api source code is \ahrefloc{src/lib/compiler/front/typer/types/type-types.api}{src/lib/compiler/front/typer/types/type-types.api}.

Nomenclature: 'typ' is jargon for ``type constructor'', which may be 
informally understood as meaning just ``type''.
