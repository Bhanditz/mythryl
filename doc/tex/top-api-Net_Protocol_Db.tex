
The standard library {\tt Net\_Protocol\_Db} api defines access to basic {\sc TCP/IP} logic to 
look up a protocol by name or by number.

The {\tt Net\_Protocol\_Db} api is implemented by the \ahrefloc{pkg:net\_protocol\_db}{net\_protocol\_db} package.

The {\tt Net\_Protocol\_Db} api source code is in \ahrefloc{src/lib/std/src/socket/net-protocol-db.api}{src/lib/std/src/socket/net-protocol-db.api}.
