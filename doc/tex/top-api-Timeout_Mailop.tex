
The standard library {\tt Timeout\_Mailop} api defines timing out and sleep functionality for Mythryl the interthread communication facility.

The {\tt Timeout\_Mailop} api is implemented by the \ahrefloc{pkg:timeout\_mailop}{timeout\_mailop} package.

The {\tt Timeout\_Mailop} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/timeout-mailop.api}{src/lib/src/lib/thread-kit/src/core-thread-kit/timeout-mailop.api}.

See also: \ahrefloc{api:Mailop}{Mailop} api.
