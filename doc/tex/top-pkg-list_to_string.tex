
The standard library {\tt list\_to\_string} package implements translations between 
Lists and Strings.

The {\tt list\_to\_string} package implements the \ahrefloc{api:List\_To\_String}{List\_To\_String} API.

The {\tt list\_to\_string} package source code is in \ahrefloc{src/lib/src/list-to-string.pkg}{src/lib/src/list-to-string.pkg}.

See also:  \ahrefloc{pkg:string\_to\_list}{string\_to\_list}.
