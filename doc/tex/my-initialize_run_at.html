13.4.43  initialize_run_at

The standard library initialize_run_at package implements support for run_at.

The initialize_run_at package implements the empty api (all functionality is via side-effects at load time).

The initialize_run_at package source code is in src/lib/src/lib/thread-kit/src/glue/initialize-run-at.pkg.
