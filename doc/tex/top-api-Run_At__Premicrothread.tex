
The standard library {\tt Run\_At\_\_Premicrothread} api defines access to functionality for registering 
functions to be run at system startup or shutdown. 

The {\tt Run\_At\_\_Premicrothread} api is implemented by the \ahrefloc{src/lib/std/src/nj/run-at--premicrothread.pkg}{src/lib/std/src/nj/run-at--premicrothread.pkg} package.

The {\tt Run\_At\_\_Premicrothread} api source code is in \ahrefloc{src/lib/std/src/nj/run-at--premicrothread.api}{src/lib/std/src/nj/run-at--premicrothread.api}.
