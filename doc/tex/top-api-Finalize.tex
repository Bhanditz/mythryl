
The standard library {\tt Finalize} api defines access to a facility allowing 
computations to be triggered by garbage collection of designated storage chunks.

You may generate implementations of the {\tt Finalize} api via 
{\tt finalize\_g} in \ahrefloc{src/lib/src/finalize-g.pkg}{src/lib/src/finalize-g.pkg}.

The {\tt Finalize} api source code is in  \ahrefloc{src/lib/src/finalize.api}{src/lib/src/finalize.api}.

See also:   \ahrefloc{api:Finalized\_Chunk}{Finalized\_Chunk}.
