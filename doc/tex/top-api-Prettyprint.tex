
The standard library {\tt Prettyprint} api defines access to simple indented-text formatting.

The {\tt Prettyprint} api is implemented by the \ahrefloc{pkg:prettyprint}{prettyprint} package.

The {\tt Prettyprint} api source code is in \ahrefloc{src/lib/prettyprint/big/src/prettyprint.pkg}{src/lib/prettyprint/big/src/prettyprint.pkg}.

See also: \ahrefloc{api:Simple\_Prettyprinter}{Simple\_Prettyprinter}.
