
The standard library {\tt Linking\_Mapstack} api defines access to one of the three 
core compiler state tables, the other two being 
\ahrefloc{api:Inlining\_Mapstack}{Inlining\_Mapstack} and 
\ahrefloc{api:Symbolmapstack}{Symbolmapstack}.

The {\tt Linking\_Mapstack} api is implemented by the \ahrefloc{pkg:linking\_mapstack}{linking\_mapstack} package.

The {\tt Linking\_Mapstack} api source code is in \ahrefloc{src/lib/compiler/execution/linking-mapstack/linking-mapstack.api}{src/lib/compiler/execution/linking-mapstack/linking-mapstack.api}.
