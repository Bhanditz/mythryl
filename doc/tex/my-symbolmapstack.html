13.5.47  symbolmapstack

The standard library symbolmapstack package implements one of the three core compiler state tables, the other two being linking_mapstack and inlining_mapstack.

The symbolmapstack package implements the Symbolmapstack api.

The symbolmapstack package source code is in src/lib/compiler/front/typer-stuff/symbolmapstack/symbolmapstack.pkg.

The above information is manually maintained and may contain errors.
