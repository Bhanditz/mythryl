
The standard library {\tt scanf} package implements text-scanning functionality 
inspired by the C {\tt scanf()} function.

The {\tt scanf} package implements the \ahrefloc{api:Scanf}{Scanf} API.

The {\tt scanf} package source code is in \ahrefloc{src/lib/src/scanf.pkg}{src/lib/src/scanf.pkg}.
