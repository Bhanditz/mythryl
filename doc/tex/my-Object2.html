14.4.62  Object2

The standard library Object2 api defines the interface to the root class of objects with equality. This is temporary development scaffolding.

The Object2 api is implemented by the object2 package.

The Object2 api source code is in src/lib/src/object2.api.

The above information is manually maintained and may contain errors.
