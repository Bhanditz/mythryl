
The standard library {\tt Catlist} api defines access to lists supporting constant-time concatenation.

The {\tt Catlist} api is implemented by the \ahrefloc{pkg:catlist}{catlist} package.

The {\tt Catlist} api source code is in \ahrefloc{src/lib/std/src/catlist.api}{src/lib/std/src/catlist.api}.
