
The standard library {\tt vector\_slice} package implements access to subvectors within a 
\ahrefloc{pkg:vector}{vector} instance.  It allows referencing such subvectors in 
constant time and space without duplication of data.

The {\tt vector\_slice} package implements the \ahrefloc{api:Vector\_Slice}{Vector\_Slice} api.

The {\tt vector\_slice} package source code is in \ahrefloc{src/lib/std/src/vector-slice.pkg}{src/lib/std/src/vector-slice.pkg}.
