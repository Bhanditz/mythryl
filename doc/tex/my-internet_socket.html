13.4.49  internet_socket

The standard library internet_socket package implements access to TCP and UDP Internet Protocol sockets for multithreaded Mythryl programs.

The internet_socket package implements the Internet_Socket API.

The internet_socket package source code is in src/lib/std/src/socket/internet-socket.pkg.

See also: internet_socket__premicrothread package.
