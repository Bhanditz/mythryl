
The compiler {\tt Sourcecode\_Info} api defines access to functionality for tracking 
the file from which a compilation unit was read.

The {\tt Sourcecode\_Info} api is implemented by the \ahrefloc{pkg:sourcecode\_info}{sourcecode\_info} package.

The {\tt Sourcecode\_Info} api source code is in \ahrefloc{src/lib/compiler/front/basics/source/sourcecode-info.api}{src/lib/compiler/front/basics/source/sourcecode-info.api}.
