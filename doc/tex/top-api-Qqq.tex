
The standard library {\tt Qqq\_Client} api defines access to base-level Qqq functionality.

The {\tt Qqq\_Client} api is implemented by the \ahrefloc{pkg:qqq\_client\_for\_library\_in\_main\_process}{qqq\_client\_for\_library\_in\_main\_process} 
the \ahrefloc{pkg:qqq\_client\_for\_library\_in\_c\_subprocess}{qqq\_client\_for\_library\_in\_c\_subprocess} 
and packages.

The {\tt Qqq\_Client} api source code is in \ahrefloc{src/opt/qqq/src/qqq-client.api}{src/opt/qqq/src/qqq-client.api}.

See also:  \ahrefloc{api:Qqq\_Client\_Driver}{Qqq\_Client\_Driver}.
