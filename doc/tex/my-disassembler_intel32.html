13.5.13  disassembler_intel32

The standard library disassembler_intel32 package implements disassembly of intel32 object code from machine language to assembly language form, primarily in service of inspection of object code and executable files produced by the Mythryl compiler. This is currently a nonfunctional work in progress.

The disassembler_intel32 package implements the Disassembler_Intel32 api.

The disassembler_intel32 package source code is in src/lib/src/disassembler-intel32.pkg.

The above information is manually maintained and may contain errors.
