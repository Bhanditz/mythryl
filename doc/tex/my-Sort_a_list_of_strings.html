5.5.15  Sort a list of strings

Sort a list of strings: 

    eval:  sort  string::(>)   [ "red", "green", "blue" ];
    ["blue", "green", "red"]

Another way: 

    eval:  list_mergesort::sort string::(>)   [ "red", "green", "blue" ];

    ["blue", "green", "red"]
