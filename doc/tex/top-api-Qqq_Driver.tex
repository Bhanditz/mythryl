
The standard library {\tt Qqq\_Client\_Driver} api defines access to base-level Qqq 
driver functionality.

The {\tt Qqq\_Client\_Driver} api is implemented by the \ahrefloc{pkg:qqq\_client\_driver\_for\_library\_in\_c\_subprocess}{qqq\_client\_driver\_for\_library\_in\_c\_subprocess} 
and \ahrefloc{pkg:qqq\_client\_driver\_for\_library\_in\_main\_process}{qqq\_client\_driver\_for\_library\_in\_main\_process} 
packages.

The {\tt Qqq\_Client\_Driver} api source code is in \ahrefloc{src/opt/qqq/src/qqq-client-driver.api}{src/opt/qqq/src/qqq-client-driver.api}.

See also:  \ahrefloc{api:Qqq\_Client}{Qqq\_Client}.
