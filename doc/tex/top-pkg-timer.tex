
The standard library {\tt timer} package implements functionality tracking 
the time consumption of the program.

The {\tt timer} package implements the \ahrefloc{api:Timer}{Timer} API.

The {\tt timer} package source code is in \ahrefloc{src/lib/std/src/timer.pkg}{src/lib/std/src/timer.pkg}, 
a proxy for 
{\tt internal\_timer} from  \ahrefloc{src/lib/std/src/internal-timer.pkg}{src/lib/std/src/internal-timer.pkg}.
