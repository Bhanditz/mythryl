
The compiler {\tt Stampmapstack} api defines access to functionality tracking modules 
in the compilation process.    (In this context, a ``module'' is essentially what you get from 
compiling one sourcefile.)

The {\tt Stampmapstack} api is implemented by the \ahrefloc{pkg:stampmapstack}{stampmapstack} package.

The {\tt Stampmapstack} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/modules/stampmapstack.pkg}{src/lib/compiler/front/typer-stuff/modules/stampmapstack.pkg}.
