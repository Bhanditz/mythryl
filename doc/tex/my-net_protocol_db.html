13.4.76  net_protocol_db

The standard library net_protocol_db package implements basic TCP/IP logic to look up a protocol by name or by number.

The net_protocol_db package implements the Net_Protocol_Db API.

The net_protocol_db package source code is in src/lib/std/src/socket/net-protocol-db.pkg.

The above information is manually maintained and may contain errors.
