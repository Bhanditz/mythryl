
The standard library {\tt Bit\_Flags} api defines access to set operations on bit vectors. 

The {\tt Bit\_Flags} api is implemented in \ahrefloc{src/lib/std/src/bit-flags-g.pkg}{src/lib/std/src/bit-flags-g.pkg}. 
That generic gets invoked in \ahrefloc{src/lib/std/src/psx/posix-io.pkg}{src/lib/std/src/psx/posix-io.pkg}, 
\ahrefloc{src/lib/std/src/psx/posix-file.pkg}{src/lib/std/src/psx/posix-file.pkg}, 
\ahrefloc{src/lib/std/src/psx/posix-process.pkg}{src/lib/std/src/psx/posix-process.pkg}, 
\ahrefloc{src/lib/std/src/psx/posix-file-system-64.pkg}{src/lib/std/src/psx/posix-file-system-64.pkg}, 
\ahrefloc{src/lib/std/src/psx/posix-io-64.pkg}{src/lib/std/src/psx/posix-io-64.pkg} and 
\ahrefloc{src/lib/std/src/psx/posix-tty.pkg}{src/lib/std/src/psx/posix-tty.pkg}.

The {\tt Bit\_Flags} api source code is in \ahrefloc{src/lib/std/src/bit-flags.api}{src/lib/std/src/bit-flags.api}.
