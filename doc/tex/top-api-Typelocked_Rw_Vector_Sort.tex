
The standard library {\tt Typelocked\_Rw\_Vector\_Sort} api defines access to sort algorithms 
defined over mutable vectors specialized to hold a particular value type.

Implementations of the {\tt Typelocked\_Rw\_Vector\_Sort} api may be generated 
using {\tt rw\_vector\_quicksort\_g} from \ahrefloc{src/lib/src/rw-vector-quicksort-g.pkg}{src/lib/src/rw-vector-quicksort-g.pkg}.

The {\tt Typelocked\_Rw\_Vector\_Sort} api source code is in \ahrefloc{src/lib/src/typelocked-rw-vector-sort.api}{src/lib/src/typelocked-rw-vector-sort.api}.
