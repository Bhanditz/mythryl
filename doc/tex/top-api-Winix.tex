
The standard library {\tt Winix} api defines the root cross-platform 
I/O api for multithread programming.

The {\tt Winix} api is implemented by the \ahrefloc{pkg:winix}{winix} package.

The {\tt Winix} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/winix/winix.api}{src/lib/src/lib/thread-kit/src/winix/winix.api}.
