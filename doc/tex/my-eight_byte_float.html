13.2.17  eight_byte_float

The standard library eight_byte_float package implements basic 64-bit floating point arithmetic functionality. It is a proxy for eight_byte_float_guts from src/lib/std/src/eight-byte-float-guts.pkg.

The eight_byte_float package implements the Float API.

The eight_byte_float package source code is in src/lib/std/eight-byte-float.pkg.
