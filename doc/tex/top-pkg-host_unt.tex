
The standard library {\tt host\_unt} package implements host unsigned integer functionality.

The {\tt host\_unt} package implements the \ahrefloc{api:Unt}{Unt} API.

The {\tt host\_unt} package is defined in \ahrefloc{src/lib/std/host-unt.pkg}{src/lib/std/host-unt.pkg} 
ultimately as a wrapper for {\tt one\_word\_unt\_guts} from 
\ahrefloc{src/lib/std/one-word-unt.pkg}{src/lib/std/one-word-unt.pkg}.
