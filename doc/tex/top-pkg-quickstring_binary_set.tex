
The standard library {\tt quickstring\_binary\_set} package implements 
\ahrefloc{pkg:quickstring\_\_premicrothread}{quickstring\_\_premicrothread}-keyed sets.  (Possibly a misnomer, 
since operations like intersection and union are not supported.)

The {\tt quickstring\_binary\_set} package implements the \ahrefloc{api:Set}{Set} API 
via {\tt binary\_sap\_g} from 
\ahrefloc{src/lib/src/binary-set-g.pkg}{src/lib/src/binary-set-g.pkg}.

The {\tt quickstring\_binary\_set} package source code is in \ahrefloc{src/lib/src/quickstring-binary-set.pkg}{src/lib/src/quickstring-binary-set.pkg}.

See also: \ahrefloc{pkg:quickstring\_binary\_map}{quickstring\_binary\_map}.
