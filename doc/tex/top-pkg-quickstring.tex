
The standard library {\tt quickstring} package implements hashed strings supporting 
constant-time equality testing, with a global lock on the shared hashtable for multimicrothread safety.

The {\tt quickstring} package implements the \ahrefloc{api:Quickstring}{Quickstring} api.

The {\tt quickstring} package source code is in \ahrefloc{src/lib/src/quickstring.pkg}{src/lib/src/quickstring.pkg}.

See also:  \ahrefloc{pkg:quickstring\_\_premicrothread}{quickstring\_\_premicrothread}.
