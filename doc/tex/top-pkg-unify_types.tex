
The compiler {\tt unify\_types} package implements {\tt unify} functionality 
for type deduction.  Mythryl uses Prolog-style ``unification'' to perform type 
deduction;  this module implements the needed {\tt unify\_types} operation. 

The {\tt unify\_types} package implements the \ahrefloc{api:Unify\_Types}{Unify\_Types} api.

The {\tt unify\_types} package source code is in \ahrefloc{src/lib/compiler/front/typer/types/unify-types.pkg}{src/lib/compiler/front/typer/types/unify-types.pkg}.
