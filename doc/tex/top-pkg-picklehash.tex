
The compiler {\tt picklehash} package implements unique identifiers for 
compilation units by first serializing and then hashing them.

The {\tt picklehash} package implements the \ahrefloc{api:Picklehash}{Picklehash} API.

The {\tt picklehash} package source code is in \ahrefloc{src/lib/compiler/front/basics/map/picklehash.pkg}{src/lib/compiler/front/basics/map/picklehash.pkg}.
