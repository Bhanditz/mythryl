
The standard library {\tt eight\_byte\_float} package implements basic 64-bit floating point arithmetic functionality. 
It is a proxy for {\tt eight\_byte\_float\_guts} from \ahrefloc{src/lib/std/src/eight-byte-float-guts.pkg}{src/lib/std/src/eight-byte-float-guts.pkg}.

The {\tt eight\_byte\_float} package implements the \ahrefloc{api:Float}{Float} API.

The {\tt eight\_byte\_float} package source code is in \ahrefloc{src/lib/std/eight-byte-float.pkg}{src/lib/std/eight-byte-float.pkg}.
