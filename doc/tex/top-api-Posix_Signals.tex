
The standard library {\tt Posix\_Signals} api defines 
masking, trapping and handling of posix interprocess signals.  It also provides similar handling 
for a pseudo-signal generated by the Mythryl runtime when the garbage collector 
runs.

The {\tt Posix\_Signals} api is implemented by the \ahrefloc{pkg:posix\_signals}{posix\_signals} package.

The {\tt Posix\_Signals} api source code is in \ahrefloc{src/lib/std/src/posix/posix-signals.api}{src/lib/std/src/posix/posix-signals.api}.
