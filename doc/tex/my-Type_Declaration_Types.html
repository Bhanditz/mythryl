14.5.48  Type_Declaration_Types

The compiler Type_Declaration_Types api defines access to type representation and manipulation functionality used in the front end.

The Type_Declaration_Types api is implemented by the type_declaration_types package.

The Type_Declaration_Types api source code is in src/lib/compiler/front/typer-stuff/types/type-declaration-types.api.

The above information is manually maintained and may contain errors.
