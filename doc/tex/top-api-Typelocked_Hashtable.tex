
The standard library {\tt Typelocked\_Hashtable} api defines access to hashtables 
specialized to a particular key type.

The {\tt Typelocked\_Hashtable} api is implemented by the 
\ahrefloc{pkg:int\_hashtable}{int\_hashtable}, 
\ahrefloc{pkg:unt\_hashtable}{unt\_hashtable} 
 and 
\ahrefloc{pkg:quickstring\_hashtable}{quickstring\_hashtable} packages. 

The {\tt Typelocked\_Hashtable} api source code is in \ahrefloc{src/lib/src/typelocked-hashtable.api}{src/lib/src/typelocked-hashtable.api}.
