
The standard library {\tt rw\_float\_vector} package is a proxy for the 
\ahrefloc{pkg:rw\_vector\_of\_eight\_byte\_floats}{rw\_vector\_of\_eight\_byte\_floats} package.

The {\tt rw\_float\_vector} package implements the \ahrefloc{api:Typelocked\_Rw\_Vector}{Typelocked\_Rw\_Vector} API.

The {\tt rw\_float\_vector} package source code is in proxied in \ahrefloc{src/lib/std/rw-float-vector.pkg}{src/lib/std/rw-float-vector.pkg}.
