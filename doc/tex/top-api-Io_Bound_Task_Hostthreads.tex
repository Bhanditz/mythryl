
The standard library {\tt Io\_Bound\_Task\_Hostthreads} api defines access to hostthreads which offload 
I/O-intensive computations from the main threadkit hostthread (or, more generally, any slow 
operations which are not CPU-bound).

The {\tt Io\_Bound\_Task\_Hostthreads} api is implemented by the \ahrefloc{pkg:io\_bound\_task\_hostthreads}{io\_bound\_task\_hostthreads} package.

The {\tt Io\_Bound\_Task\_Hostthreads} api source code is in \ahrefloc{src/lib/std/src/hostthread/io-bound-task-hostthreads.api}{src/lib/std/src/hostthread/io-bound-task-hostthreads.api}.

See also:   \ahrefloc{api:Cpu\_Bound\_Task\_Hostthreads}{Cpu\_Bound\_Task\_Hostthreads}.

See also:   \ahrefloc{api:Io\_Wait\_Hostthread}{Io\_Wait\_Hostthread}.
