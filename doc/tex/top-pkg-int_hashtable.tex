
The standard library {\tt int\_hashtable} package implements hashtables specialized to integer keys.

The {\tt int\_hashtable} package implements the \ahrefloc{api:Typelocked\_Hashtable}{Typelocked\_Hashtable} api.

The {\tt int\_hashtable} package source code is in \ahrefloc{src/lib/src/int-hashtable.pkg}{src/lib/src/int-hashtable.pkg}.

See also:  \ahrefloc{pkg:unt\_hashtable}{unt\_hashtable}.
