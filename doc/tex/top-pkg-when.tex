
The standard library {\tt when} package defines a simpler alternative to 
the \ahrefloc{api:Winix\_Io}{Winix\_Io} interface for finding I/O descriptors 
ready for reading or writing.

The {\tt when} is not intended to replace 
\ahrefloc{api:Winix\_Io}{Winix\_Io} in all applications, just to simplify 
many simple common cases.

The {\tt when} package implements the \ahrefloc{api:When}{When} api.

The {\tt when} package source code is in \ahrefloc{src/lib/src/when.pkg}{src/lib/src/when.pkg}.

See also:  \ahrefloc{api:Winix\_Io}{Winix\_Io}.
