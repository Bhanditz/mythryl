
The standard library {\tt vector\_slice\_of\_chars} package implements slicing over 
\ahrefloc{pkg:vector\_of\_chars}{vector\_of\_chars} instances.  This allows constant-time, 
constant-space references to subvectors within a \ahrefloc{pkg:vector\_of\_chars}{vector\_of\_chars}.

The {\tt vector\_slice\_of\_chars} package implements the \ahrefloc{api:Typelocked\_Vector\_Slice}{Typelocked\_Vector\_Slice} api.

The {\tt vector\_slice\_of\_chars} package source code is in \ahrefloc{src/lib/std/src/vector-slice-of-chars.pkg}{src/lib/std/src/vector-slice-of-chars.pkg}.
