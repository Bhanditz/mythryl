13.4.21  file__premicrothread

The standard library file__premicrothread package is a synonym for winix_text_file_for_posix__premicrothread on Posix-based platforms such as Linux; it will be a synonym for winix_text_file_for_win32 on 32-bit Windows platforms; on other platforms it will be a synonym for a similarly appropriate platform-specific package.

The file__premicrothread package implements the Winix_Text_File_For_Os API.

On posix platforms the file__premicrothread package source code is in src/lib/std/src/posix/file–premicrothread.pkg.

The above information is manually maintained and may contain errors.
