
The standard library {\tt oneshot\_maildrop} package implements in essence concurrency-safe refcells specialized to being set only once.
These are often used for return values from interthread messages.

The {\tt oneshot\_maildrop} package implements the \ahrefloc{api:Oneshot\_Maildrop}{Oneshot\_Maildrop} API.

The {\tt oneshot\_maildrop} package source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/oneshot-maildrop.pkg}{src/lib/src/lib/thread-kit/src/core-thread-kit/oneshot-maildrop.pkg}.

See also: \ahrefloc{pkg:maildrop}{maildrop}.
