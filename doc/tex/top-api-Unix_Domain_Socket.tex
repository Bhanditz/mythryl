
The standard library {\tt Unix\_Domain\_Socket} api supports reading and writing 
of streams and datagrams via unix domain sockets.

The {\tt Unix\_Domain\_Socket} api is implemented by the \ahrefloc{pkg:unix\_domain\_socket}{unix\_domain\_socket} package.

The {\tt Unix\_Domain\_Socket} api source code is in \ahrefloc{src/lib/std/src/socket/unix-domain-socket.api}{src/lib/std/src/socket/unix-domain-socket.api}.

See also: \ahrefloc{api:Unix\_Domain\_Socket\_\_Premicrothread}{Unix\_Domain\_Socket\_\_Premicrothread} api.
