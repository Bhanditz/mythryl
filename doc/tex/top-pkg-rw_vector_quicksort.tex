
The standard library {\tt rw\_vector\_quicksort} package implements an engineered version of quicksort due to 
Bentley and McIlroy.

The {\tt rw\_vector\_quicksort} package implements the \ahrefloc{api:Rw\_Vector\_Sort}{Rw\_Vector\_Sort} API.

The {\tt rw\_vector\_quicksort} package source code is in \ahrefloc{src/lib/src/rw-vector-quicksort.pkg}{src/lib/src/rw-vector-quicksort.pkg}.
