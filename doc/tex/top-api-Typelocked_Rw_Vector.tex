
The standard library {\tt Typelocked\_Rw\_Vector} api defines access to mutable vectors specialized 
to hold a particular element type such as {\tt Char}.

The {\tt Typelocked\_Rw\_Vector} api is implemented by the 
\ahrefloc{pkg:rw\_vector\_of\_chars}{rw\_vector\_of\_chars}, 
\ahrefloc{pkg:rw\_vector\_of\_one\_byte\_unts}{rw\_vector\_of\_one\_byte\_unts} and 
\ahrefloc{pkg:rw\_vector\_of\_eight\_byte\_floats}{rw\_vector\_of\_eight\_byte\_floats} packages.

Additional implementations of the {\tt Typelocked\_Rw\_Vector} api may be generated 
using {\tt typelocked\_rw\_vector\_g} from \ahrefloc{src/lib/src/typelocked-rw-vector-g.pkg}{src/lib/src/typelocked-rw-vector-g.pkg}.

The {\tt Typelocked\_Rw\_Vector} api source code is in \ahrefloc{src/lib/std/src/typelocked-rw-vector.api}{src/lib/std/src/typelocked-rw-vector.api}.

See also: \ahrefloc{api:Typelocked\_Vector}{Typelocked\_Vector}.
