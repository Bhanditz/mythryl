
The standard library {\tt paired\_lists} package implements  
analogues of the list functions {\tt zip}, {\tt fold\_left} etc which operate in parallel 
on pairs of lists instead of on single lists.

The {\tt paired\_lists} package implements the \ahrefloc{api:Paired\_Lists}{Paired\_Lists} API.

The {\tt paired\_lists} package source code is in \ahrefloc{src/lib/std/src/paired-lists.pkg}{src/lib/std/src/paired-lists.pkg}.
