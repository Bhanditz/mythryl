
The standard library {\tt Net\_Service\_Db} api defines access to basic {\sc TCP/IP} logic to 
look up a service by name or by number.

The {\tt Net\_Service\_Db} api is implemented by the \ahrefloc{pkg:net\_service\_db}{net\_service\_db} package.

The {\tt Net\_Service\_Db} api source code is in \ahrefloc{src/lib/std/src/socket/net-service-db.api}{src/lib/std/src/socket/net-service-db.api}.
