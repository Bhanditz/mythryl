
The standard library {\tt int} package implements default-precision fixed-precision integer arithmetic.

The {\tt int} package implements the \ahrefloc{api:Int}{Int} api.

The {\tt int} package source code is in \ahrefloc{src/lib/std/int.pkg}{src/lib/std/int.pkg}, 
which is a proxy for \ahrefloc{src/lib/std/src/tagged-int-guts.pkg}{src/lib/std/src/tagged-int-guts.pkg}.
