
The standard library {\tt float64\_vector\_slice} package implements slicing over 
\ahrefloc{pkg:vector\_of\_eight\_byte\_floats}{vector\_of\_eight\_byte\_floats} instances.  This allows constant-time, 
constant-space references to subvectors within a \ahrefloc{pkg:vector\_of\_eight\_byte\_floats}{vector\_of\_eight\_byte\_floats}.

The {\tt float64\_vector\_slice} package implements the \ahrefloc{api:Typelocked\_Vector\_Slice}{Typelocked\_Vector\_Slice} API.

The {\tt float64\_vector\_slice} package source code is in \ahrefloc{src/lib/std/src/vector-slice-of-eight-byte-floats.pkg}{src/lib/std/src/vector-slice-of-eight-byte-floats.pkg}; 
the bulk of the actual implementation code may be found in \ahrefloc{src/lib/std/src/vector-slice.pkg}{src/lib/std/src/vector-slice.pkg}.
