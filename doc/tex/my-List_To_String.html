14.2.23  List_To_String

The standard library List_To_String api defines access to functionality tranlating Lists to Strings.

The List_To_String api is implemented by the list_to_string package.

The List_To_String api source code is in src/lib/src/list-to-string.api.

See also: String_To_List.The above information is manually maintained and may contain errors. 

api {   list_to_string' :
        {between:String, first:String, last:String, to_string:X -> String} -> List(X ) -> String;
    list_to_string : (X -> String) -> List(X ) -> String;};
