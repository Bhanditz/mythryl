13.4.52  io_startup_and_shutdown

The standard library io_startup_and_shutdown package implements support for automatically closing open I/O streams on exit.

The io_startup_and_shutdown package implements the Io_Startup_And_Shutdown api.

The io_startup_and_shutdown package source code is in src/lib/std/src/io/io-startup-and-shutdown.pkg.

The above information is manually maintained and may contain errors.
