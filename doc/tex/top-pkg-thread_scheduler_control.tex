
The standard library {\tt thread\_scheduler} package implements control of the Mythryl thread scheduler, in particular starting and stopping it.

The {\tt thread\_scheduler\_control} package implements the \ahrefloc{api:Thread\_Scheduler\_Control}{Thread\_Scheduler\_Control} API.

The {\tt thread\_scheduler\_control} package source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/posix/thread-scheduler-control.pkg}{src/lib/src/lib/thread-kit/src/posix/thread-scheduler-control.pkg}.

See also: \ahrefloc{pkg:thread\_scheduler\_is\_running}{thread\_scheduler\_is\_running} package.
