
The compiler {\tt type\_types} package implements various types used within 
the compiler for representing types in the program being compiled.

The {\tt type\_types} package implements the \ahrefloc{api:Type\_Types}{Type\_Types} API.

The {\tt type\_types} package source code is \ahrefloc{src/lib/compiler/front/typer/types/type-types.pkg}{src/lib/compiler/front/typer/types/type-types.pkg}.
