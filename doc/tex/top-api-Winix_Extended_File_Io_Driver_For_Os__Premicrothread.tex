
The standard library {\tt Winix\_Extended\_Io\_Driver\_For\_Os} api defines the low-level api exported by the 
Mythryl platform-specific file I/O driver for a particular operating system.

The {\tt Winix\_Extended\_Io\_Driver\_For\_Os} api is implemented by the {\tt winix\_data\_file\_for\_os\_g} generic from 
\ahrefloc{src/lib/std/src/io/winix-data-file-for-os-g--premicrothread.pkg}{src/lib/std/src/io/winix-data-file-for-os-g--premicrothread.pkg} and 
the {\tt winix\_text\_file\_for\_os\_g} generic from 
\ahrefloc{src/lib/std/src/io/winix-text-file-for-os-g--premicrothread.pkg}{src/lib/std/src/io/winix-text-file-for-os-g--premicrothread.pkg}.

See also: \ahrefloc{api:Winix\_Text\_File\_For\_Os}{Winix\_Text\_File\_For\_Os} api.

See also: \ahrefloc{api:Winix\_Data\_File\_For\_Os}{Winix\_Data\_File\_For\_Os} api.
