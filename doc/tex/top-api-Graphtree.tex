The {\tt Graphtree} api is implemented by the \ahrefloc{src/lib/std/graphtree/graphtree-g.pkg}{src/lib/std/graphtree/graphtree-g.pkg} generic package.

The {\tt Graphtree} api source code is in \ahrefloc{src/lib/std/graphtree/graphtree.api}{src/lib/std/graphtree/graphtree.api}.
