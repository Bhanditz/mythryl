
The standard library {\tt String\_To\_List} api defines access to functionality translating 
Strings to Lists.

The {\tt String\_To\_List} api is implemented by the \ahrefloc{pkg:string\_to\_list}{string\_to\_list} package.

The {\tt String\_To\_List} api source code is in \ahrefloc{src/lib/src/string-to-list.api}{src/lib/src/string-to-list.api}.

See also:  \ahrefloc{api:List\_To\_String}{List\_To\_String}.
