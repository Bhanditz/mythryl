
The {\tt dfa\_engine} package implements one of the 
standard library regular expression evaluation engines.

The {\tt dfa\_engine} package implements the {\tt Regular\_Expression\_Engine} defined in \ahrefloc{src/lib/regex/backend/regular-expression-engine.api}{src/lib/regex/backend/regular-expression-engine.api} API.

The {\tt dfa\_engine} package source code is in \ahrefloc{src/lib/regex/backend/dfa-engine.pkg}{src/lib/regex/backend/dfa-engine.pkg}.
