13.2.12  dir_tree

The standard library dir_tree package implements simple iteration over host file system directory trees, vaguely like posix find.

The dir_tree package implements the Dir_Tree API.

The dir_tree package source code is in src/lib/src/dir-tree.pkg.

See also: symlink_tree.

The above information is manually maintained and may contain errors.
