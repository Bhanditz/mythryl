
The standard library {\tt Internet\_Socket} api defines access to TCP and UDP Internet Protocol sockets for multithreaded Mythryl programs.

The {\tt Internet\_Socket} api is implemented by the \ahrefloc{pkg:internet\_socket}{internet\_socket} package.

The {\tt Internet\_Socket} api source code is in \ahrefloc{src/lib/std/src/socket/internet-socket.api}{src/lib/std/src/socket/internet-socket.api}.

See also:  \ahrefloc{api:Internet\_Socket\_\_Premicrothread}{Internet\_Socket\_\_Premicrothread} package.
