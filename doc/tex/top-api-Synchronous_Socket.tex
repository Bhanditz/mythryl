
The standard library {\tt Synchronous\_Socket} api consists of that subset of the 
\ahrefloc{api:Socket}{Socket} api which includes no non-blocking operations.

The {\tt Synchronous\_Socket} api is not implemented as such.

The {\tt Synchronous\_Socket} api source code is in \ahrefloc{src/lib/std/src/socket/synchronous-socket.api}{src/lib/std/src/socket/synchronous-socket.api}.
