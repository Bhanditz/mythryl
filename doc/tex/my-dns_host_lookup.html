13.4.15  dns_host_lookup

The standard library dns_host_lookup package implements basic low-level TCP/IP functionality to look up internet hosts by name and by address and so forth.

The dns_host_lookup package implements the Dns_Host_Lookup api.

The dns_host_lookup package source code is in src/lib/std/src/socket/dns-host-lookup.pkg.

The above information is manually maintained and may contain errors.
