13.2.26  io_with

The standard library io_with package implements functionality allowing evaluation of a computation in the context of a different input or output stream than the current default.

The io_with package implements the Io_With API.

The io_with package source code is in src/lib/src/io-with.pkg.

The above information is manually maintained and may contain errors.
