
The standard library {\tt Posix\_Socket\_Junk} api defines access to support functionality for posix socket handling.

The {\tt Posix\_Socket\_Junk} api is implemented by the \ahrefloc{pkg:posix\_socket\_junk}{posix\_socket\_junk} package.

The {\tt Posix\_Socket\_Junk} api source code is in \ahrefloc{src/lib/internet/posix-socket-junk.api}{src/lib/internet/posix-socket-junk.api}.
