13.4.131  rw_matrix_of_one_byte_unts

The standard library rw_matrix_of_one_byte_unts package implements mutable matrices specialized to unsigned-byte values.

The rw_matrix_of_one_byte_unts package implements the Typelocked_Rw_Matrix api.
