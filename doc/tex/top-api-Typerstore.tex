
The compiler {\tt Typerstore} api defines access to tracking of 
values during typechecking.

The {\tt Typerstore} api is implemented by the \ahrefloc{pkg:typerstore}{typerstore} package.

The {\tt Typerstore} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/modules/typerstore.api}{src/lib/compiler/front/typer-stuff/modules/typerstore.api}.
