
The standard library {\tt cpu\_timer} package implements basic cpu-time metering 
functionality.

The {\tt cput\_timer} package implements the \ahrefloc{api:Cpu\_Timer}{Cpu\_Timer} api.

The {\tt cpu\_timer} package source code is in \ahrefloc{src/lib/std/src/cpu-timer.pkg}{src/lib/std/src/cpu-timer.pkg}.
