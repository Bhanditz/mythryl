
The standard library {\tt Spawn\_\_Premicrothread} api defines access to functionality for conveniently 
firing up subprocesses.  This functionality is built on top of  
\ahrefloc{api:Posixlib}{Posixlib} {\tt fork} functionality;  it encapsulates most of the 
work required to do a useful {\tt fork} of a subprocess.

The {\tt Spawn\_\_Premicrothread} api is implemented by the \ahrefloc{pkg:spawn\_\_premicrothread}{spawn\_\_premicrothread} package.

The {\tt Spawn\_\_Premicrothread} api source code is in \ahrefloc{src/lib/std/src/posix/spawn--premicrothread.api}{src/lib/std/src/posix/spawn--premicrothread.api}.
