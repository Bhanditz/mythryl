
The standard library {\tt Rw\_Queue} api defines the basic mutable queues used 
throughout the Mythryl codebase.

The {\tt Rw\_Queue} api is implemented by the \ahrefloc{pkg:rw\_queue}{rw\_queue} package.

The {\tt Rw\_Queue} api source code is in \ahrefloc{src/lib/src/rw-queue.api}{src/lib/src/rw-queue.api}.

See also: \ahrefloc{api:Queue}{Queue}.
