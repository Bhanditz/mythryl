
The standard library {\tt Uncaught\_Exception\_Reporting} api defines uncaught-exception reporting for multi-thread Mythryl programs.

The {\tt Uncaught\_Exception\_Reporting} api is implemented by the \ahrefloc{pkg:uncaught\_exception\_reporting}{uncaught\_exception\_reporting} package.

The {\tt Uncaught\_Exception\_Reporting} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/lib/uncaught-exception-reporting.api}{src/lib/src/lib/thread-kit/src/lib/uncaught-exception-reporting.api}.
