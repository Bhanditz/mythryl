
The standard library {\tt suspension} package implements support for lazy evaluation 
(suspensions).

The {\tt suspension} package has no explicit API definition.

The {\tt suspension} package source code is in \ahrefloc{src/lib/std/src/nj/suspension.pkg}{src/lib/std/src/nj/suspension.pkg}.

See also the {\tt lazy} package in \ahrefloc{src/lib/std/src/nj/lazy.pkg}{src/lib/std/src/nj/lazy.pkg}.
