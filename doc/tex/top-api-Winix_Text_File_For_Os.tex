
The standard library {\tt Winix\_Text\_File\_For\_Os} api defines access to basic low-level 
file I/O for multi-threaded programming.

The {\tt Winix\_Text\_File\_For\_Os} api is implemented by \ahrefloc{src/lib/std/src/io/winix-text-file-for-os-g.pkg}{src/lib/std/src/io/winix-text-file-for-os-g.pkg}.

The {\tt Winix\_Text\_File\_For\_Os} api is implemented by \ahrefloc{pkg:winix\_text\_file\_for\_posix}{winix\_text\_file\_for\_posix}.

The {\tt Winix\_Text\_File\_For\_Os} api source code is in \ahrefloc{src/lib/std/src/io/winix-text-file-for-os.api}{src/lib/std/src/io/winix-text-file-for-os.api}.
