
The standard library {\tt Printf\_Field} api defines access to support functionality 
shared by the \ahrefloc{pkg:sfprintf}{sfprintf} and \ahrefloc{pkg:scanf}{scanf} packages.

The {\tt Bool} api is implemented by the \ahrefloc{pkg:bool}{bool} package.

The {\tt Bool} api source code is in \ahrefloc{src/lib/std/src/bool.api}{src/lib/std/src/bool.api}.
