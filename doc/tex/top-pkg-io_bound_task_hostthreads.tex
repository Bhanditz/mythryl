
The standard library {\tt io\_bound\_task\_hostthreads} package implements hostthreads which offload 
I/O-intensive computations from the main threadkit hostthread  (or, more generally, any slow 
operations which are not CPU-bound).

The {\tt io\_bound\_task\_hostthreads} package implements the \ahrefloc{api:Io\_Bound\_Task\_Hostthreads}{Io\_Bound\_Task\_Hostthreads} api.

The {\tt io\_bound\_task\_hostthreads} package source code is in \ahrefloc{src/lib/std/src/hostthread/io-bound-task-hostthreads.pkg}{src/lib/std/src/hostthread/io-bound-task-hostthreads.pkg}.

See also:   \ahrefloc{pkg:cpu\_bound\_task\_hostthreads}{cpy\_bound\_task\_hostthreads}.
