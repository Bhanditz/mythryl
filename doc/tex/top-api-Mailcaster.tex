
The standard library {\tt Mailcaster} api defines access to a one-to-many inter-thread communication mechanism.

The {\tt Mailcaster} api is implemented by the \ahrefloc{pkg:mailcaster}{mailcaster} package.

The {\tt Mailcaster} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/lib/mailcaster.api}{src/lib/src/lib/thread-kit/src/lib/mailcaster.api}.
