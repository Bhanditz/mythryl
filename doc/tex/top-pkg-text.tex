
The standard library {\tt text} package implements access to the functionality of seven different string-related 
packages: 
\ahrefloc{pkg:char}{char},  
\ahrefloc{pkg:string}{string}, 
\ahrefloc{pkg:substring}{substring}, 
\ahrefloc{pkg:vector\_of\_chars}{vector\_of\_chars}, 
\ahrefloc{pkg:rw\_vector\_of\_chars}{rw\_vector\_of\_chars}, 
\ahrefloc{pkg:vector\_slice\_of\_chars}{vector\_slice\_of\_chars}, 
\ahrefloc{pkg:rw\_vector\_slice\_of\_chars}{rw\_vector\_slice\_of\_chars}.

The {\tt text} package implemented the \ahrefloc{api:Text}{Text} api.

The {\tt text} package source code is in \ahrefloc{src/lib/std/src/text.pkg}{src/lib/std/src/text.pkg}.
