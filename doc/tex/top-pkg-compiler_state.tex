
The compiler {\tt compiler\_state} package defines management of the main 
state information needed during compilation of one source file.

The {\tt compiler\_state} package implements the \ahrefloc{api:Compiler\_State}{Compiler\_State} API.

The {\tt compiler\_state} package source code is in \ahrefloc{src/lib/compiler/toplevel/interact/compiler-state.pkg}{src/lib/compiler/toplevel/interact/compiler-state.pkg}.
