
The compiler {\tt Compiler\_Mapstack\_Set} api defines the toplevel symbol table 
functionality used by \ahrefloc{api:Compiler\_State}{Compiler\_State}.

The {\tt Compiler\_Mapstack\_Set} api is implemented by the \ahrefloc{pkg:compiler\_mapstack\_set}{compiler\_mapstack\_set} package.

The {\tt Compiler\_Mapstack\_Set} api source code is in \ahrefloc{src/lib/compiler/toplevel/compiler-state/compiler-mapstack-set.api}{src/lib/compiler/toplevel/compiler-state/compiler-mapstack-set.api}.
