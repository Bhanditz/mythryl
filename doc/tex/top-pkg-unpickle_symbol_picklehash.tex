
The compiler {\tt unpickle\_symbol\_picklehash} package implements a pair of support 
functions for module unpickling.

The {\tt unpickle\_symbol\_picklehash} package implements an anonymous api declared inline in the source.

The {\tt unpickle\_symbol\_picklehash} package source code is in \ahrefloc{src/lib/compiler/front/semantic/pickle/symbol-and-picklehash-unpickling.pkg}{src/lib/compiler/front/semantic/pickle/symbol-and-picklehash-unpickling.pkg}.

See also:  \ahrefloc{pkg:unpickle\_module}{unpickle\_module}.
