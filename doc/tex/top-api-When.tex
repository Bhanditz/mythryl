
The standard library {\tt When} api defines a simpler alternative to 
the \ahrefloc{api:Winix\_Io}{Winix\_Io} interface for finding I/O descriptors 
ready for reading or writing.

{\tt When} is not intended to replace 
\ahrefloc{api:Winix\_Io}{Winix\_Io} in all applications, just to simplify 
many simple common cases.

The {\tt When} api is implemented by the \ahrefloc{pkg:when}{when} package.

The {\tt When} api source code is in \ahrefloc{src/lib/src/when.api}{src/lib/src/when.api}.

See also:  \ahrefloc{api:Winix\_Io}{Winix\_Io}.
