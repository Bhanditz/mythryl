
The standard library {\tt list\_cross\_product} package maps functions over the 
cross product of two lists, which is to say, over all combinations of one element 
from the first list and one from the second.

The {\tt list\_cross\_product} package implements the \ahrefloc{api:List\_Cross\_Product}{List\_Cross\_Product} API.

The {\tt list\_cross\_product} package source code is in \ahrefloc{src/lib/src/list-cross-product.pkg}{src/lib/src/list-cross-product.pkg}.
