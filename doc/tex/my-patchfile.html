13.4.93  patchfile

The standard library patchfile package implements support for patching content into sourcefiles in regions delimited by 

#    # Do not edit this or following lines --- they are autobuilt.
#    ...
#    # Do not edit this or preceding lines --- they are autobuilt.

The patchfile package implements the Patchfile api.

The patchfile package source code is in src/lib/make-library-glue/patchfile.pkg.

See also: patchfiles.

The above information is manually maintained and may contain errors.
