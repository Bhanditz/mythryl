
The standard library {\tt int\_red\_black\_map} package implements key 
to value mappings via the red-black trees, the standard library workhorse.

The {\tt int\_red\_black\_map} package implements the \ahrefloc{api:Map}{Map} api.

The {\tt int\_red\_black\_map} package source code is in \ahrefloc{src/lib/src/int-red-black-map.pkg}{src/lib/src/int-red-black-map.pkg}.
