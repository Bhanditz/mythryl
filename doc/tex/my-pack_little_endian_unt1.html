13.4.91  pack_little_endian_unt1

The standard library pack_little_endian_unt1 package implements access to non-native (in-Mythryl) functionality for serializing and unserializing thirty-two-bit unsigned integer values, which is to say, packing and unpacking them from bytestreams.

The pack_little_endian_unt1 package implements the Pack_Unt API.

The pack_little_endian_unt1 package source code is in src/lib/std/src/pack-little-endian-unt1.pkg.

The above information is manually maintained and may contain errors.
