
The standard library {\tt unix\_domain\_socket\_\_premicrothread} package supports reading and writing 
of streams and datagrams via unix domain sockets.

The {\tt unix\_domain\_socket\_\_premicrothread} package implements the \ahrefloc{api:Unix\_Domain\_Socket}{Unix\_Domain\_Socket} api.

The {\tt unix\_domain\_socket\_\_premicrothread} package source code is in \ahrefloc{src/lib/std/src/socket/unix-domain-socket--premicrothread.pkg}{src/lib/std/src/socket/unix-domain-socket--premicrothread.pkg}.

See also: \ahrefloc{api:Unix\_Domain\_Socket}{Unix\_Domain\_Socket} api.
