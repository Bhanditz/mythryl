
The compiler {\tt Symbol\_Path} api defines access to functionality for 
manipulating lists of \ahrefloc{api:Symbol}{Symbol} instances representing access chains like 
{\tt a::b::c}.

The {\tt Symbol\_Path} api is implemented by the \ahrefloc{pkg:symbol\_path}{symbol\_path} package.

The {\tt Symbol\_Path} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/basics/symbol-path.api}{src/lib/compiler/front/typer-stuff/basics/symbol-path.api}.
