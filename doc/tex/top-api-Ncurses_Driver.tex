
The standard library {\tt Ncurses\_Client\_Driver} api defines access to base-level Ncurses 
driver functionality.

The {\tt Ncurses\_Client\_Driver} api is implemented by the \ahrefloc{pkg:ncurses\_client\_driver\_for\_library\_in\_c\_subprocess}{ncurses\_client\_driver\_for\_library\_in\_c\_subprocess}
and \ahrefloc{pkg:ncurses\_client\_driver\_for\_library\_in\_main\_process}{ncurses\_client\_driver\_for\_library\_in\_main\_process}
packages.

The {\tt Ncurses\_Client\_Driver} api source code is in \ahrefloc{src/glu/ncurses/src/ncurses-client-driver.api}{src/glu/ncurses/src/ncurses-client-driver.api}.

See also:  \ahrefloc{api:Ncurses\_Client}{Ncurses\_Client}.
