
The standard library {\tt Microthread\_Preemptive\_Scheduler} api defines the scheduling queues and preemption mechanisms for microthreaded Mythryl programs.

The {\tt Microthread\_Thread\_Scheduler} api is implemented by the \ahrefloc{pkg:microthread\_preemptive\_scheduler}{microthread\_preemptive\_scheduler} package.

The {\tt Microthread\_Thread\_Scheduler} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/microthread-preemptive-scheduler.api}{src/lib/src/lib/thread-kit/src/core-thread-kit/microthread-preemptive-scheduler.api}.

See also: \ahrefloc{api:Thread\_Scheduler\_Control}{Thread\_Scheduler\_Control} api.
