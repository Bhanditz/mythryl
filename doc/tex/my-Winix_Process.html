14.4.152  Winix_Process

The standard library Winix_Process api defines a sub-api of the Winix api.

The standard library Winix_Process api is implemented in the winix_process package.

The standard library Winix_Process api source code is in src/lib/src/lib/thread-kit/src/winix/winix-process.api.

See also: Winix_Process__Premicrothread.

The above information is manually maintained and may contain errors.
