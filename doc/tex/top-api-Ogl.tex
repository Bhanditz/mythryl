
The standard library {\tt Opengl\_Client} api defines access to base-level Opengl functionality.

The {\tt Opengl\_Client} api is implemented by the \ahrefloc{pkg:opengl\_client\_for\_library\_in\_main\_process}{opengl\_client\_for\_library\_in\_main\_process} 
the \ahrefloc{pkg:opengl\_client\_for\_library\_in\_c\_subprocess}{opengl\_client\_for\_library\_in\_c\_subprocess} 
and packages.

The {\tt Opengl\_Client} api source code is in \ahrefloc{src/opt/opengl/src/opengl-client.api}{src/opt/opengl/src/opengl-client.api}.

See also:  \ahrefloc{api:Opengl\_Client\_Driver}{Opengl\_Client\_Driver}.
