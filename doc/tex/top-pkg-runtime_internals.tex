
The standard library {\tt runtime\_internals} package defines access to functionality 
such as control of the garbage collector.

The {\tt runtime\_internals} package implements the 
\ahrefloc{api:Runtime\_Internals}{Runtime\_Internals} api.

The {\tt runtime\_internals} package source code is in \ahrefloc{src/lib/std/src/nj/runtime-internals.pkg}{src/lib/std/src/nj/runtime-internals.pkg}.
