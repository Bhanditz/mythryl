
The compiler {\tt Symbol} api defines access to the primary compiler frontend 
representation of user source code identifiers.

The {\tt Symbol} api is implemented by the \ahrefloc{pkg:symbol}{symbol} package.

The {\tt Symbol} api source code is in \ahrefloc{src/lib/compiler/front/basics/map/symbol.api}{src/lib/compiler/front/basics/map/symbol.api}.
