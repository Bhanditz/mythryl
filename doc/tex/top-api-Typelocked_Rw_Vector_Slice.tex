
The standard library {\tt Typelocked\_Rw\_Vector\_Slice} api defines access to slices (subvectors) 
of mutable vectors specialized to hold a particular value type.

The {\tt Typelocked\_Rw\_Vector\_Slice} api is implemented 
by {\tt rw\_float64\_vector\_slice} from \ahrefloc{src/lib/std/src/rw-vector-slice-of-eight-byte-floats.pkg}{src/lib/std/src/rw-vector-slice-of-eight-byte-floats.pkg}, 
by {\tt rw\_unt8\_vector\_slice} from  \ahrefloc{src/lib/std/src/rw-vector-slice-of-one-byte-unts.pkg}{src/lib/std/src/rw-vector-slice-of-one-byte-unts.pkg}, 
by {\tt rw\_vector\_slice} from  \ahrefloc{src/lib/std/src/rw-vector-slice.pkg}{src/lib/std/src/rw-vector-slice.pkg}, 
and by {\tt rw\_char\_vector\_slice} from \ahrefloc{src/lib/std/src/rw-vector-slice-of-chars.pkg}{src/lib/std/src/rw-vector-slice-of-chars.pkg}. 

The {\tt Typelocked\_Rw\_Vector\_Slice} api source code is in \ahrefloc{src/lib/std/src/typelocked-rw-vector-slice.api}{src/lib/std/src/typelocked-rw-vector-slice.api}.
