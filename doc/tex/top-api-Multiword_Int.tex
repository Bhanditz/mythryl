
The standard library {\tt Multiword\_Int} api defines access to indefinite-precision integer functions and facilities.

The {\tt Multiword\_Int} api is implemented by the \ahrefloc{pkg:multiword\_int}{multiword\_int} package.
which is a proxy for \ahrefloc{src/lib/std/src/multiword-int-guts.pkg}{src/lib/std/src/multiword-int-guts.pkg}.

The {\tt Multiword\_Int} api source code is in \ahrefloc{src/lib/std/src/multiword-int.api}{src/lib/std/src/multiword-int.api}.

See also:  \ahrefloc{api:Int}{Int}.
