
The standard library {\tt Plain\_Socket} api defines access to vanilla {\sc TCP/IP} socket functionality for multithreaded Mythryl programs.

The {\tt Plain\_Socket} api is implemented by the \ahrefloc{pkg:plain\_socket}{plain\_socket} package.

The {\tt Plain\_Socket} api source code is in \ahrefloc{src/lib/std/src/socket/plain-socket.api}{src/lib/std/src/socket/plain-socket.api}.

See also:  \ahrefloc{api:Plain\_Socket\_\_Premicrothread}{Plain\_Socket\_\_Premicrothread} api.
