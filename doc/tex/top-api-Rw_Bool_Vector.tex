
The standard library {\tt Rw\_Bool\_Vector} api defines access to a mutable vector of booleans.

The {\tt  Rw\_Bool\_Vector} api is implemented by the \ahrefloc{pkg:rw\_bool\_vector}{rw\_bool\_vector} package.

The {\tt Rw\_Bool\_Vector} api source code is in \ahrefloc{src/lib/src/rw-bool-vector.api}{src/lib/src/rw-bool-vector.api}.
