13.4.7  code_segment

The standard library code_segment package implements the code segment representation used by the Mythryl compiler.

The code_segment package implements the Code_Segment api.

The code_segment package source code is in src/lib/compiler/execution/code-segments/code-segment.pkg.

The above information is manually maintained and may contain errors.
