13.4.69  mailqueue

The standard library mailqueue package implements unbounded-length thread-to-thread message queues.

The mailqueue package implements the Mailqueue API.

The mailqueue package source code is in src/lib/src/lib/thread-kit/src/core-thread-kit/mailqueue.pkg.

The above information is manually maintained and may contain errors.
