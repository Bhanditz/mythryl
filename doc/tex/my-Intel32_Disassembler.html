14.5.14  Intel32_Disassembler

The standard library Disassembler_Intel32 api defines access to disassembly of intel32 object code from machine language to assembly language form, primarily in service of inspection of object code and executable files produced by the Mythryl compiler.

The Disassembler_Intel32 api is implemented by the disassembler_intel32 package.

The Disassembler_Intel32 api source code is in src/lib/src/disassembler-intel32.api.

The above information is manually maintained and may contain errors. 

api {
