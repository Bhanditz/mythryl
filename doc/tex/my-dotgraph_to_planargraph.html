13.4.18  dotgraph_to_planargraph

The standard library dotgraph_to_planargraph package implements access to internal graph-visualization functionality.

The dotgraph_to_planargraph package implements the Dotgraph_To_Planagraph api.

The dotgraph_to_planargraph package source code is in src/lib/std/dot/dotgraph-to-planargraph.pkg. The above information is manually maintained and may contain errors.
