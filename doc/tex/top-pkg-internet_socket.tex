
The standard library {\tt internet\_socket} package implements access to TCP and UDP Internet Protocol sockets for multithreaded Mythryl programs.

The {\tt internet\_socket} package implements the \ahrefloc{api:Internet\_Socket}{Internet\_Socket} API.

The {\tt internet\_socket} package source code is in \ahrefloc{src/lib/std/src/socket/internet-socket.pkg}{src/lib/std/src/socket/internet-socket.pkg}.

See also:  \ahrefloc{pkg:internet\_socket\_\_premicrothread}{internet\_socket\_\_premicrothread} package.
