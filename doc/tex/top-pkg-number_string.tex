
The standard library {\tt number\_string} package implements string scanning 
and number extraction functionality.  This is an older package predating the C-inspired  
\ahrefloc{pkg:scanf}{scanf} package.

The {\tt number\_string} package implements the \ahrefloc{api:Number\_String}{Number\_String} api.

The {\tt number\_string} package source code is in \ahrefloc{src/lib/std/src/number-string.pkg}{src/lib/std/src/number-string.pkg}.

See also:   \ahrefloc{pkg:scanf}{scanf}.
