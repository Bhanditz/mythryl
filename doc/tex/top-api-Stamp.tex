
The compiler {\tt Stamp} api defines access to functionality for generating unique 
identifiers labelling various datastructure elements for the code being compiled.

The {\tt Stamp} api is implemented by the \ahrefloc{pkg:stamp}{stamp} package.

The {\tt Stamp} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/basics/stamp.api}{src/lib/compiler/front/typer-stuff/basics/stamp.api}.
