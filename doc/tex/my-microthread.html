13.4.71  microthread

The standard library microthread package implements basic thread functionality for multithreaded Mythryl programs. (Mythryl threads are lightweight userspace constructs not to be confused with posix threads, which are uniformly termed “hostthreads” in Mythryl documents.)

The microthread package implements the Microthread api.

The microthread package source code is in src/lib/src/lib/thread-kit/src/core-thread-kit/microthread.pkg. The above information is manually maintained and may contain errors.
