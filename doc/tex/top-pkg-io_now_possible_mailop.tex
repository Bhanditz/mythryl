
The standard library {\tt io\_now\_possible\_mailop} package implements an internal support facility used by multithreaded Mythryl programs.
It is not normally of direct interest to the application programmer.

The {\tt io\_now\_possible\_mailop} package implements the \ahrefloc{api:Io\_Now\_Possible\_Mailop}{Io\_Now\_Possible\_Mailop} api.

The {\tt io\_now\_possible\_mailop} package source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/io-now-possible-mailop.pkg}{src/lib/src/lib/thread-kit/src/core-thread-kit/io-now-possible-mailop.pkg}.
