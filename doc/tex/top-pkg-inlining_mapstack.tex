
The standard library {\tt inlining\_mapstack} package implements one of the three tables 
making up core compiler state, the other two being the  
\ahrefloc{pkg:linking\_mapstack}{linking\_mapstack} and 
\ahrefloc{pkg:symbolmapstack}{symbolmapstack}.

The {\tt inlining\_mapstack} package implements the \ahrefloc{api:Inlining\_Mapstack}{Inlining\_Mapstack} API.

The {\tt inlining\_mapstack} package source code is in \ahrefloc{src/lib/compiler/toplevel/compiler-state/inlining-mapstack.pkg}{src/lib/compiler/toplevel/compiler-state/inlining-mapstack.pkg}.
