

The standard library {\tt Typelocked\_Vector\_Slice} api defines access to slices of vectors 
specialized to hold a particular type, such as {\tt Char}.  (Vector slices allow constant-time, 
constant-space reference to arbitrary subvectors of an existing vector.)

The {\tt Typelocked\_Vector\_Slice} api is implemented by the 
\ahrefloc{pkg:vector\_slice\_of\_chars}{vector\_slice\_of\_chars}, 
\ahrefloc{pkg:vector\_slice\_of\_one\_byte\_unts}{vector\_slice\_of\_one\_byte\_unts} and 
\ahrefloc{pkg:vector\_slice\_of\_eight\_byte\_floats}{vector\_slice\_of\_eight\_byte\_floats} packages.

The {\tt Typelocked\_Vector\_Slice} api source code is in \ahrefloc{src/lib/std/src/typelocked-vector-slice.api}{src/lib/std/src/typelocked-vector-slice.api}.

See also: \ahrefloc{api:Typelocked\_Vector}{Typelocked\_Vector}.
