13.4.171  unt_hashtable

The standard library unt_hashtable package implements hashtables specialized to unsigned integer keys.

The unt_hashtable package implements the Typelocked_Hashtable api.

The unt_hashtable package source code is in src/lib/src/unt-hashtable.pkg.

See also: int_hashtable.
