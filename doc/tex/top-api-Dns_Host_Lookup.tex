
The standard library {\tt Dns\_Host\_Lookup} api defines access to basic low-level {\sc TCP/IP} 
functionality to look up internet hosts by name and by address and so forth.

The {\tt Dns\_Host\_Lookup} api is implemented by the \ahrefloc{pkg:dns\_host\_lookup}{dns\_host\_lookup} package.

The {\tt Dns\_Host\_Lookup} api source code is in \ahrefloc{src/lib/std/src/socket/dns-host-lookup.api}{src/lib/std/src/socket/dns-host-lookup.api}.
