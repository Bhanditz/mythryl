
The standard library {\tt Inlining\_Mapstack} api defines access to one of the three 
core compiler state tables, the other two being 
\ahrefloc{api:Linking\_Mapstack}{Linking\_Mapstack} and 
\ahrefloc{api:Symbolmapstack}{Symbolmapstack}.

The {\tt Inlining\_Mapstack} api is implemented by the \ahrefloc{pkg:inlining\_mapstack}{inlining\_mapstack} package.

The {\tt Inlining\_Mapstack} api source code is in \ahrefloc{src/lib/compiler/toplevel/compiler-state/inlining-mapstack.api}{src/lib/compiler/toplevel/compiler-state/inlining-mapstack.api}.
