
The standard library {\tt Stream} api defines access to part of the mythryl-yacc parser-generator support infrastructure.

The {\tt Stream} api is implemented by the \ahrefloc{pkg:stream}{stream} package.

The {\tt Stream} api source code is in \ahrefloc{src/app/yacc/lib/base.api}{src/app/yacc/lib/base.api}.
