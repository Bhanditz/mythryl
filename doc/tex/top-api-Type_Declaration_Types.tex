
The compiler {\tt Type\_Declaration\_Types} api defines access to type representation and manipulation 
functionality used in the front end.

The {\tt Type\_Declaration\_Types} api is implemented by the \ahrefloc{pkg:type\_declaration\_types}{type\_declaration\_types} package.

The {\tt Type\_Declaration\_Types} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/types/type-declaration-types.api}{src/lib/compiler/front/typer-stuff/types/type-declaration-types.api}.
