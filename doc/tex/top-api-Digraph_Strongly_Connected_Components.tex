
The standard library {\tt Digraph\_Strongly\_Connected\_Components} api defines access to functionality 
for finding strongly connected components in directed graphs.

Packages implementing the {\tt Digraph\_Strongly\_Connected\_Components} api may be generated 
using the \ahrefloc{src/lib/src/digraph-strongly-connected-components-g.pkg}{src/lib/src/digraph-strongly-connected-components-g.pkg} generic.

The {\tt Digraph\_Strongly\_Connected\_Components} api source code is in \ahrefloc{src/lib/src/digraph-strongly-connected-components.api}{src/lib/src/digraph-strongly-connected-components.api}.
