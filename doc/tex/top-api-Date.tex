
The standard library {\tt Date} 
api defines access to the Posix {\tt strftime()} function and related functionality.

The {\tt Date} api is implemented by the \ahrefloc{pkg:date}{date} package.

The {\tt Datel} api source code is in \ahrefloc{src/lib/std/src/date.api}{src/lib/std/src/date.api}.
