13.4.73  mythryl_callable_c_library_interface

The standard library mythryl_callable_c_library package implements support for Mythryl access to C-level library functionality, e.g. Posix-library functionality.

The mythryl_callable_c_library package implements the Mythryl_Callable_C_Library api.

The mythryl_callable_c_library package source code is in src/lib/std/src/unsafe/mythryl-callable-c-library-interface.pkg.

The above information is manually maintained and may contain errors.
