13.4.94  patchfiles

The standard library patchfiles package implements support for patching content into sets of sourcefiles in regions delimited by 

#    # Do not edit this or following lines --- they are autobuilt.
#    ...
#    # Do not edit this or preceding lines --- they are autobuilt.

The patchfiles package implements the Patchfiles api.

The patchfiles package source code is in src/lib/make-library-glue/patchfiles.pkg.

See also: patchfile.
