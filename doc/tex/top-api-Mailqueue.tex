
The standard library {\tt Mailqueue} api defines access to unbounded-length thread-to-thread message queues.

The {\tt Mailqueue} api is implemented by the \ahrefloc{pkg:mailqueue}{mailqueue} package.

The {\tt Mailqueue} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/mailqueue.api}{src/lib/src/lib/thread-kit/src/core-thread-kit/mailqueue.api}.
