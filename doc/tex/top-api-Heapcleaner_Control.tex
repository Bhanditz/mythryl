
The standard library {\tt Heapcleaner\_Control} api defines access to simple garbage collector control.

The {\tt Heapcleaner\_Control} api is implemented by the \ahrefloc{pkg:heapcleaner\_control}{heapcleaner\_control} package.

The {\tt Heapcleaner\_Control} api source code is in \ahrefloc{src/lib/std/src/nj/heapcleaner-control.api}{src/lib/std/src/nj/heapcleaner-control.api}.
