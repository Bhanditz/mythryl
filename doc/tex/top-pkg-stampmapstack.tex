
The compiler {\tt stampmapstack} package tracks information about modules being 
compiled.  (In this context, a ``module'' is essentially what you get from 
compiling one sourcefile.)

The {\tt stampmapstack} package implements the \ahrefloc{api:Stampmapstack}{Stampmapstack} API.

The {\tt stampmapstack} package source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/modules/stampmapstack.pkg}{src/lib/compiler/front/typer-stuff/modules/stampmapstack.pkg}.
