
The standard library {\tt expanding\_rw\_vector} package implements vectors which 
spontaneously expand as needed to contain requested slots.

The {\tt expanding\_rw\_vector} package implements the \ahrefloc{api:Expanding\_Rw\_Vector}{Expanding\_Rw\_Vector} api.


The {\tt expanding\_rw\_vector} api source code is in \ahrefloc{src/lib/src/expanding-rw-vector.pkg}{src/lib/src/expanding-rw-vector.pkg}.

See also:  {\tt dynamic\_rw\_vector} in \ahrefloc{src/lib/src/dynamic-rw-vector.pkg}{src/lib/src/dynamic-rw-vector.pkg}.
