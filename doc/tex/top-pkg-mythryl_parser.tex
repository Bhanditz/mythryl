
The compiler {\tt mythryl\_parser} package implements the core compiler Mythryl syntax 
parser functionality.

The {\tt mythryl\_parser} package implements the \ahrefloc{api:Mythryl\_Parser}{Mythryl\_Parser} API.

The {\tt mythryl\_parser} package source code is in \ahrefloc{src/lib/compiler/front/parser/main/mythryl-parser.pkg}{src/lib/compiler/front/parser/main/mythryl-parser.pkg}.

The actual Mythryl grammar definition is in 
{\tt src/lib/compiler/front/parser/yacc/mythryl.grammar}; the lexer definition is in 
{\tt src/lib/compiler/front/parser/lex/mythryl.lex}.
