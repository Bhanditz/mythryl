13.4.1  ansi_terminal

The standard library ansi_terminal package implements an ANSI terminal emulator.

The ansi_terminal package implements the Ansi_Terminal api.

The ansi_terminal package source code is in src/lib/src/make-ansi-terminal-escape-sequence.pkg.

The above information is manually maintained and may contain errors.
