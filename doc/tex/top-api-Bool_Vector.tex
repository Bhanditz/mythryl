
The standard library {\tt Bool\_Vector} api defines immutable vectors of Bools. 
(For mutable bitvectors see the \ahrefloc{api:Rw\_Bool\_Vector}{Rw\_Bool\_Vector} API.)

The {\tt Bool\_Vector} api is not separately implemented:  See the {\tt src/lib/src/bool-vector.pkg}.

The {\tt Bool\_Vector} api source code is in \ahrefloc{src/lib/src/bool-vector.api}{src/lib/src/bool-vector.api}.
