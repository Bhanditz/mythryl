
The standard library {\tt Int} api defines access to fixed-precision integer functions and facilities.

The {\tt Int} api is implemented by the \ahrefloc{pkg:int}{int}, 
\ahrefloc{pkg:tagged\_int}{tagged\_int}, 
\ahrefloc{pkg:one\_word\_int}{one\_word\_int} and 
\ahrefloc{pkg:two\_word\_int}{two\_word\_int} packages.

The {\tt Int} api source code is in \ahrefloc{src/lib/std/src/int.api}{src/lib/std/src/int.api}.

See also:  \ahrefloc{api:Multiword\_Int}{Multiword\_Int}.
