14.4.81  Quickstring

The standard library Quickstring api defines access to hashed strings supporting constant-time equality testing.

The Quickstring api is implemented by the quickstring__premicrothread package.

The Quickstring api is also implemented by the quickstring package.

The Quickstring api source code is in src/lib/src/quickstring.api.

The above information is manually maintained and may contain errors. 

api {
    Quickstring;
    from_string : String -> Quickstring;
    from_substring : Substring -> Quickstring;
    to_string : Quickstring -> String;
    same : (Quickstring , Quickstring) -> Bool;
    compare : (Quickstring , Quickstring) -> Order;
    lex_compare : (Quickstring , Quickstring) -> Order;
