
The compiler {\tt Unify\_Types} api defines access to {\tt unify} functionality 
for type deduction.  Mythryl uses Prolog-style ``unification'' to perform type 
deduction;  this module implements the needed {\tt unify\_types} operation. 

The {\tt Unify\_Types} api is implemented by the \ahrefloc{pkg:unify\_types}{unify\_types} package.

The {\tt Unify\_Types} api source code is in \ahrefloc{src/lib/compiler/front/typer/types/unify-types.pkg}{src/lib/compiler/front/typer/types/unify-types.pkg}.
