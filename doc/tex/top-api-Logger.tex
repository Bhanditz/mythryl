
The standard library {\tt Logger} api defines functionality for logging execution of Mythryl programs.

The {\tt Logger} api is implemented by the \ahrefloc{pkg:logger}{logger} package.

The {\tt Logger} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/lib/logger.api}{src/lib/src/lib/thread-kit/src/lib/logger.api}.

For more information see the {\tt src/A.LOGGER.OVERVIEW} file in the source distribution.
