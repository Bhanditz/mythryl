
The standard library {\tt say} package implemetns functions which write string lists 
to various output streams.  This is an older package predating (for example) \ahrefloc{pkg:sfprintf}{sfprintf}.

The {\tt say} package implements the \ahrefloc{api:Say}{Say} API.

The {\tt say} package source code is in \ahrefloc{src/lib/std/src/io/say.pkg}{src/lib/std/src/io/say.pkg}.

See also:  \ahrefloc{pkg:sfprintf}{sfprintf}.
