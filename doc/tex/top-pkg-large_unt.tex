
The standard library {\tt large\_unt} package implements 
unsigned integer arithmetic;  it is currently a proxy for 
{\tt one\_word\_unt\_guts} from \ahrefloc{src/lib/std/one-word-unt.pkg}{src/lib/std/one-word-unt.pkg}.

The {\tt large\_unt} package implements the \ahrefloc{api:Unt}{Unt} API.

The {\tt large\_unt} package is defined in \ahrefloc{src/lib/std/large-unt.pkg}{src/lib/std/large-unt.pkg}.

See also:  \ahrefloc{pkg:large\_int}{large\_int}.

See also:  \ahrefloc{pkg:multiword\_int}{multiword\_int}.
