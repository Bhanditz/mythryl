
The standard library {\tt linking\_mapstack} package implements one of the three mapstacks 
making up core compiler state, the other two being the  
\ahrefloc{pkg:inlining\_mapstack}{inlining\_mapstack} and 
\ahrefloc{pkg:symbolmapstack}{symbolmapstack}.

The {\tt linking\_mapstack} package implements the \ahrefloc{api:Linking\_Mapstack}{Linking\_Mapstack} API.

The {\tt linking\_mapstack} package source code is in \ahrefloc{src/lib/compiler/execution/linking-mapstack/linking-mapstack.pkg}{src/lib/compiler/execution/linking-mapstack/linking-mapstack.pkg}.
