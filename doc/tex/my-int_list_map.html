13.2.19  int_list_map

The standard library int_list_map package implements finite maps on integer keys via sorted lists.

The int_list_map package implements the Map API.

The int_list_map package source code is in src/lib/src/int-list-map.pkg.

The above information is manually maintained and may contain errors.
