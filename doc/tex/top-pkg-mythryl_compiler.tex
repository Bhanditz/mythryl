
The standard library {\tt mythryl\_compiler} package represents the compiler for the current platform.

The {\tt mythryl\_compiler} package implements the {\tt Mythryl\_Compiler} api.

The {\tt mythryl\_compiler} package is defined in \ahrefloc{src/lib/core/compiler/set-mythryl\_compiler-to-mythryl\_compiler\_for\_intel32\_posix.pkg}{src/lib/core/compiler/set-mythryl\_compiler-to-mythryl\_compiler\_for\_intel32\_posix.pkg}.
