
The compiler {\tt Types} api defines access to type representation and manipulation 
functionality used in the front end.

The {\tt Types} api is implemented by the \ahrefloc{pkg:types}{types} package.

The {\tt Types} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/types/types.api}{src/lib/compiler/front/typer-stuff/types/types.api}.
