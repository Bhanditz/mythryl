
The standard library {\tt cpu\_bound\_task\_hostthreads} package implements hostthreads which offload 
CPU-intensive computations from the main threadkit hostthread.

The {\tt cpu\_bound\_task\_hostthreads} package implements the \ahrefloc{api:Cpu\_Bound\_Task\_Hostthreads}{Cpu\_Bound\_Task\_Hostthreads} api.

The {\tt cpu\_bound\_task\_hostthreads} package source code is in \ahrefloc{src/lib/std/src/hostthread/cpu-bound-task-hostthreads.pkg}{src/lib/std/src/hostthread/cpu-bound-task-hostthreads.pkg}.
