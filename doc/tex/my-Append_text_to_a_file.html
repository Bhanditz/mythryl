5.5.47  Append text to a file

Append text to a file: 

    fd = file::open_for_append "foo.txt";
    file::write (fd, "Let none ignorant of geometry enter here.\n"); 
    fprintf      fd  "%s alone has looked on Beauty bare.\n" "Euclid";
    file::flush  fd;
    file::close  fd;
