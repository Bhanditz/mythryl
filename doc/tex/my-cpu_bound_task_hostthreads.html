13.4.8  cpu_bound_task_hostthreads

The standard library cpu_bound_task_hostthreads package implements hostthreads which offload CPU-intensive computations from the main threadkit hostthread.

The cpu_bound_task_hostthreads package implements the Cpu_Bound_Task_Hostthreads api.

The cpu_bound_task_hostthreads package source code is in src/lib/std/src/hostthread/cpu-bound-task-hostthreads.pkg.

See also: io_bound_task_hostthreads.
