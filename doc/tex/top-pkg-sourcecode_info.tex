
The compiler {\tt sourcecode\_info} package implements functionality for tracking 
the file from which a compilation unit was read.

The {\tt sourcecode\_info} package implements the \ahrefloc{api:Sourcecode\_Info}{Sourcecode\_Info} api.

The {\tt sourcecode\_info} package source code is in \ahrefloc{src/lib/compiler/front/basics/source/sourcecode-info.pkg}{src/lib/compiler/front/basics/source/sourcecode-info.pkg}.
