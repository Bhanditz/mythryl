
The standard library {\tt Io\_With} api defines an interface for evaluating 
a computation in the context of a different input or output file than the 
current default.

The {\tt Io\_With} api is implemented by the \ahrefloc{pkg:io\_with}{io\_with} package.

The {\tt Io\_With} api source code is in \ahrefloc{src/lib/src/io-with.api}{src/lib/src/io-with.api}.
