
The standard library {\tt Cpu\_Bound\_Task\_Hostthreads} api defines access to hostthreads which offload 
CPU-intensive computations from the main threadkit hostthread.

The {\tt Cpu\_Bound\_Task\_Hostthreads} api is implemented by the \ahrefloc{pkg:cpu\_bound\_task\_hostthreads}{cpu\_bound\_task\_hostthreads} package.

The {\tt Cpu\_Bound\_Task\_Hostthreads} api source code is in \ahrefloc{src/lib/std/src/hostthread/cpu-bound-task-hostthreads.api}{src/lib/std/src/hostthread/cpu-bound-task-hostthreads.api}.

See also:   \ahrefloc{api:Io\_Bound\_Task\_Hostthreads}{Io\_Bound\_Task\_Hostthreads}.

See also:   \ahrefloc{api:Io\_Wait\_Hostthread}{Io\_Wait\_Hostthread}.
