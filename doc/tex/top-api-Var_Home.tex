The compiler {\tt Var\_Home} api defines access to storage class information for a 
variable.

The {\tt Var\_Home} api is implemented by the \ahrefloc{pkg:var\_home}{var\_home} package.

The {\tt Var\_Home} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/basics/varhome.api}{src/lib/compiler/front/typer-stuff/basics/varhome.api}.
