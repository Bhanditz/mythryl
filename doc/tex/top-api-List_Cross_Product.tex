
The standard library {\tt List\_Cross\_Product} api defines access to functionality 
mapping functions over cross products of lists, which is to say, over all pairs of 
one element from the first list and one from the second.

The {\tt List\_Cross\_Product} api is implemented by the \ahrefloc{pkg:list\_cross\_product}{list\_cross\_product} package.

The {\tt List\_Cross\_Prodcut} api source code is in \ahrefloc{src/lib/src/list-cross-product.api}{src/lib/src/list-cross-product.api}.
