
The standard library {\tt threadkit\_run\_at} package implements hooks for 
functions to be invoked at startup or shutdown of multithread programs.

The {\tt threadkit\_run\_at} package implements the \ahrefloc{api:Threadkit\_Run\_At}{Threadkit\_Run\_At} api.

The {\tt threadkit\_run\_at} package source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/run-at.pkg}{src/lib/src/lib/thread-kit/src/core-thread-kit/run-at.pkg}.
