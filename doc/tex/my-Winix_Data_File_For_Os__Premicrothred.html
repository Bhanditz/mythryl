14.4.143  Winix_Data_File_For_Os__Premicrothred

The standard library Winix_Data_File_For_Os__Premicrothread api is a superset of the Winix_File_For_Os__Premicrothread api.

The Winix_Data_File_For_Os__Premicrothread api is implemented by the package winix_data_file_for_posix__premicrothread in src/lib/std/src/posix/winix-data-file-for-posix–premicrothread.pkg,src/lib/std/src/io/winix-data-file-for-os-g–premicrothread.pkg, and src/lib/std/src/win32/winix-data-file-for-win32.pkg.

The Winix_Data_File_For_Os api source code is in src/lib/std/src/io/winix-data-file-for-os–premicrothread.api.

The above information is manually maintained and may contain errors.
