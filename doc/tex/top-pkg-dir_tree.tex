
The standard library {\tt dir\_tree} package implements simple iteration 
over host file system directory trees, vaguely like posix {\tt find}.

The {\tt dir\_tree} package implements the \ahrefloc{api:Dir\_Tree}{Dir\_Tree} API.

The {\tt dir\_tree} package source code is in \ahrefloc{src/lib/src/dir-tree.pkg}{src/lib/src/dir-tree.pkg}.

See also:   \ahrefloc{pkg:symlink\_tree}{symlink\_tree}.
