
The standard library {\tt Dot\_Graphtree} api defines access to our in-memory representation 
for {\tt Graphvix} {\tt .dot} files.

The {\tt Dot\_Graphtree} api is implemented by the \ahrefloc{pkg:dot\_graphtree}{dot\_graphtree} package.

The {\tt Dot\_Graphtree} api source code is in \ahrefloc{src/lib/std/dot/dot-graphtree.api}{src/lib/std/dot/dot-graphtree.api}.
