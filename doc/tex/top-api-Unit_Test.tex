
The compiler {\tt Unit\_Test} api defines access to the unit testing framework 
invoked by {\tt make check}.

The {\tt Unit\_Test} api is implemented by the \ahrefloc{pkg:unit\_test}{unit\_test} package.

The {\tt Unit\_Test} api source code is in \ahrefloc{src/lib/src/unit-test.api}{src/lib/src/unit-test.api}.
