13.4.11  dfa_engine

The dfa_engine package implements one of the standard library regular expression evaluation engines.

The dfa_engine package implements the Regular_Expression_Engine defined in src/lib/regex/backend/regular-expression-engine.api API.

The dfa_engine package source code is in src/lib/regex/backend/dfa-engine.pkg.
