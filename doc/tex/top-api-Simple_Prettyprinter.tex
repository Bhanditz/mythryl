
The standard library {\tt Simple\_Prettyprinter} api defines 
a fast, simple prettyprinter.  Its major advantage is that 
the complete implementation is only 300 lines of code --- you 
can read and understand it completely in an hour.

The {\tt Simple\_Prettyprinter} api is implemented by the \ahrefloc{pkg:simple\_prettyprinter}{simple\_prettyprinter} package.

The {\tt Simple\_Prettyprinter} api source code is in \ahrefloc{src/lib/prettyprint/simple/simple-prettyprinter.api}{src/lib/prettyprint/simple/simple-prettyprinter.api}.

See also: \ahrefloc{api:Prettyprint}{Prettyprint}.
