
The compiler {\tt Mythryl\_Parser} api defines access to core compiler Mythryl syntax 
parser functionality.

The {\tt Mythryl\_Parser} api is implemented by the \ahrefloc{pkg:mythryl\_parser}{mythryl\_parser} package.

The {\tt Mythryl\_Parser} api source code is in \ahrefloc{src/lib/compiler/front/parser/main/mythryl-parser.pkg}{src/lib/compiler/front/parser/main/mythryl-parser.pkg}.
