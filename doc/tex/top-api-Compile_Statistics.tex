
The compiler {\tt Compile\_Statistics} api defines access to the compiler infrastructure 
for creating, maintaining, aggregating and reporting counts of various compile-related 
events.

The {\tt Compile\_Statistics} api is implemented by the \ahrefloc{pkg:compile\_statistics}{compile\_statistics} package.

The {\tt Compile\_Statistics} api source code is in \ahrefloc{src/lib/compiler/front/basics/stats/compile-statistics.api}{src/lib/compiler/front/basics/stats/compile-statistics.api}.
