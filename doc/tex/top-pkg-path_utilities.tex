
The standard library {\tt path\_utilities} package implements simple cross-platform file-finding functionality.

The {\tt path\_utilities} package implements the \ahrefloc{api:Path\_Utilities}{Path\_Utilitiesl} API.

The {\tt path\_utilities} package source code is in \ahrefloc{src/lib/src/path-utilities.pkg}{src/lib/src/path-utilities.pkg}.
