14.4.112  Timeout_Mailop

The standard library Timeout_Mailop api defines timing out and sleep functionality for Mythryl the interthread communication facility.

The Timeout_Mailop api is implemented by the timeout_mailop package.

The Timeout_Mailop api source code is in src/lib/src/lib/thread-kit/src/core-thread-kit/timeout-mailop.api.

See also: Mailop api.

The above information is manually maintained and may contain errors.
