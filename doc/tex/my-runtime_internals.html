13.4.126  runtime_internals

The standard library runtime_internals package defines access to functionality such as control of the garbage collector.

The runtime_internals package implements the Runtime_Internals api.

The runtime_internals package source code is in src/lib/std/src/nj/runtime-internals.pkg.
