
The standard library {\tt Vector\_Slice} package defines access to subvectors within a 
\ahrefloc{api:Vector}{Vector} instance.  This allows referencing such subvectors in 
constant time and space without duplication of data.

The {\tt Vector\_Slice} api is implemented by the \ahrefloc{pkg:vector\_slice}{vector\_slice} package.

The {\tt Vector\_Slice} api source code is in \ahrefloc{src/lib/std/src/vector-slice.api}{src/lib/std/src/vector-slice.api}.


See also: \ahrefloc{api:Vector}{Vector}.
