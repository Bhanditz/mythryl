
The compiler {\tt symbol\_path} package implements 
manipulation of lists of \ahrefloc{api:Symbol}{Symbol} instances representing access chains like 
{\tt a::b::c}.

The {\tt symbol\_path} package implements the \ahrefloc{api:Symbol\_Path}{Symbol\_Path} API.

The {\tt symbol\_path} package source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/basics/symbol-path.pkg}{src/lib/compiler/front/typer-stuff/basics/symbol-path.pkg}.
