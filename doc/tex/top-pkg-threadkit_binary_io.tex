
The standard library {\tt threadkit\_data\_file} package implements the {\tt data\_file\_io} stack 
 --- access to basic low-level file binary read and write functionality.

The {\tt threadkit\_data\_file} package implements the \ahrefloc{api:Threadkit\_Data\_File\_Io}{Threadkit\_Data\_File\_Io} api.

The {\tt threadkit\_data\_file} package source code is in \ahrefloc{src/lib/std/src/posix/data-file.pkg}{src/lib/std/src/posix/data-file.pkg}.
