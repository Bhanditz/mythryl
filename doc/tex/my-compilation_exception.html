13.5.4  compilation_exception

The compiler compilation_exception package defines the standard compile-error exception.

The compilation_exception package has no explicit API.

The compilation_exception package source code is in src/lib/compiler/front/basics/map/compilation-exception.pkg.

The above information is manually maintained and may contain errors.
