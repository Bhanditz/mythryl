13.5.39  property_list

The compiler property_list package implements an arbitrary datastructure annotation facility patterned after Stephen Week’s annotation library for Mlton.

The property_list package implements the Property_List API.

The property_list package source code is in src/lib/src/property-list.pkg.

The above information is manually maintained and may contain errors.
