13.4.37  host_unt

The standard library host_unt package implements host unsigned integer functionality.

The host_unt package implements the Unt API.

The host_unt package is defined in src/lib/std/host-unt.pkg ultimately as a wrapper for one_word_unt_guts from src/lib/std/one-word-unt.pkg.
