
The standard library {\tt rw\_bool\_vector} implements a mutable vector of Bools. 
(For immutable boolvectors see the \ahrefloc{api:Bool\_Vector}{Bool\_Vector} API.)

The {\tt rw\_bool\_vector} package implements the \ahrefloc{api:Rw\_Bool\_Vector}{Rw\_Bool\_Vector} API.

The {\tt rw\_bool\_vector} package source code is in \ahrefloc{src/lib/src/rw-bool-vector.pkg}{src/lib/src/rw-bool-vector.pkg}.
