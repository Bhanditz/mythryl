
The standard library {\tt Io\_Startup\_And\_Shutdown} api defines tracking of open I/O streams, to close them cleanly at exit.

The {\tt Io\_Startup\_And\_Shutdown} api is implemented by the \ahrefloc{pkg:io\_startup\_and\_shutdown}{io\_startup\_and\_shutdown} package.

The {\tt Io\_Startup\_And\_Shutdown} api source code is in \ahrefloc{src/lib/std/src/io/io-startup-and-shutdown.api}{src/lib/std/src/io/io-startup-and-shutdown.api}.

See also: \ahrefloc{api:Io\_Startup\_And\_Shutdown\_\_Premicrothread}{Io\_Startup\_And\_Shutdown\_\_Premicrothread}.
