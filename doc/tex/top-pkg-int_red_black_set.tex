

The standard library {\tt int\_red\_black\_set} package implements sets of integers using red-black trees.

The {\tt int\_red\_black\_set} package implements the \ahrefloc{api:Set}{Set} api.

The {\tt int\_red\_black\_set} package source code is in \ahrefloc{src/lib/src/int-red-black-set.pkg}{src/lib/src/int-red-black-set.pkg}.
