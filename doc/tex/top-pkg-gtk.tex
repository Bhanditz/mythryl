
The {\tt gtk\_client\_for\_library\_in\_main\_process} package implements base-level access to the Gtk widget 
toolkit.

The {\tt gtk\_client\_for\_library\_in\_main\_process} package implements the \ahrefloc{api:Gtk\_Client}{Gtk\_Client} API.

The {\tt gtk\_client\_for\_library\_in\_main\_process} package source code is in \ahrefloc{src/opt/gtk/src/gtk-client-for-library-in-main-process.pkg}{src/opt/gtk/src/gtk-client-for-library-in-main-process.pkg}; 
it merely invokes the {\tt gtk\_client\_g} generic package from 
\ahrefloc{src/opt/gtk/src/gtk-client-g.pkg}{src/opt/gtk/src/gtk-client-g.pkg}; with the {\tt gtk\_client\_driver\_for\_library\_in\_main\_process} package from 
\ahrefloc{src/opt/gtk/src/gtk-client-driver-for-library-in-main-process.pkg}{src/opt/gtk/src/gtk-client-driver-for-library-in-main-process.pkg} as argument. 

See also:  \ahrefloc{pkg:gtk\_via\_pipe}{gtk\_via\_pipe}.
See also:  \ahrefloc{pkg:easy\_gtk}{easy\_gtk}.
