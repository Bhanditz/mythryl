
The standard library {\tt symlink\_tree} package implements simple iteration 
over host file system directory trees, vaguely like posix {\tt find}.  It 
differs from \ahrefloc{pkg:dir\_tree}{dir\_tree} only in that it follows 
symlinks.

The {\tt symlink\_tree} package implements the \ahrefloc{api:Dir\_Tree}{Dir\_Tree} API.

The {\tt symlink\_tree} package source code is in \ahrefloc{src/lib/src/symlink-tree.pkg}{src/lib/src/symlink-tree.pkg}.

See also:   \ahrefloc{pkg:dir\_tree}{dir\_tree}.
