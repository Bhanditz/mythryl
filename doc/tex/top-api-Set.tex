
The standard library {\tt Set} api defines an interface for datastructures  
which implement set operations, typically via some sort of binary tree 
datastructure.

The {\tt Set} api is implemented by the 
\ahrefloc{pkg:int\_binary\_set}{int\_binary\_set}, 
\ahrefloc{pkg:int\_list\_set}{int\_list\_set} 
\ahrefloc{pkg:unt\_red\_black\_set}{unt\_red\_black\_set}, 
\ahrefloc{pkg:int\_red\_black\_set}{int\_red\_black\_set} and 
\ahrefloc{pkg:string\_set}{string\_set} 
packages.

Red-black trees are the de facto standard Mythryl balanced binary 
tree type, hence 
\ahrefloc{pkg:int\_red\_black\_set}{int\_red\_black\_set} may be considered 
the standard tree-based integer Set implementation and 
\ahrefloc{pkg:string\_set}{string\_set} the standard tree-based string Set 
implementation.

The {\tt Set} api source code is in \ahrefloc{src/lib/src/set.api}{src/lib/src/set.api}.
