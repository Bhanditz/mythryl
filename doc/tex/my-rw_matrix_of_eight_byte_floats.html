13.4.132  rw_matrix_of_eight_byte_floats

The standard library rw_matrix_of_eight_byte_floats package implements mutable matrices specialized to 64-bit float values.

The rw_matrix_of_eight_byte_floats package implements the Typelocked_Rw_Matrix api.
