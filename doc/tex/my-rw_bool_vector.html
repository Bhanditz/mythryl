13.4.127  rw_bool_vector

The standard library rw_bool_vector implements a mutable vector of Bools. (For immutable boolvectors see the Bool_Vector API.)

The rw_bool_vector package implements the Rw_Bool_Vector API.

The rw_bool_vector package source code is in src/lib/src/rw-bool-vector.pkg.

The above information is manually maintained and may contain errors.
