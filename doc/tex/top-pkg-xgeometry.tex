
The standard library {\tt xgeometry} package implements the basic geometry types and operations.

The {\tt xgeometry} package implements the \ahrefloc{api:Xgeometry}{Xgeometry} api.

The {\tt xgeometry} package source code is in \ahrefloc{src/lib/std/2d/xgeometry.pkg}{src/lib/std/2d/xgeometry.pkg}.

See also:  \ahrefloc{pkg:xgeometry\_float}{xgeometry\_float} package.
