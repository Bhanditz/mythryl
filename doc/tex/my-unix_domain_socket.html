13.4.170  unix_domain_socket

The standard library unix_domain_socket package supports reading and writing of streams and datagrams via unix domain sockets.

The unix_domain_socket package implements the Unix_Domain_Socket api.

The unix_domain_socket package source code is in src/lib/std/src/socket/unix-domain-socket.pkg.

See also: unix_domain_socket__premicrothread package. The above information is manually maintained and may contain errors.
