
The compiler {\tt unify\_typoids} package implements {\tt unify} functionality 
for type inference.  Mythryl uses Prolog-style ``unification'' to perform type 
inference;  this module implements the needed {\tt unify\_typoids} operation. 

The {\tt unify\_typoids} package implements the \ahrefloc{api:Unify\_Typoids}{Unify\_Typoids} api.

The {\tt unify\_typoids} package source code is in \ahrefloc{src/lib/compiler/front/typer/types/unify-typoids.pkg}{src/lib/compiler/front/typer/types/unify-typoids.pkg}.
