
The {\tt Typelocked\_Rw\_Matrix} api defines the external interface to the {\tt rw\_matrix\_of\_one\_byte\_unts} and {\tt rw\_matrix\_of\_eight\_byte\_floats} packages.

The {\tt Typelocked\_Rw\_Matrix} api is implemented by the \ahrefloc{pkg:rw\_matrix\_of\_one\_byte\_unts}{rw\_matrix\_of\_one\_byte\_unts} and \ahrefloc{pkg:rw\_matrix\_of\_eight\_byte\_floats}{rw\_matrix\_of\_eight\_byte\_floats} packages.

The {\tt Typelocked\_Rw\_Matrix} api source code is \ahrefloc{src/lib/std/src/typelocked-rw-matrix.api}{src/lib/std/src/typelocked-rw-matrix.api}.
