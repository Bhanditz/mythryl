13.4.152  suspension

The standard library suspension package implements support for lazy evaluation (suspensions).

The suspension package has no explicit API definition.

The suspension package source code is in src/lib/std/src/nj/suspension.pkg.

See also the lazy package in src/lib/std/src/nj/lazy.pkg.

The above information is manually maintained and may contain errors.
