
The compiler {\tt heap\_debug} package implements access to compiler infrastructure 
for debugging heap corruption problems. 

The {\tt heap\_debug} package implements the \ahrefloc{api:Heap\_Debug}{Heap\_Debug} API. 

The {\tt heap\_debug} package source code is in \ahrefloc{src/lib/std/src/nj/heap-debug.pkg}{src/lib/std/src/nj/heap-debug.pkg}.
