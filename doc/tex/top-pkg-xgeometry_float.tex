
The standard library {\tt xgeometry\_float} package implements a floating-point version of the basic geometry types and operations from \ahrefloc{pkg:xgeometry}{xgeometry} api.

The {\tt xgeometry\_float} package currently lacks a proper separate api definition.

The {\tt xgeometry\_float} package source code is in \ahrefloc{src/lib/std/2d/xgeometry-float.pkg}{src/lib/std/2d/xgeometry-float.pkg}.

See also:  \ahrefloc{pkg:xgeometry}{xgeometry} package.
