14.4.156  Winix_Pure_Text_File_For_Os

The standard library Winix_Pure_Text_File_For_Os api is a sub-api of the Winix_File_For_Os api.

The Winix_Pure_Text_File_For_Os api is implemented by the src/lib/std/src/io/winix-text-file-for-os-g.pkg package.

The Winix_Pure_Text_File_For_Os api source code is in src/lib/std/src/io/winix-pure-text-file-for-os.api.

See also: Winix_Pure_File_For_Os. The above information is manually maintained and may contain errors.
