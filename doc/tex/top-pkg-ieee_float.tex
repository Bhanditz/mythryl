
The standard library {\tt ieee\_float} package implements {\sc IEEE} 
standard floating point 
functionality beyond core arithmetic functions (which are implemented in core modules).

The {\tt ieee\_float} package implements the \ahrefloc{api:Ieee\_Float}{Ieee\_Float} API.

The {\tt ieee\_float} package source code is in \ahrefloc{src/lib/std/src/ieee-float.pkg}{src/lib/std/src/ieee-float.pkg}.
