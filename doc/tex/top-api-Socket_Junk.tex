
The standard library {\tt Socket\_Junk} api defines additional convenience functions on sockets, beyond those defined in  \ahrefloc{api:Socket\_\_Premicrothread}{Socket\_\_Premicrothread}.

The {\tt Socket\_Junk} api is implemented by the \ahrefloc{pkg:socket\_junk}{socket\_junk} package.

The {\tt Socket\_Junk} api source code is in \ahrefloc{src/lib/internet/socket-junk.api}{src/lib/internet/socket-junk.api}.
