
The standard library {\tt spawn} package implements functionality for conveniently 
firing up subprocesses in multithreaded Mythryl programs.

The {\tt spawn} package implements the \ahrefloc{api:Threadkit\_Spawn}{Threadkit\_Spawn} API.

The {\tt spawn} package source code is in \ahrefloc{src/lib/std/src/posix/spawn.pkg}{src/lib/std/src/posix/spawn.pkg}.
