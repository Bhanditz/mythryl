13.4.157  thread_scheduler_control

The standard library thread_scheduler package implements control of the Mythryl thread scheduler, in particular starting and stopping it.

The thread_scheduler_control package implements the Thread_Scheduler_Control API.

The thread_scheduler_control package source code is in src/lib/src/lib/thread-kit/src/posix/thread-scheduler-control.pkg.

See also: thread_scheduler_is_running package.

The above information is manually maintained and may contain errors.
