
The standard library {\tt Root\_Object} api defines the interface of the root object class for oop subclassing purposes.

The {\tt Root\_Object} api is implemented by the \ahrefloc{pkg:root\_object}{root\_object} package.

The {\tt Root\_Object} api source code is in \ahrefloc{src/lib/src/root-object.api}{src/lib/src/root-object.api}.
