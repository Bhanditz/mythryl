
The standard library {\tt Symbolmapstack} api defines access to one of the three 
core compiler state tables, the other two being 
\ahrefloc{api:Linking\_Mapstack}{Linking\_Mapstack} and 
\ahrefloc{api:Inlining\_Mapstack}{Inlining\_Mapstack}.

The {\tt Symbolmapstack} api is implemented by the \ahrefloc{pkg:symbolmapstack}{symbolmapstack} package.

The {\tt Symbolmapstack} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/symbolmapstack/symbolmapstack.api}{src/lib/compiler/front/typer-stuff/symbolmapstack/symbolmapstack.api}.
