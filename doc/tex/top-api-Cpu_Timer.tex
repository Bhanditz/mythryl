
The standard library {\tt Cpu\_Timer} api defines access to basic cpu time consumption 
metering functionality.

The {\tt Cpu\_Timer} api is implemented by the \ahrefloc{pkg:cpu\_timer}{cpu\_timer} package.

The {\tt Cpu\_Timer} api source code is in \ahrefloc{src/lib/std/src/cpu-timer.api}{src/lib/std/src/cpu-timer.api}.
