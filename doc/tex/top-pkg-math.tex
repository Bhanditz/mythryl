
The standard library {\tt math} package implements basic mathematical functionality, 
in particular trig and log functions.

The {\tt math} package implements the \ahrefloc{api:Math}{Math} API.

The intel32 {\tt math} package source code is in \ahrefloc{src/lib/std/src/math64-intel32.pkg}{src/lib/std/src/math64-intel32.pkg} 
which gets pulled into several places, notably 
{\tt float64\_guts} in \ahrefloc{src/lib/std/src/eight-byte-float-guts.pkg}{src/lib/std/src/eight-byte-float-guts.pkg} 
which gets proxied as {\tt float64} in 
\ahrefloc{src/lib/std/eight-byte-float.pkg}{src/lib/std/eight-byte-float.pkg} which gets 
proxied as (e.g.) {\tt float} and {\tt large\_float} in 
\ahrefloc{src/lib/std/types-only/bind-largest32.pkg}{src/lib/std/types-only/bind-largest32.pkg}.
