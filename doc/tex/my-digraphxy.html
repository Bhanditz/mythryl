13.4.13  digraphxy

The standard library digraphxy package implements fully persistent directed graphs with two type variables (’X’, ’Y’) for node and edge contents.

The digraphxy package implements the Digraphxy api.

The digraphxy package source code is in src/lib/src/digraphxy.pkg.

See also: digraph.
