
The standard library {\tt Opencv\_Client} api defines access to base-level Opencv functionality.

The {\tt Opencv\_Client} api is implemented by the \ahrefloc{pkg:opencv\_client\_for\_library\_in\_main\_process}{opencv\_client\_for\_library\_in\_main\_process} 
the \ahrefloc{pkg:opencv\_client\_for\_library\_in\_c\_subprocess}{opencv\_client\_for\_library\_in\_c\_subprocess} 
and packages.

The {\tt Opencv\_Client} api source code is in \ahrefloc{src/opt/opencv/src/opencv-client.api}{src/opt/opencv/src/opencv-client.api}.

See also:  \ahrefloc{api:Opencv\_Client\_Driver}{Opencv\_Client\_Driver}.
