13.4.101  platform_properties

The standard library platform_properties package implements access to information on the underlying host hardware architecture and operating system.

The platform_properties package implements the Platform_Properties.

The platform_properties package source code is in src/lib/std/src/nj/platform-properties.pkg.

The above information is manually maintained and may contain errors.
