
The compiler {\tt Unparse\_Type} api defines access to functionality dumping type information 
for debugging and documentation purposes.

The {\tt Unparse\_Type} api is implemented by the \ahrefloc{pkg:unparse\_type}{unparse\_type} package.

The {\tt Unparse\_Type} api source code is in \ahrefloc{src/lib/compiler/front/typer/print/unparse-type.pkg}{src/lib/compiler/front/typer/print/unparse-type.pkg}.
