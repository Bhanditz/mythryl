
The standard library {\tt makelib} package implements top-level Mythryl make / compile functionality.

The {\tt makelib} package implements the \ahrefloc{api:Makelib}{Makelib} API.

The {\tt makelib} package source code is in \ahrefloc{src/lib/core/makelib/makelib.pkg}{src/lib/core/makelib/makelib.pkg}, 
which is a wrapper for \ahrefloc{src/lib/core/internal/makelib-internal.pkg}{src/lib/core/internal/makelib-internal.pkg};  the 
interesting code is in the {\tt makelib\_g} definition in  
\ahrefloc{src/app/makelib/main/makelib-g.pkg}{src/app/makelib/main/makelib-g.pkg}.
