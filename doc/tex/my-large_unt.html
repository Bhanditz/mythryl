13.4.59  large_unt

The standard library large_unt package implements unsigned integer arithmetic; it is currently a proxy for one_word_unt_guts from src/lib/std/one-word-unt.pkg.

The large_unt package implements the Unt API.

The large_unt package is defined in src/lib/std/large-unt.pkg.

See also: large_int.
