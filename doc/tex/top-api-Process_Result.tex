
The standard library {\tt Process\_Result} api defines support functionality for the \ahrefloc{pkg:process\_deathwatch}{process\_deathwatch} package.

The {\tt Process\_Result} api is implemented by the \ahrefloc{pkg:process\_result}{process\_result} package.

The {\tt Process\_Result} api source code is in \ahrefloc{src/lib/std/src/threadkit/process-result.api}{src/lib/std/src/threadkit/process-result.api}.

See also:  \ahrefloc{api:Process\_Deathwatch}{Process\_Deathwatch} api.
