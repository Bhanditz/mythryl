The standard library {\tt Process\_Commandline} api defines access to basic commandline parsing.

The {\tt Process\_Commandline} api is implemented by the \ahrefloc{pkg:process\_commandline}{process\_commandline} package.

The {\tt Process\_Commandline} api source code is in \ahrefloc{src/lib/src/process-commandline.api}{src/lib/src/process-commandline.api}.
