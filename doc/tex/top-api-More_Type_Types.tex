
The compiler {\tt More\_Type\_Types} api exports various types used within 
the compiler for representing types in the program being compiled.

The {\tt More\_Type\_Types} api is implemented by the \ahrefloc{pkg:more\_type\_types}{more\_type\_types} package.

The {\tt More\_Type\_Types} api source code is \ahrefloc{src/lib/compiler/front/typer/types/more-type-types.api}{src/lib/compiler/front/typer/types/more-type-types.api}.
