13.4.38  ieee_float

The standard library ieee_float package implements IEEE standard floating point functionality beyond core arithmetic functions (which are implemented in core modules).

The ieee_float package implements the Ieee_Float API.

The ieee_float package source code is in src/lib/std/src/ieee-float.pkg.

The above information is manually maintained and may contain errors.
