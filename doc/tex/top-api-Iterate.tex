
The standard library {\tt Iterate} api defines the {\tt iterate} {\tt repeat} and {\tt forloop} iterator functions.

The {\tt Iterate} api is implemented by the \ahrefloc{pkg:iterate}{iterate} package.

The {\tt Iterate} api source code is in \ahrefloc{src/lib/src/iterate.api}{src/lib/src/iterate.api}.
