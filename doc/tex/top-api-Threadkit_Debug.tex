
The standard library {\tt Threadkit\_Debug} api defines debugging support for the threadkit core.

The {\tt Threadkit\_Debug} api is implemented by the \ahrefloc{pkg:threadkit\_debug}{threadkit\_debug} package.

The {\tt Threadkit\_Debug} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/threadkit-debug.api}{src/lib/src/lib/thread-kit/src/core-thread-kit/threadkit-debug.api}.
