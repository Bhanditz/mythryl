14.4.79  Priority

The standard library Priority api provides an argument API for generics which generate priority queues.

The Priority api source code is in src/lib/src/priority.api.

See also: Priority_Queue. The above information is manually maintained and may contain errors. 

api {
    Priority;
    Item;
    compare : (Priority , Priority) -> Order;
    priority : Item -> Priority;};
