14.4.159  Winix_Text_File_For_Os__Premicrothread

The standard library Winix_Text_File_For_Os__Premicrothread api defines basic platform-dependent text-mode file I/O.

The Winix_Text_File_For_Os__Premicrothread api is implemented by the winix_text_file_for_posix__premicrothread package on posix platforms.

The Winix_Text_File_For_Os__Premicrothread api source code is in src/lib/std/src/io/winix-text-file-for-os–premicrothread.api.

The above information is manually maintained and may contain errors.
