
The standard library {\tt Wallclock\_Timer} api defines access absolute-time functionality.

The {\tt Wallclock\_Timer} api is implemented by the \ahrefloc{pkg:wallclock\_timer}{wallclock\_timer} package.

The {\tt Wallclock\_Timer} api source code is in \ahrefloc{src/lib/std/src/wallclock-timer.api}{src/lib/std/src/wallclock-timer.api}.
