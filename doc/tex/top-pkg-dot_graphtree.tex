
The standard library {\tt dot\_graphtree} package implements in-memory representation 
of {\tt Graphviz} {\tt .dot} files.

The {\tt dot\_graphtree} package implements the \ahrefloc{api:Dot\_Graphtree}{Dot\_Graphtree} api.

The {\tt dot\_graphtree} package source code is in \ahrefloc{src/lib/std/dot/dot-graphtree.pkg}{src/lib/std/dot/dot-graphtree.pkg}.
