14.4.148  Winix_File_For_Os__Premicrothread

The standard library Winix_File_For_Os__Premicrothread api defines a cross-platform interface file I/O interface which is implemented by platform-specific code.

The Winix_File_For_Os__Premicrothread api is include-ed by src/lib/std/src/io/winix-data-file-for-os–premicrothread.api.
