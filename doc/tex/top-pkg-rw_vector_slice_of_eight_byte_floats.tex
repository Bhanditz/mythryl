
The standard library {\tt rw\_float64\_vector\_slice} package implements access to subvectors within a 
\ahrefloc{pkg:rw\_vector\_of\_eight\_byte\_floats}{rw\_vector\_of\_eight\_byte\_floats} instance.  It allows referencing such subvectors in 
constant time and space without duplication of data.

The {\tt rw\_float64\_vector\_slice} package implements the \ahrefloc{api:Typelocked\_Rw\_Vector\_Slice}{Typelocked\_Rw\_Vector\_Slice} API.

The {\tt rw\_float64\_vector\_slice} package source code is in \ahrefloc{src/lib/std/src/rw-vector-slice-of-eight-byte-floats.pkg}{src/lib/std/src/rw-vector-slice-of-eight-byte-floats.pkg} 
and proxied in \ahrefloc{src/lib/std/rw-float-vector-slice.pkg}{src/lib/std/rw-float-vector-slice.pkg}.
