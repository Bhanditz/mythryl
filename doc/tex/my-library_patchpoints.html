13.4.62  library_patchpoints

The standard library library_patchpoints package tracks lists of files and patchpoints needed by the library-glue dis/integration scripts etc.

The library_patchpoints package source code is in src/lib/make-library-glue/library-patchpoints.pkg.

The above information is manually maintained and may contain errors.
