13.4.64  logger

The standard library logger package implements functionality for logging execution of Mythryl programs.

The logging package implements by the Logger api.

The logging package source code is in src/lib/src/lib/thread-kit/src/lib/logger.pkg.

For more information see the src/A.LOGGER.OVERVIEW file in the distribution sourcetree.

The above information is manually maintained and may contain errors.
