
The standard library {\tt unt\_red\_black\_set} package implements sets of unsigned integers 
using red-black trees.

The {\tt unt\_red\_black\_set} package implements the \ahrefloc{api:Set}{Set} api.

The {\tt unt\_red\_black\_set} package source code is in \ahrefloc{src/lib/src/unt-red-black-set.pkg}{src/lib/src/unt-red-black-set.pkg}.

See also:  \ahrefloc{pkg:int\_red\_black\_set}{int\_red\_black\_set}.
