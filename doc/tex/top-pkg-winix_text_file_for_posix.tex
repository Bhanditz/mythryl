file I/O for multi-microthreaded programming.

The {\tt winix\_text\_file\_for\_posix} package implements the \ahrefloc{api:Threadkit\_Winix\_Text\_File\_For\_Os}{Threadkit\_Winix\_Text\_File\_For\_Os} api.
