## standard-x11-atoms.pkg
#
# Atoms are short integer representations
# of strings maintained by the X server.
#
# Here we implement the standard set of
# atoms defined by the X Inter-Client
# Communication Convention (ICCC),
# extracted from X11/Xatom.h.
#
# We use atom_primary to label standard
# atom PRIMARY etc.
#
# See also:
#
#     src/lib/x-kit/xclient/src/iccc/atom-imp-old.pkg
#     src/lib/x-kit/xclient/src/iccc/atom-old.pkg

# Compiled by:
#     src/lib/x-kit/xclient/xclient-internals.sublib
