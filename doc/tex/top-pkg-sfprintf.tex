
The standard library {\tt sfprintf} package implements a text formatting facility inspired 
by the C {\tt printf()} function.

The standard toplevel {\tt printf} function is based upon the {\tt 
sfprintf} package with assists from the Mythryl parser and the {\tt 
printf\_format\_string\_to\_raw\_syntax} package from 
\ahrefloc{src/lib/compiler/front/parser/raw-syntax/printf-format-string-to-raw-syntax.pkg}{src/lib/compiler/front/parser/raw-syntax/printf-format-string-to-raw-syntax.pkg}.

The {\tt sfprintf} package implements the \ahrefloc{api:Sfprintf}{Sfprintf} API.

The {\tt sfprintf} package source code is in \ahrefloc{src/lib/src/sfprintf.pkg}{src/lib/src/sfprintf.pkg}.

For an alternate {\tt printf()} inspired package, see \ahrefloc{pkg:printf\_combinator}{printf\_combinator}.

See also:  \ahrefloc{pkg:printf\_field}{printf\_field}.
