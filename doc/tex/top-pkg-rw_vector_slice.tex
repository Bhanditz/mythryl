
The standard library {\tt rw\_vector\_slice} package implements access to subvectors within a 
\ahrefloc{pkg:rw\_vector}{rw\_vector} instance.  It allows referencing such subvectors in 
constant time and space without duplication of data.

The {\tt rw\_vector\_slice} package implements the \ahrefloc{api:Rw\_Vector\_Slice}{Rw\_Vector\_Slice} API.

The {\tt rw\_vector\_slice} package source code is in \ahrefloc{src/lib/std/src/rw-vector-slice.pkg}{src/lib/std/src/rw-vector-slice.pkg}.
