
The {\tt mythryl\_compiler} package provides top-level access to Mythryl compiler 
functionality for non-compiler software.

The {\tt mythryl\_compiler} package implements the {\tt Mythryl\_Compiler} API defined in 
\ahrefloc{src/lib/compiler/toplevel/compiler/mythryl-compiler.api}{src/lib/compiler/toplevel/compiler/mythryl-compiler.api}.

On intel32 Linux the {\tt mythryl\_compiler} package gets defined \ahrefloc{src/lib/core/compiler/set-mythryl_compiler-to-mythryl_compiler_for_intel32_posix.pkg}{src/lib/core/compiler/set-mythryl_compiler-to-mythryl_compiler_for_intel32_posix.pkg} 
as {\tt mythryl\_compiler\_for\_intel32\_posix} which in turn comes from \ahrefloc{src/lib/compiler/toplevel/compiler/mythryl-compiler-for-intel32-posix.pkg}{src/lib/compiler/toplevel/compiler/mythryl-compiler-for-intel32-posix.pkg}; 
other platforms work similarly.

See also: \ahrefloc{pkg:makelib}{makelib} package.
