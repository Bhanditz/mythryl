The {\tt process\_deathwatch} package implements the \ahrefloc{api:Process\_Deathwatch}{Process\_Deathwatch} api.

The {\tt process\_deathwatch} package source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/process-deathwatch.pkg}{src/lib/src/lib/thread-kit/src/process-deathwatch.pkg}.
