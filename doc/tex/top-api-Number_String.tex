
The standard library {\tt Number\_String} api defines access to string scanning 
and number extraction functionality.  This is an older API predating the C-inspired  
\ahrefloc{api:Scanf}{Scanf} api.

The {\tt Number\_String} api is implemented by the \ahrefloc{pkg:number\_string}{number\_string} package.

The {\tt Number\_String} api source code is in \ahrefloc{src/lib/std/src/number-string.api}{src/lib/std/src/number-string.api}.

See also:   \ahrefloc{api:Scanf}{Scanf}.
