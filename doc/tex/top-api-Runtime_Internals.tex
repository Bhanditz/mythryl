
The standard library {\tt Runtime\_Internals} api defines access to functionality 
such as control of the garbage collector.

The {\tt Runtime\_Internals} api is implemented by  
{\tt runtime\_internals} in 
\ahrefloc{src/lib/std/src/nj/runtime-internals.pkg}{src/lib/std/src/nj/runtime-internals.pkg}.

The {\tt Runtime\_Internals} api source code is in \ahrefloc{src/lib/std/src/nj/runtime-internals.api}{src/lib/std/src/nj/runtime-internals.api}.
