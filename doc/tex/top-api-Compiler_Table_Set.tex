
The compiler {\tt Compiler\_Table\_Set} api defines access to the top-level 
symbol table functionality needed by the \ahrefloc{pkg:compiler\_state}{compiler\_state} package.

The {\tt Compiler\_Table\_Set} api is implemented by the \ahrefloc{pkg:compiler\_table\_set}{compiler\_table\_set} package.

The {\tt Compiler\_Table\_Set} api source code is in \ahrefloc{src/lib/compiler/toplevel/compiler-state/compiler-mapstack-set.api}{src/lib/compiler/toplevel/compiler-state/compiler-mapstack-set.api}.

See also:  \ahrefloc{api:Compiler\_State}{Compiler\_State}.
