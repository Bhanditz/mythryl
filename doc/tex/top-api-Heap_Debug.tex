
The compiler {\tt Heap\_Debug} api defines access to the compiler infrastructure 
for debugging heap corruption problems. 

The {\tt Heap\_Debug} api is implemented by the \ahrefloc{pkg:heap\_debug}{heap\_debug} package. 

The {\tt Heap\_Debug} api source code is in \ahrefloc{src/lib/std/src/nj/heap-debug.api}{src/lib/std/src/nj/heap-debug.api}.
