
The standard library {\tt simple\_union\_find} package implements efficient disjoint-set {\tt union} 
and {\tt find} operations.  For algorithmic background, see \ahref{\unionfind}{wikipedia union/find}.

The {\tt simple\_union\_find} package implements the \ahrefloc{api:Union\_Find}{Union\_Find} API.

The {\tt simple\_union\_find} package source code is in \ahrefloc{src/lib/src/disjoint-sets-with-constant-time-union-simple-version.pkg}{src/lib/src/disjoint-sets-with-constant-time-union-simple-version.pkg}.

See also:  \ahrefloc{pkg:union\_find}{union\_find}.
