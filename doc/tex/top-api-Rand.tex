
The standard library {\tt Rand} api defines access to a traditional multiplicative-congruential 
style pseudo-random number generator.  (The \ahrefloc{api:Random}{Random} API gives access 
to a more modern, higher quality pseudo-random number generator.)

The {\tt Rand} api is implemented by the \ahrefloc{pkg:rand}{rand} package.

The {\tt Rand} api source code is in \ahrefloc{src/lib/src/rand.api}{src/lib/src/rand.api}.
