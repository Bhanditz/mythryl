
The compiler {\tt makelib\_client\_boss} package is part of the internal 
support for distributed program compilation using multiple processes 
potentially on multiple hosts.  This facility is a nonfunctional 
work in progress.

The {\tt makelib\_client\_boss} package implements the \ahrefloc{src/app/makelib/concurrency/makelib-client-boss.pkg}{Makelib\_Client\_Boss} api.

The {\tt makelib\_client\_boss} package source code is in \ahrefloc{src/app/makelib/concurrency/makelib-client-boss.pkg}{src/app/makelib/concurrency/makelib-client-boss.pkg}.

See also: \ahrefloc{pkg:makelib}{makelib} package.

See also: \ahrefloc{src/app/makelib/main/makelib-server.pkg}{src/app/makelib/main/makelib-server.pkg}.
