
The standard library {\tt thread\_deathwatch} package implements functionality to detect and report unexpected thread terminations.

The {\tt thread\_deathwatch} package implements the \ahrefloc{api:Thread\_Deathwatch}{Thread\_Deathwatch} API.

The {\tt thread\_deathwatch} package source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/lib/thread-deathwatch.pkg}{src/lib/src/lib/thread-kit/src/lib/thread-deathwatch.pkg}.
