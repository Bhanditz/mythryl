13.4.155  template_hostthread

The standard library template_hostthread package provides a clone-and-mutate template for defining new dedicated server hostthreads.

The template_hostthread package implements the Template_Hostthread api.

The template_hostthread package source code is in src/lib/std/src/hostthread/template-hostthread.pkg.

The above information is manually maintained and may contain errors.
