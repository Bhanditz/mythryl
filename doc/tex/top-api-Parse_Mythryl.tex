
The compiler {\tt Parse\_Mythryl} api defines access to parsing of Mythryl source code syntax.

The {\tt Parse\_Mythryl} api is implemented by the \ahrefloc{pkg:parse\_mythryl}{parse\_mythryl} package.

The {\tt Parse\_Mythryl} api source code is in \ahrefloc{src/lib/compiler/front/parser/main/parse-mythryl.pkg}{src/lib/compiler/front/parser/main/parse-mythryl.pkg}.
