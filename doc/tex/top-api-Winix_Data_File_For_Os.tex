
The standard library {\tt Threadkit\_Winix\_Data\_File\_For\_Os} api defines access to basic low-level 
file binary read and write functionality.

The {\tt Threadkit\_Winix\_Data\_File\_For\_Os} api is implemented by the {\tt threadkit\_winix\_data\_file\_for\_os\_g} generic package.

The {\tt Threadkit\_Winix\_Data\_File\_For\_Os} api is implemented by the \ahrefloc{pkg:threadkit\_winix\_data\_file\_for\_posix}{threadkit\_winix\_data\_file\_for\_posix} package.

The {\tt Threadkit\_Winix\_Data\_File\_For\_Os} api source code is in \ahrefloc{src/lib/std/src/io/threadkit-winix-data-file-for-os.api}{src/lib/std/src/io/threadkit-winix-data-file-for-os.api}.
