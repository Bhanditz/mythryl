
The standard library {\tt threadkit\_driver\_for\_posix} package implements 
the interface to an OS specific module that glues the various 
OS-specific scheduling operations together (i.e., timeouts, I/O, signals, etc).

The {\tt threadkit\_driver\_for\_posix} package implements the \ahrefloc{api:Threadkit\_Driver\_For\_Os}{Threadkit\_Driver\_For\_Os} api.

The {\tt threadkit\_driver\_for\_posix} package source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/posix/threadkit-driver-for-posix.pkg}{src/lib/src/lib/thread-kit/src/posix/threadkit-driver-for-posix.pkg}.
