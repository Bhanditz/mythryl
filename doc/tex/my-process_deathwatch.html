13.4.106  process_deathwatch

The standard library process_deathwatch package gives multithreaded Mythryl program access to exit status of host-os subprocesses.

The process_deathwatch package implements the Process_Deathwatch api.

The process_deathwatch package source code is in src/lib/src/lib/thread-kit/src/process-deathwatch.pkg.
