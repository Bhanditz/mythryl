13.4.54  io_now_possible_mailop

The standard library io_now_possible_mailop package implements an internal support facility used by multithreaded Mythryl programs. It is not normally of direct interest to the application programmer.

The io_now_possible_mailop package implements the Io_Now_Possible_Mailop api.

The io_now_possible_mailop package source code is in src/lib/src/lib/thread-kit/src/core-thread-kit/io-now-possible-mailop.pkg. The above information is manually maintained and may contain errors.
