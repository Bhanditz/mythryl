
The standard library {\tt Say} api defines access to functions which write string lists 
to various output streams.  This is an older API predating (for example) \ahrefloc{api:Sfprintf}{Sfprintf}.

The {\tt Say} api is implemented by the \ahrefloc{pkg:say}{say} package.

The {\tt Say} api source code is in \ahrefloc{src/lib/std/src/io/say.pkg}{src/lib/std/src/io/say.pkg}.

See also:  \ahrefloc{api:Sfprintf}{Sfprintf}.
