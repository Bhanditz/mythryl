13.4.151  string_junk

The standard library string_junk package implements extended support for strings, such as atoi, atod, basename, dirname, trim.

The string_junk package implements the String_Junk api.

The string_junk package source code is in src/lib/std/src/string-junk.pkg.

See also: string.
