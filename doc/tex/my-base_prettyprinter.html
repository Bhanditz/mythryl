13.4.3  base_prettyprinter

The standard library base_prettyprinter package implements an underlying layer of standard_prettyprinter.

The base_predttyprinter package implements the Base_Prettyprinter api.

The base_prettyprinter package source code is in src/lib/prettyprint/big/src/base-prettyprinter.pkg.
