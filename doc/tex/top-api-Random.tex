
The standard library {\tt Random} api defines access to a subtract-with-borrow style pseudo-random 
number generator.

The {\tt Random} api is implemented by the \ahrefloc{pkg:random}{random} package.

The {\tt Random} api source code is in \ahrefloc{src/lib/src/random.api}{src/lib/src/random.api}.

See also:  \ahrefloc{api:Rand}{Rand} API.
