
The standard library {\tt int\_list\_map} package implements 
finite maps on integer keys via sorted lists.

The {\tt int\_list\_map} package implements the \ahrefloc{api:Map}{Map} API.

The {\tt int\_list\_map} package source code is in \ahrefloc{src/lib/src/int-list-map.pkg}{src/lib/src/int-list-map.pkg}.
