
The standard library {\tt dotgraph\_to\_planargraph} package implements access to internal graph-visualization functionality. 

The {\tt dotgraph\_to\_planargraph} package implements the \ahrefloc{api:Dotgraph\_To\_Planargraph}{Dotgraph\_To\_Planagraph} api.

The {\tt dotgraph\_to\_planargraph} package source code is in \ahrefloc{src/lib/std/dot/dotgraph-to-planargraph.pkg}{src/lib/std/dot/dotgraph-to-planargraph.pkg}.
