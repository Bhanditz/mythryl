
The standard library {\tt integer} package implements indefinite-precision integer arithmetic.

The {\tt integer} package implements the \ahrefloc{api:Multiword\_Int}{Multiword\_Int} API, 
which is a superset of the vanilla \ahrefloc{api:Int}{Int} API.

The {\tt integer} package source code is in \ahrefloc{src/lib/std/multiword-int.pkg}{src/lib/std/multiword-int.pkg} 
which is a proxy for \ahrefloc{src/lib/std/src/multiword-int-guts.pkg}{src/lib/std/src/multiword-int-guts.pkg}.
