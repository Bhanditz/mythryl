13.4.99  planfile

The standard library planfile package implements hashtables specialized to integer keys. reads one or more .plan files like src/opt/gtk/etc/gtk-construction.plan, parses and validate the contents, and returns them as a list of paragraphs.

The planfile package implements the Planfile api.

The planfile package source code is in src/lib/make-library-glue/planfile.pkg.

See also: planfile_junk.
