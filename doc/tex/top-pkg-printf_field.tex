
The standard library {\tt printf\_field} package implements support functionality shared 
by the \ahrefloc{pkg:sfprintf}{sfprintf} and \ahrefloc{pkg:scanf}{scanf} packages.

The {\tt printf\_field} package API is specified anonymously inline in the source.

The {\tt printf\_field} package source code is in \ahrefloc{src/lib/src/printf-field.pkg}{src/lib/src/printf-field.pkg}.
