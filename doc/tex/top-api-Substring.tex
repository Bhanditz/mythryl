
The standard library {\tt Substring} api defines access to functionality allowing 
parts of \ahrefloc{api:String}{String} instances to be referenced in 
constant time in space.

The {\tt Substring} api is implemented by the \ahrefloc{pkg:substring}{substring} package.

The {\tt Substring} api source code is in \ahrefloc{src/lib/core/init/substring.api}{src/lib/core/init/substring.api} 
from which it propagates to (e.g.) \ahrefloc{src/lib/std/src/substring.api}{src/lib/std/src/substring.api}.

See also:  \ahrefloc{api:String}{String}.
