
The standard library {\tt float} package implements the basic floating point arithmetic facility. 
It is a proxy for {\tt float64\_guts} from \ahrefloc{src/lib/std/src/eight-byte-float-guts.pkg}{src/lib/std/src/eight-byte-float-guts.pkg}.

The {\tt float} package implements the \ahrefloc{api:Float}{Float} API.

The {\tt float} package source code is in \ahrefloc{src/lib/std/float.pkg}{src/lib/std/float.pkg}.

See also:  \ahrefloc{pkg:float64}{float64}.
