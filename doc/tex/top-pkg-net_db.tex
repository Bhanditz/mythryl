
The standard library {\tt net\_db} package implements basic low-level {\sc TCP/IP} 
functionality to look up internet networks by name and by address and so forth.

The {\tt net\_db} package implements the \ahrefloc{api:Net\_Db}{Net\_Db} API.

The {\tt net\_db} package source code is in \ahrefloc{src/lib/std/src/socket/net-db.pkg}{src/lib/std/src/socket/net-db.pkg}.
