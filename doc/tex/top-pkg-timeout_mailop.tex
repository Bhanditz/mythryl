
The standard library {\tt timeout\_mailop} package implements timing out and sleep functionality for Mythryl the interthread communication facility.

The {\tt timeout\_mailop} package implements the \ahrefloc{api:Timeout\_Mailop}{Timeout\_Mailop} api.

The {\tt timeout\_mailop} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/timeout-mailop.pkg}.
