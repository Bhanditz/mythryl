
The standard library {\tt net\_protocol\_db} package implements basic {\sc TCP/IP} logic to 
look up a protocol by name or by number.

The {\tt {\tt net\_protocol\_db}} package implements the \ahrefloc{api:Net\_Protocol\_Db}{Net\_Protocol\_Db} API.

The {\tt {\tt net\_protocol\_db}} package source code is in \ahrefloc{src/lib/std/src/socket/net-protocol-db.pkg}{src/lib/std/src/socket/net-protocol-db.pkg}.
