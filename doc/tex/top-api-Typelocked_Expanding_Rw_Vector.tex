
The standard library {\tt Typelocked\_Expanding\_Rw\_Vector} api defines access to mutable 
vectors which expand dynamically as needed.

Implementations of the {\tt Typelocked\_Expanding\_Rw\_Vector} api 
may be generated via {\tt expanding\_rw\_vector\_g} 
from \ahrefloc{src/lib/src/expanding-rw-vector-g.pkg}{src/lib/src/expanding-rw-vector-g.pkg}.

The {\tt Typelocked\_Expanding\_Rw\_Vector} api source code is in \ahrefloc{src/lib/src/typelocked-expanding-rw-vector.api}{src/lib/src/typelocked-expanding-rw-vector.api}.

See also:  \ahrefloc{api:Expanding\_Rw\_Vector}{Expanding\_Rw\_Vector}.
