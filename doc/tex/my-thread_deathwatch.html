13.4.156  thread_deathwatch

The standard library thread_deathwatch package implements functionality to detect and report unexpected thread terminations.

The thread_deathwatch package implements the Thread_Deathwatch API.

The thread_deathwatch package source code is in src/lib/src/lib/thread-kit/src/lib/thread-deathwatch.pkg.
