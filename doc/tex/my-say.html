13.4.139  say

The standard library say package implemetns functions which write string lists to various output streams. This is an older package predating (for example) sfprintf.

The say package implements the Say API.

The say package source code is in src/lib/std/src/io/say.pkg.

See also: sfprintf.

The above information is manually maintained and may contain errors.
