
The compiler {\tt Property\_List} api defines access to a general-purpose datastructure annotation 
facility patterned after Stephen Week's Mlton annotation facility.

The {\tt Property\_List} api is implemented by the \ahrefloc{pkg:property\_list}{property\_list} package.

The {\tt Property\_List} api source code is in \ahrefloc{src/lib/src/property-list.api}{src/lib/src/property-list.api}.
