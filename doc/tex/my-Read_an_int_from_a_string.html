5.5.11  Read an int from a string

Read an int from a string: 

    f = atoi "23";

Read an int from a string, II: 

    f = int::from_string "23";

Read an int from a string, III: 

    eval:  sscanf "123" "%d";

    THE [INT 123]

    eval:  scanf::sscanf "123" "%d";

    THE [INT 123]
