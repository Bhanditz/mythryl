
The standard library {\tt posix\_signals} package implements 
masking, trapping and handling of posix interprocess signals.  It also provides similar handling 
for a pseudo-signal generated by the Mythryl runtime when the garbage collector 
runs.

The {\tt posix\_signals} package implements the \ahrefloc{api:Posix\_Signals}{Posix\_Signals} api.

The {\tt posix\_signals} package source code is in \ahrefloc{src/lib/std/src/posix/posix-signals.pkg}{src/lib/std/src/posix/posix-signals.pkg}.
