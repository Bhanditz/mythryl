
The standard library {\tt rw\_vector\_slice\_of\_one\_byte\_unts} package implements access to subvectors within a 
\ahrefloc{pkg:rw\_vector\_of\_one\_byte\_unts}{rw\_vector\_of\_one\_byte\_unts} instance.  It allows referencing such subvectors in 
constant time and space without duplication of data.

The {\tt rw\_vector\_slice\_of\_one\_byte\_unts} package implements the \ahrefloc{api:Typelocked\_Rw\_Vector\_Slice}{Typelocked\_Rw\_Vector\_Slice} API.

The {\tt rw\_vector\_slice\_of\_one\_byte\_unts} package source code is in \ahrefloc{src/lib/std/src/rw-vector-slice-of-one-byte-unts.pkg}{src/lib/std/src/rw-vector-slice-of-one-byte-unts.pkg}.
