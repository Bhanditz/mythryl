13.4.130  rw_float_vector

The standard library rw_float_vector package is a proxy for the rw_vector_of_eight_byte_floats package.

The rw_float_vector package implements the Typelocked_Rw_Vector API.

The rw_float_vector package source code is in proxied in src/lib/std/rw-float-vector.pkg.

The above information is manually maintained and may contain errors.
