13.2.63  text

The standard library text package implements access to the functionality of seven different string-related packages: char, string, substring, vector_of_chars, rw_vector_of_chars, vector_slice_of_chars, rw_vector_slice_of_chars.

The text package implemented the Text api.

The text package source code is in src/lib/std/src/text.pkg.

The above information is manually maintained and may contain errors.
