
The standard library {\tt Oneshot\_Maildrop} api defines access to maildrops specialized to being set only once.

The {\tt Oneshot\_Maildrop} api is implemented by the \ahrefloc{pkg:oneshot\_maildrop}{oneshot\_maildrop} package.

The {\tt Oneshot\_Maildrop} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/oneshot-maildrop.api}{src/lib/src/lib/thread-kit/src/core-thread-kit/oneshot-maildrop.api}.
