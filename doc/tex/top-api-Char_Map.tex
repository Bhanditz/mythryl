

The standard library {\tt Char\_Map} api defines simple pure maps from 
type Char to other types.

The {\tt Char\_Map} api is implemented by the \ahrefloc{pkg:char\_map}{char\_map} package.

The {\tt Char\_Map} api source code is in \ahrefloc{src/lib/src/char-map.api}{src/lib/src/char-map.api}.
