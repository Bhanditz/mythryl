13.2.61  symbol_path

The compiler symbol_path package implements manipulation of lists of Symbol instances representing access chains like a::b::c.

The symbol_path package implements the Symbol_Path API.

The symbol_path package source code is in src/lib/compiler/front/typer-stuff/basics/symbol-path.pkg.

The above information is manually maintained and may contain errors.
