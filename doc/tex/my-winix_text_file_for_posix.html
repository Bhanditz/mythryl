13.4.189  winix_text_file_for_posix

The standard library winix_text_file_for_posix package implements access to basic low-level file I/O for multi-microthreaded programming.

The winix_text_file_for_posix package implements the Winix_Text_File_For_Os api.

The winix_text_file_for_posix package source code is in src/lib/std/src/posix/winix-text-file-for-posix.pkg.
