14.4.92  Set_Sigalrm_Frequency

The standard library Set_Sigalrm_Frequency api defines access to host OS interval timer functionality. On Posix this is setitimer(), on MS-Windows this is win32StartTimer().

The Set_Sigalrm_Frequency api is implemented by the set_sigalrm_frequency package.

The Set_Sigalrm_Frequency api source code is in src/lib/std/src/nj/set-sigalrm-frequency.api.

The above information is manually maintained and may contain errors. 

api {
    tick : Void -> time::Time;
    set_sigalrm_frequency : Null_Or(time::Time ) -> Void;};
