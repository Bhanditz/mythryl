
The standard library {\tt regex} package implements Perl-style regular expression matching 
over strings.

This is the library-standard regular expression style.  For example, the default 
\ahrefloc{pkg:scripting\_globals}{binding} of the {\tt \verb|=~|} infix operator in 
\ahrefloc{src/app/makelib/main/makelib-g.pkg}{src/app/makelib/main/makelib-g.pkg} is taken 
from the {\tt regex} package.

The {\tt regex} package implements the \ahrefloc{api:Regular\_Expression\_Matcher}{Regular\_Expression\_Matcher} API.

The {\tt regex} package source code is in \ahrefloc{src/lib/regex/regex.pkg}{src/lib/regex/regex.pkg}. 
It simply invokes {\tt regular\_expression\_matcher\_g} from \ahrefloc{src/lib/regex/glue/regular-expression-matcher-g.pkg}{src/lib/regex/glue/regular-expression-matcher-g.pkg}.
