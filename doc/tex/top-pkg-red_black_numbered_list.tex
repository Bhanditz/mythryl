

The standard library {\tt red\_black\_numbered\_list} package implements a red-black tree 
datastructure which both maps keys to values and also allows keys to 
be accessed by sequential number within the ordered keyset.

The {\tt red\_black\_numbered\_list} package implements the \ahrefloc{api:Numbered\_List}{Numbered\_List} API.

The {\tt red\_black\_numbered\_list} package source code is in \ahrefloc{src/lib/src/red-black-numbered-list.pkg}{src/lib/src/red-black-numbered-list.pkg} 
and is proxied by \ahrefloc{pkg:sequence}{sequence}.

\ahrefloc{src/lib/src/red-black-numbered-set-g.pkg}{src/lib/src/red-black-numbered-set-g.pkg}.
