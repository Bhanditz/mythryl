
The standard library {\tt logger} package implements functionality for logging execution of Mythryl programs.

The {\tt logging} package implements by the \ahrefloc{api:Logger}{Logger} api.

The {\tt logging} package source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/lib/logger.pkg}{src/lib/src/lib/thread-kit/src/lib/logger.pkg}. 

For more information see the {\tt src/A.LOGGER.OVERVIEW} file in the distribution sourcetree.
