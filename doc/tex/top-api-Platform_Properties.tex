
The standard library {\tt Host\_Info} api defines access to information on the underlying host hardware architecture and operating system.

The {\tt Host\_Info} api is implemented by the {\tt host\_info} package in \ahrefloc{src/lib/std/src/nj/platform-properties.pkg}{src/lib/std/src/nj/platform-properties.pkg}.

The {\tt Host\_Info} api source code is in \ahrefloc{src/lib/std/src/nj/platform-properties.api}{src/lib/std/src/nj/platform-properties.api}.
