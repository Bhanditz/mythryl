
The standard library {\tt symbolmapstack} package implements one of the three tables 
making up core compiler state, the other two being the  
\ahrefloc{pkg:linking\_mapstack}{linking\_mapstack} and 
\ahrefloc{pkg:inlining\_mapstack}{inlining\_mapstack}.

The {\tt symbolmapstack} package implements the \ahrefloc{api:Symbolmapstack}{Symbolmapstack} API.

The {\tt symbolmapstack} package source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/symbolmapstack/symbolmapstack.pkg}{src/lib/compiler/front/typer-stuff/symbolmapstack/symbolmapstack.pkg}.
