
The standard library {\tt Net\_Db} api defines access to basic low-level {\sc TCP/IP} 
functionality to look up internet networks by name and by address and so forth.

The {\tt Net\_Db} api is implemented by the \ahrefloc{pkg:net\_db}{net\_db} package.

The {\tt Net\_Db} api source code is in \ahrefloc{src/lib/std/src/socket/net-db.api}{src/lib/std/src/socket/net-db.api}.
