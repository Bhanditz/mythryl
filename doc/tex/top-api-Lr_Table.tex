
The standard library {\tt Lr\_Table} api defines access to part of the mythryl-yacc parser-generator support infrastructure.

The {\tt Lr\_Table} api is implemented by the \ahrefloc{pkg:lr\_table}{lr\_table} package.

The {\tt Lr\_Table} api source code is in \ahrefloc{src/app/yacc/lib/base.api}{src/app/yacc/lib/base.api}.
