
The standard library {\tt fixed\_int} package is a proxy for {\tt one\_word\_int\_guts} from 
\ahrefloc{src/lib/std/src/one-word-int-guts.pkg}{src/lib/std/src/one-word-int-guts.pkg} which 
It provides core fixed-integer arithmetic capabilities.

The {\tt fixed\_int} package 
implements the {\tt Int} API from \ahrefloc{src/lib/std/src/int.api}{src/lib/std/src/int.api}.

The {\tt fixed\_int} package source code is in \ahrefloc{src/lib/std/fixed-int.pkg}{src/lib/std/fixed-int.pkg}.
