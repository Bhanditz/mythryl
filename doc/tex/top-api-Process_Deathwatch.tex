
The standard library {\tt Process\_Deathwatch} api gives multithreaded Mythryl program access to exit status of host-os subprocesses.

The {\tt Process\_Deathwatch} api is implemented by the \ahrefloc{pkg:process\_deathwatch}{process\_deathwatch} package.

The {\tt Process\_Deathwatch} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/process-deathwatch.api}{src/lib/src/lib/thread-kit/src/process-deathwatch.api}.
