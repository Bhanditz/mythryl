
The compiler {\tt Compiler\_State} api defines access to the top-level 
file compilation state.

The {\tt Compiler\_State} api is implemented by the \ahrefloc{pkg:compiler\_state}{compiler\_state} package.

The {\tt Compiler\_State} api source code is in \ahrefloc{src/lib/compiler/toplevel/interact/compiler-state.api}{src/lib/compiler/toplevel/interact/compiler-state.api}.
