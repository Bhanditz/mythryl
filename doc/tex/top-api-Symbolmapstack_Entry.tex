
The compiler {\tt Symbolmapstack\_Entry} api defines access to the contents of one symbol 
table slot in the compiler.

The {\tt Symbolmapstack\_Entry} api is implemented by the \ahrefloc{pkg:symbolmapstack\_entry}{symbolmapstack\_entry} package.

The {\tt Symbolmapstack\_Entry} api source code is in \ahrefloc{src/lib/compiler/front/typer-stuff/symbolmapstack/symbolmapstack-entry.api}{src/lib/compiler/front/typer-stuff/symbolmapstack/symbolmapstack-entry.api}.
