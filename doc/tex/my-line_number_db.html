13.5.21  line_number_db

The standard library line_number_db package implements functionality for tracking regions in source-code files, primarily for issuing error messages.

The line_number_db package implements the Line_Number_Db package.

The line_Number_db package source code is in src/lib/compiler/front/basics/source/line-number-db.pkg. The above information is manually maintained and may contain errors.
