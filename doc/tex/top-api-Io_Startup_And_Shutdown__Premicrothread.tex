
The standard library {\tt Io\_Startup\_And\_Shutdown\_\_Premicrothread} api defines tracking of open I/O streams, to close them cleanly at exit.

The {\tt Io\_Startup\_And\_Shutdown\_\_Premicrothread} api is implemented by the \ahrefloc{pkg:io\_startup\_and\_shutdown\_\_premicrothread}{io\_startup\_and\_shutdown\_\_premicrothread} package.

The {\tt Io\_Startup\_And\_Shutdown\_\_Premicrothread} api source code is in \ahrefloc{src/lib/std/src/io/io-startup-and-shutdown--premicrothread.api}{src/lib/std/src/io/io-startup-and-shutdown--premicrothread.api}.

See also: \ahrefloc{api:Io\_Startup\_And\_Shutdown}{Io\_Startup\_And\_Shutdown}.
