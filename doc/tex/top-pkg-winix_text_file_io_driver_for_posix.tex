
The standard libary {\tt winix\_text\_file\_io\_driver\_for\_posix} package implements base-level input/output functionality for multithreaded Mythryl programs on posix operating systems.

The {\tt winix\_text\_file\_io\_driver\_for\_posix} package implements a superset of the \ahrefloc{api:Winix\_Extended\_File\_Io\_Driver\_For\_Os}{Winix\_Extended\_File\_Io\_Driver\_For\_Os} api.

The {\tt winix\_text\_file\_io\_driver\_for\_posix} package source code is in \ahrefloc{src/lib/std/src/posix/winix-text-file-io-driver-for-posix.pkg}{src/lib/std/src/posix/winix-text-file-io-driver-for-posix.pkg}.

See also: \ahrefloc{pkg:winix\_text\_file\_io\_driver\_for\_posix}{winix\_text\_file\_io\_driver\_for\_posix} package.
