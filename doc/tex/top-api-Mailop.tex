
The standard library {\tt Mailop} api defines mailop values and combinators --- basic Mythryl inter-thread communication functionality.

An extended version of the {\tt Mailop} api is implemented by the \ahrefloc{pkg:mailop}{mailop} package.

The {\tt Mailop} api source code is in \ahrefloc{src/lib/src/lib/thread-kit/src/core-thread-kit/mailop.api}{src/lib/src/lib/thread-kit/src/core-thread-kit/mailop.api}.
