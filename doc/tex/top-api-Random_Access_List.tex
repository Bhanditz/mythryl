
The standard library {\tt Random\_Access\_List} api defines lists which support efficient 
random access as well as traditional list-style access and operations.

The {\tt Random\_Access\_List} api is implemented by the \ahrefloc{pkg:binary\_random\_access\_list}{binary\_random\_access\_list} package.

The {\tt Random\_Access\_List} api source code is in \ahrefloc{src/lib/src/random-access-list.api}{src/lib/src/random-access-list.api}.
