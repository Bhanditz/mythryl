
The standard library {\tt simple\_prettyprinter} package defines 
a fast, simple prettyprinter.  Its major advantage is that 
the complete implementation is only 300 lines of code --- you 
can read and understand it completely in an hour.

The {\tt simple\_prettyprinter} package implements the \ahrefloc{api:Simple\_Prettyprinter}{Simple\_Prettyprinter} api.

The {\tt simple\_prettyprinter} package source code is in \ahrefloc{src/lib/prettyprint/simple/simple-prettyprinter.pkg}{src/lib/prettyprint/simple/simple-prettyprinter.pkg}.

See also: \ahrefloc{pkg:prettyprint}{prettyprint}.
