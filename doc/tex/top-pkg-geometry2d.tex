
The standard library {\tt geometry2d} package implements the basic geometry types and operations.

The {\tt geometry2d} package implements the \ahrefloc{api:Geometry2d}{Geometry2d} api.

The {\tt geometry2d} package source code is in \ahrefloc{src/lib/std/2d/geometry2d.pkg}{src/lib/std/2d/geometry2d.pkg}.

See also:  \ahrefloc{pkg:geometry2d\_float}{geometry2d\_float} package.
