13.2.25  io_exceptions

The standard library io_exceptions package implements exceptions raised during basic file input/output functionality.

The io_exceptions package implements the Io_Exceptions API.

The io_exceptions package source code is in src/lib/std/src/io/io-exceptions.pkg.

The above information is manually maintained and may contain errors.
