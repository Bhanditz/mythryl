
The standard library {\tt Scanf} api defines access to text-scanning functionality 
inspired by the C {\tt scanf()} function.

The {\tt Scanf} api is implemented by the \ahrefloc{pkg:scanf}{scanf} package.

The {\tt Scanf} api source code is in \ahrefloc{src/lib/src/scanf.api}{src/lib/src/scanf.api}.
