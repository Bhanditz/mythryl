
The standard library {\tt List\_To\_String} api defines access to functionality tranlating 
Lists to Strings.

The {\tt List\_To\_String} api is implemented by the \ahrefloc{pkg:list\_to\_string}{list\_to\_string} package.

The {\tt List\_To\_String} api source code is in \ahrefloc{src/lib/src/list-to-string.api}{src/lib/src/list-to-string.api}.

See also:  \ahrefloc{api:String\_To\_List}{String\_To\_List}.
